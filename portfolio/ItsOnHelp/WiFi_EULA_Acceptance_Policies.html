WiFi EULA Acceptance Policies

WiFi EULA (End User License Agreement) policies are designed to allow ItsOn-powered devices to automatically accept EULA agreements when connecting to free WiFi networks where device users have already accepted those agreements. Free WiFi networks are typically public and do not require additional credentials, but do require EULA agreements for access. Rather than needing to accept the EULA every time that network is accessed, this policy allows for automatic acceptance for the defined time period. 

The ItsOn software can detect if a EULA has changed. If it has, the policy will not automatically connect the device and the device user will be prompted to accept the new EULA.

EULA acceptance policies have one setting.

Settings Description
Reset automatic EULA acceptance data every ... days Defines how long an ItsOn-enabled device saves EULA credentials after the user initially accepts them for a WiFI network. Select from 5, 10, 15, 20, 30 (default), 60, and 90 days.
