Add a Service Policy to a Network Feature

You can add a service policy to a network feature in 2 ways:

  • Add a new, completely blank service policy and configure it from scratch.
  • Add a new service policy as a copy of an existing service policy and modify the configuration of the copy.

The latter is especially helpful if you have existing service policies that are complex and/or well tested and you want to make only minor changes to a new one.

Add a New Blank Service Policy to a Network Feature

  1. Click Add Policy >
    • Data > Service Policy - to add a data service policy
    • Voice > Service Policy - to add a voice service policy
    • Messaging > Service Policy - to add a message service policy
  2. Click on the policy name to change it from the default name. The default name is something like New [Policy Type] Policy #, where [Policy Type] is either VOICE, DATA, or MESSAGING and # is a number. (Give it a meaningful name. This will help you find the policy when you are choosing from lists of policies.) The limit is 32 characters.
  3. If it is a voice or messaging service policy in a network feature, optionally add network enforcement by selecting a Bucket ID. If it is a data service policy in a network feature, optionally add network enforcement by selecting a Rating Group.
  4. Add one or more components or policy events.
  5. Click the Unlock button  (to save and lock).

Add a Copy of an Existing Service Policy to a Network Feature

  1. Click Add Policy >
    • Data > Copy Service Policy From - to select an existing data service policy
    • Voice > Copy Service Policy From - to select an existing voice service policy
    • Messaging >  Copy Service Policy From - to select an existing message service policy
  2. Find the policy you want to make a copy of, click to select it, and then click Add.
  3. Enter a new policy name to change it from the default name. The default name is something like [Policy Name] (Copy), where [Policy Name] is the name of the original service policy. (Give it a meaningful name. This will help you find the policy when you are choosing from lists of policies.) The limit is 32 characters.
  4. Click Save Copy.
  5. Make any desired changes.
  6. Click the Unlock button  (to save and lock).
