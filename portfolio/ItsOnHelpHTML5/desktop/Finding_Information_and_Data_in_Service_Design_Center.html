Finding Information and Data in Service Design Center

Service Design Center contains a lot of information and data, data about policies and policy components in policy management, data about products and restrictions and offers in catalog management, and data about accounts and subscribers and devices in customer service. In many cases, data is presented in tables, and those tables provide several ways for you to view and find the information you're looking for. Each row in the table shows key information about each item, and you can operate on the information that is presented. On pages with tables, you can

  • Sort
  • Filter
  • Search
  • Page

Sorting

Sort the data in tables by clicking on the header of any column that contains a icon, a  icon, or a icon. In some cases, you must hover your mouse pointer over the column header to see the sort icon.

In some tables, Service Design Center indicates which column is being used for sorting the table by highlighting the header. The header will also indicate the sort order: an up arrow ()  for lowest-to-highest order, and a down arrow () for highest-to-lowest sort. Filtering

Some tables in Service Design Center allow you to filter what rows the table displays by data you define. You can filter in 2 ways, by selecting specific data values (for table columns that contain a fixed set of values) or by entering your own text (for table columns that can contain any data).

Tables where you can filter on column data have a  icon in the column header. Click on the  icon to filter the table data.

In table columns that contain specific sets of values, such as product types (on the catalog product page) or statuses (on the Account page in the Subscriptions tables), when you click on the filter icon, you are presented with either a drop-down list, which means you can select one of the values to view all records that use that value, or a list with check boxes, which means that you can select one or more values and view all the records that use all the values you select.

In table columns that contain any value, when you click on the filter icon, you are presented with a plain text box. Enter text in the text field and click Filter. The resulting view is only the records that contain what you entered.

To remove the filtering, click the filter icon in in the column header of the column that is being filtered and click Clear.

Searching

Service Design Center offers as-you-type searching on many pages. These searches do not have a Search button next to the search box.

To search, start typing in the Search box. With each character you type, the items that remain visible in the list will be only those with names that contain what you have entered.

On pages where there is a Search button next to the search box, searches only occur after you click the Search button, not as you type.

Paging

Many list pages contain more items than can easily fit on a single page. Click on the Previous or Next buttons at the bottom to navigate sequentially through the pages, or click a number to jump to that page.
