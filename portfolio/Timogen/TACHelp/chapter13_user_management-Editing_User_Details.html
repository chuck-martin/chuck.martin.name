Editing User Details

Only users with the Technical Admin function can edit user details.

To edit user details
1 In the Function drop-down list, select Technical Admin.
2 Click the Access Manager tab, then click Users. The user list page appears.
3 Click a User ID link. The detail page for that user appears.
4 Make any desired changes to the user details. For more information, see "User Details".
5 Click Save.
