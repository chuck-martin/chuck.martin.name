Generating Resolution Plans

When a sandbox is created from an alert, a Resolution Plan is generated automatically by Timogen Adaptive Controller based on the sandbox configuration. If you make changes to the configuration, or if you create a sandbox separate from an alert, you can generate a new Resolution Plan. For more information, see "Resolution Plans".

To generate a Resolution Plan
1 View a sandbox. (See "Viewing Sandboxes".)
2 Click Generate Resolution Plan. A message appears allowing you to view a summary of the sandbox configuration.
3 Click Submit.
4 In the confirmation dialog box, click OK.
