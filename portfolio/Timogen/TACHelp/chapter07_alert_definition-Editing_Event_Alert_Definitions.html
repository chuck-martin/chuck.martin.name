Editing Event Alert Definitions

To edit an Event Alert definition
1 In the Role drop-down list, select Business Admin.
2 Click the Alert Definition tab, then click Alert Definition List.
3 Click the Event Alert tab.
4 Select one or more Items and Locations and select an Alert Category (or select Any to search for alerts for any Item or in any Location or Category), then click Search.
A list of alerts that fit your selection appears in the Alert Category section.
5 Click for the alert you want to edit. The Edit Event Alert Definition page appears.
6 Make any desired changes to Thresholds, Resolvers, and Escalation. (See "Event Alert Thresholds", "Event Alert Resolvers", and "Event Alert Escalation" for details.
You cannot change the Item, Category, or Location of an existing alert.
7 Click Save.
