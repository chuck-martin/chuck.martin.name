Finding an Impact Alert's Cause

An Impact Alert can occur as a result of one or more related Event Alerts. You can easily find the cause of any Impact Alert when that cause is an Event Alert.

To find an Impact Alert's cause
1 View the detail page of the alert you want to find the cause of. (See "Viewing Alerts".)
2 Click Root Cause.
· If the cause was a single Event Alert, you'll see its detail page.
· If the cause was two or more Event Alerts, you'll see a list of the alerts.
