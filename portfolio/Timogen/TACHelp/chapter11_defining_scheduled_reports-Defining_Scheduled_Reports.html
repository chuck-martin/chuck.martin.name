Defining Scheduled Reports

The Business Admin user can schedule reports to be run periodically. Timogen Adaptive Controller provides five types of reports:

· Alert Summary reports
· Inventory Detail reports
· PO vs. Forecast reports
· Related POs reports
· Inventory Summary by Location reports.

For each report you define, you can specify several criteria and the report generation schedule. Every time a scheduled report is run, it overwrites the previous version of that report.
