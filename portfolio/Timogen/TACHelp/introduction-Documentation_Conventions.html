Documentation Conventions

Bold
indicates operating system elements, such as filenames, directories, class names, buttons, check boxes, and menu commands
Code
indicates commands, formulae, syntax and code
Menu > Option
indicates what menu item to pick
\
indicates that a command is too long to fit in a single line
