Reconciliation/Consumption

Planned orders, actual orders and resolution tasks need to be reconciled with the data from the ERP system. Each transaction that needs to be reconciled will invoke one or more reconciliators that perform the reconciliation task.

The Consumption tab lists all demand nodes. You can edit the forecast consumptions settings for each demand node.
