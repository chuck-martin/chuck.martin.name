Selecting Problem Configuration Templates

A Problem Configuration Template contains a set of actions that can be used to resolve an alert. These templates are defined by the Business Admin user. For more details on the templates, see "Problem Configuration Templates".

To select a problem configuration template
1 View a sandbox. (See "Viewing Sandboxes".)
2 Click the Configuration tab in the lower pane, then click Template.
3 Click the button in the Load column for the problem configuration template you want to use for this sandbox.
Click on the name of the problem configuration template, to view its details.
4 Click Load.
