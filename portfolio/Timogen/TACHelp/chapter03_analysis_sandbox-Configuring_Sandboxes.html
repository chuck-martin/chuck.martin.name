Configuring Sandboxes

Configuring a sandbox includes:

· Selecting a Problem Configuration Template
· Defining a Resolution Plan
· Defining scope
· Defining business goals
· Defining constraints

You can view a sandbox configuration summary at any time.

To view a sandbox configuration summary
1 View a sandbox. (See "Viewing Sandboxes".)
2 Click the Configuration tab in the lower pane, then click Summary.
