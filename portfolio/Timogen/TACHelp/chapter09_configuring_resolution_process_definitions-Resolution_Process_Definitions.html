Resolution Process Definitions

A Resolution Process Definition comprises several Resolution Task Definitions and is used to resolve Impact Alerts. Task definitions define parameters for the tasks that can be done at a Node Group. A Resolution Process Definition is created automatically for every Node Group. See "Resolution Process" and "Node Groups".

Resolution Process Definitions can be defined only by users who have the Business Admin function.
