Deleting Scheduled Cleanups

Only users with the Technical Admin function can delete a scheduled cleanup task.

To stop a scheduled cleanup task
1 In the Function drop-down list, select Technical Admin.
2 Click the Cleanup tab, then click Scheduled Cleanup. The Scheduled Cleanup page appears
.
3 Click the Delete check box for the cleanup task your want to stop[.
4 Click Delete.
5 In the confirmation dialog box, click OK.
