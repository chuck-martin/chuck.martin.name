Inventory Detail Report Details

The Inventory Detail Report definition page contains the following fields:

Field Name
Description
Item
Select one or more Items to be included in the report.
Partner/Location
Select one or more Partners/Locations to be included in the report.
Bucket Start Date
Companies may wish to view data on a daily basis, once in five days, seven days, month etc. This time period is defined during product installation and is referred to as a `Time Bucket.'
Select the Bucket Start Date to set the beginning time to be considered.
Number of Buckets to show
Select the Number of Buckets to show to set the granularity of the report.
