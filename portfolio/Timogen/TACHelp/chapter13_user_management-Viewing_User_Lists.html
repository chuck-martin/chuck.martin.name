Viewing User Lists

To view the user list
1 In the Function drop-down list, select Technical Admin.
2 Click the Access Manager tab, then click Users. The Users page appears.
The Users page shows the following basic information about all users::
Field Name
Description
User ID
A unique ID assigned to the user.
User Name
The name of the user
Default Function
This is the default function assigned to the user.

You can:

· Create new users. See "Creating Users".
· Modify the details of any user. See "Editing User Details".
· Delete one or more users. See "Deleting Users".
