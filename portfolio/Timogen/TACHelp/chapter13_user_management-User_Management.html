User Management

User management involves setting up users and their roles in Timogen Adaptive Controller. A user belongs to at least one role. A role is a set of privileges. These privileges comprise of a set of permissions on a Node Group.

This chapter covers the following topics:

· "About Users"
· "Creating Users"
· "About Functions"

Users and roles must be defined by the Technical Admin user.
