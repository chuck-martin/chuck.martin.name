Assigning Resources to Workgroups

Workgroup membership is required for Resources so they can work on Projects in that Workgroup. Workgroup membership is also one thing needed, along with a login and password and a User Category, to log in to Project InVision.

Assigning Resources to Workgroups can be done only by users with access to Administrative functions.

To assign Resources to Workgroups

  1. Click Administration. The Project InVision Administration window appears.
    The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.
  2. Under Select Screen, click Resources.
  3. Click  Add Remove. The Workgroup Membership dialog box appears.
  4. In the Assign drop-down list, select Resource to a Workgroup.
  5. In the Workgroup drop-down list, select a Workgroup.
    The only Workgroups you can select are the Workgroups that you're a member of.
  6. Select one or more names in the Resources list
    Hold down the Shift key while clicking to select a contiguous group of names.
    Hold down the Ctrl key while clicking to select multiple discrete names.
    Click More or All to see more names in the list.
  7. Click  Add Selected.
  8. Click Submit.

You can click  Add All to add all names currently showing in the Resources list.
