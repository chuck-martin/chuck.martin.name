Logging In

To log in to Project InVision, you will need:

  • The Project InVision URL (Universal Resource Locator), also know as a web address
  • Your login name
  • Your password

When you log in to Project InVision for the first time, several ActiveX controls are downloaded to your computer. See Configuring Internet Explorer for details on how to properly configure Internet Explorer for Project InVision access.

To log in to Project InVision

  1. Start Internet Explorer.
  2. In the Address field, enter the Project InVision URL (Universal Resource Locator), then press Enter or click Go. You'll see the Project InVision Login page.
  3. If your company uses multiple Project InVision databases, select one from the Source drop-down list.
    The Source drop-down list will not appear if your company uses just one database source.
  4. In the Application drop-down list, select one of:
    • Portfolio Manager - The Project InVision program management application
    • Time Tracking - The Project InVision time-tracking application
    • Dashboard - The Project InVision Dashboard
  5. Enter your Name (user name) and Password.
  6. Click Login.

If you do not know your Name and Password, contact your Workgroup Administrator, who assigns initial passwords. If your account has been set up by your Administrator to use NT Authentication, you log on using your NT account and password.

When logging in to Portfolio Manager, you are logged in to your default Workgroup. The Workgroup Administrator defines the Workgroup when the account is set up. You may belong to more than one Workgroup but you can log on to only one Workgroup at a time. While logged on, you are limited to viewing only the data for the current Workgroup. You can switch between Workgroups while logged on by clicking the down arrow in the Navigation menu next to the current Workgroup, then selecting another Workgroup.
