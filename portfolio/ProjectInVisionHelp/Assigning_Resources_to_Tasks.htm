Assigning Resources to Tasks

In Project InVision, you assign Resources to Tasks:

  • By assigning them directly in a Task Detail form (when a Project is not associated with a project schedule file)
  • By assigning them in the project schedule file that is associated with a Project (the assignments are added to Project InVision upon synchronization)

When you assign Resources to Tasks, these Task assignments appear in the Assignment Hierarchy view for the appropriate Resource in the Time Tracking application.

To assign a Resource to a Task in Project InVision

  1. View a Tasks Detail Form.
  2. Under Pages, click Resources.
  3. Click a field in the Name column, then in the drop-down list, select a Resource name.
    You can choose only from Resources who have been assigned to the Project that this Task is a part of. See Assigning Resources to Projects.
  4. Enter the Units for the resource, a number between 0 and 1, representing the percent of time the Resource will work on the Task (1 being 100%, .5 being 50%, etc.) or enter a number of Current Work, representing the total number of hours the Resource will work on the Task between the start and finish dates.
  5. Click Calculate. If you entered Units, the Current Work hours are calculated. If you entered Current Work hours, the Unit is calculated.
  6. If desired, enter the Baseline Work for the task.
  7. Repeat steps 3 to 6 for each Resource you want to add to the Task.
  8. Click Submit.

To assign a Resource to a Task in a project schedule file

  • Assign resources to tasks directly in an associated project schedule file as you normally would. Resource assignments will be synchronized with the Tasks module upon check in of the project schedule file.
