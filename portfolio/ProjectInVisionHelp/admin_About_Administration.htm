About Administration

Users who have System Administrator or Workgroup Administrator access to Project InVision can perform a number of administrative functions with the Web-based Administration application.

To access the Web-based Administration application

  1. Log in to Project InVision.
  2. Click Administration.

The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.

The administrative functions available in the browser application include:

  • Clearing Association
  • Forcing Check In
  • Templates
  • Workgroup Membership
  • Editing User Logins
  • Pull-down Lists
