Adding Links

To add a link to a Project InVision record

  1. View a Detail Form.
  2. Under Pages, click Links.
  3. Click in the empty field in the Item column.
  4. In the drop-down list, select any option except URL.
  5. Click in the corresponding field in the Link column, the click the down arrow button.
  6. If you want to define a link to an existing record:
    1. Choose Select from Existing Items. The Find dialog box appears.
    2. Select a record from the list, then click Select.
    3. Click  Save.
    If you want to define a link to a new item:
    1. Choose Create New Item. The New Detail Form for that type of record appears.
    2. Enter the information for the new record, then click Submit.
    3. Click  Save.
  7. Click  Save.

To add a link to a Web address (URL) outside of Project InVision

  1. View a Detail Form.
  2. Under Pages, click Links.
  3. Click in the empty field in the Item column.
  4. In the drop-down list, select the URL option.
  5. Click in the corresponding field in the Link column, then click the down-arrow button. The URL dialog box appears.
  6. Enter the URL label (how the link will appear in Project InVision) and the URL (the full web address).
  7. Click Submit.
  8. Click  Save.
