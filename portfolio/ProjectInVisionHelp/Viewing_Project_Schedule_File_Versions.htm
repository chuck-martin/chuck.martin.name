Viewing Project Schedule File Versions

Project InVision saves several versions of the project schedule file that is associated with a Project InVision Project record. Every time an updated Project file is checked in to Project InVision, a new version of the file is created. You can view any version of a project schedule file that is stored with a Project InVision project, including the most recent version, known as the Master version.

Viewing a project schedule file requires you to have the project management application normally used to open that file installed on your computer.

To view a version of a project schedule file

  1. View a Project Detail form.
  2. Under Pages, click Versions.
  3. Click a version in the list.
    The version with the highest number is the most recent (Master) version.
  4. Click  Review.
    -or-
    In the Association column, click the link for the version you want to review.

Viewing a version of a project schedule file does not check it out.
