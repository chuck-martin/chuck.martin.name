Adding Notes to Timesheet Assignments

If you want to add notes to identify reasons for changing Remaining Hours or posting Actual Work Hours different from anticipated, you can add a note to the Timesheet Assignment.

To add a note to a Timesheet Assignment

  1. Click in the Approve field of the Task in the Timesheet View.
  2. Click the ellipsis button. The Notes dialog box appears.
  3. Enter your note text, then click Submit.

A  note icon (black for a newly added note, blue for a note on an approved timesheet, and red for a note on a rejected timesheet) appears in the Approve field.
