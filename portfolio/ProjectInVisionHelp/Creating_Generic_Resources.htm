Creating Generic Resources

Generic Resources include Resource information such as collections of skills, location, and availability, but without a specific person attached. Generic Resources are used for planning purposes: you can create projects, define the resources you need, then see who can fill those resources. Finally, you replace the generic resources with the people who have the needed skills.

To create a generic Resource without assigning it to a Project

  1. Under Modules, click Resources.
  2. Click  Find, in the Find using drop-down list, select Selector, select -All-, then click Select.
    -or-
    In the Selector tree, click -All-. “-All-” will appear in the Project Indicator.
  3. Click  New Resource. The New Resources Detail Form appears.
  4. Select Generic.
  5. Enter the Resource information. Fields with bold labels are required. Click the links under Pages to enter more information.
  6. Click Submit.

To create a generic Resource and assign it to a Project

  1. In a Projects list view, select a Project, then under Modules, click Resources.
    -or-
    In the Resources module, Click Find, select a Project, then click Select.
    -or-
    In the Resources module, select a Project from the Selector tree. The Project name will appear in the Project Indicator.
  2. Click  New Resource. The New Resources Detail Form appears.
  3. Select Generic.
  4. Enter the Resource information. Fields with bold labels are required. Click the links under Pages to enter more information.
  5. Click Submit.
