Creating Resources

Best practices are that the Workgroup Administrator should maintain Resources at the Enterprise level and add Resources to the appropriate Workgroups. A Resource may belong to one or more Workgroups. Only users with proper security may add Resources.

Resources may be assigned to multiple Workgroups, each Workgroup with its own customizations and configurations, such as workweek configuration, Non-Project Activities, Cost Centers, and Billing Codes.

  • Select a Primary Workgroup to use those customizations and configurations in all Workgroups where the Resource might be assigned.
  • Leave the Primary Workgroup blank to have the Workgroup-specific configurations and customizations apply to the Resource when assigned to the Workgroup.

Make sure that you're not creating a Resource that already exists in the Workgroup or the Enterprise. List views in the Resources module do not always show all Resources. If a Project is selected, only Resources assigned to that Project will be shown. Click -All- in the Selector to view all Resources. If the Resource is not in the Workgroup, he or she may still be in the Enterprise. The Workgroup Administrator can make a Resource in the Enterprise a member of the Workgroup.

To create a Resource without assigning it to a Project

  1. Under Modules, click Resources.
  2. Click  Find, in the Find using drop-down list, select Selector, select -All-, then click Select.
    -or-
    In the Selector tree, click -All-. “-All-” will appear in the Project Indicator.
  3. Click  New. The New Resources Detail Form appears.
  4. Enter the Resource information. Fields with bold labels are required. Click the links under Pages to enter more information.
  5. Click Submit.

Once a Resource is created, it can be assigned to Project teams.

To create a Resource and assign it to a Project

  1. In a Projects list view, select a Project, then under Modules, click Resources.
    -or-
    In the Resources module, click  Find, select a Project, then click Select.
    -or-
    In the Resources module, select a Project from the Selector tree. The Project name will appear in the Project Indicator.
  2. Click  New. The New Resources Detail Form appears.
  3. Enter the Resource information. Fields with bold labels are required. Click the links under Pages to enter more information.
  4. Click Submit.

When a Resource is created, it is automatically added to the Enterprise and the current Workgroup. If a Resource needs to be added to other Workgroups, contact the Workgroup Administrator.
