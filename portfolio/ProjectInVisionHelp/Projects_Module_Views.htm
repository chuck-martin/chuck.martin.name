Projects Module Views

Default Views have been created for the Projects module. Your security access determines what Views you can see.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some View names or fields may not exist or may be labeled differently than shown here.

-Standard-

This View shows general information about the major information that defines Projects. Fields: Project #, Project Title, Status, Manager, Type, Department, Request Date, Priority, Sponsor, Region, and Comments.

-Indicators-

Similar to the -Standard- view with the addition of Project Indicators.

Actual Status

This View shows specific information about the actual performance of Projects. Fields: Project #, Project Title, Manager, Sponsor, Status, Actual Start, Actual Finish, Actual Work, and Actual Cost.

Baseline

This View shows specific information about the Baseline estimates of Projects. Fields: Project #, Project Title, Manager, Sponsor, Status, Base Start, Base Finish, Base Duration, Base Work, and Base Cost.

Costs

This View shows summary cost information rolled up from the Project Tasks. Fields: Project #, Project Title, Manager, Sponsor, Status, Current Cost, Base Cost, Actual Cost, Remaining Cost, and Cost Variance.

Current Status

This View shows how Projects are progressing based on their current schedule. The current schedule is generally the most up-to-date estimate for when Tasks will be completed so, when you look at Current Status, you see how the Project is progressing based on the current estimate (which may differ from Actual). Fields: Project #, Project Title, Manager, Sponsor, Status, Current Start, Current Finish, Current Work, Current Cost, % Work Complete, and % Complete.

Current vs Baseline Dates

This View shows a comparison of current schedule dates versus the original Baseline Dates. Fields: Project #, Project Title, Manager, Sponsor, Status, Current Start, Baseline Start, Start Variance, Current Finish, Baseline Finish, and Finish Variance.

Gantt Chart

This View shows a graphical representation of the duration of Projects. Fields: Project #, Project Title, and a bar showing the duration of the Project.

List

This View shows Projects in Project number sequence. Fields: Project #, Project Title, Originator, Manager, Status, and Type.

Status

This View shows summary information about the Current Finish and Percent Complete of Projects. Comments are also included so you can see notes pertaining to the Status of each Project. Fields: Project #, Project Title, Manager, Current Finish, % Complete, and Comments.

Variance

This View shows Project variances on an individual Project basis. Fields: Project #, Project Title, Manager, Sponsor, Status, Start Variance, Finish Variance, Work Variance, and Cost Variance.

Work

This View shows the Actual Work performed on an individual Project basis. Fields: Project #, Project Title, Manager, Sponsor, Status, Current Work, Baseline Work, Remaining Work, Actual Work, and Work Variance.
