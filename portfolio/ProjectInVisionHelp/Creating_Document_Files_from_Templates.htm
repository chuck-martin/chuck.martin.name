Creating Document Files from Templates

Document templates can be stored in Project InVision and used as a basis for creating associated document files. Using templates ensures that document formats are consistent across projects.

Document Templates can be of any file type.

To create a new document from an existing template

  1. View a Document record that does not have an attached document (the Association field in a Documents list view is empty).
  2. Click  Select Template. The Select Template dialog box appears.
  3. Select a template from the Template list.
  4. Enter a file name for the new document. (You do not need to add a file extension.)
  5. Click Submit.

This process saves a copy of the template file under the new file name and associates it with the Document record. The associated file is then available to be checked out of Project InVision.
