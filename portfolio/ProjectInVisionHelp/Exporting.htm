Exporting

You can export records in any list view to a comma separated value (.CSV) file.

To export records

  1. Apply a view for the information you want to export.
  2. From the Output menu, choose Export. The Export dialog box appears.
  3. Enter a name for the file.
  4. Click Browse and select a location for the file to be saved.
  5. Click Submit.

Only the displayed information is exported. For example, if you're viewing 50 records, then only 50 records are exported. You can either click More to add more records to the view, or click All to view all records (may take some time to display), then export. The bottom right of the status bar shows how many records are being viewed and how many total records are available.
