Assigning Resources to Projects

When Resources are assigned to Projects, they are made available to the Project for Task or Task-related item assignments. You can add Resources to Projects in the Projects or the Resources module.

To assign a Resource to a Project

  1. In a Project module list, select a Project record, then click  Team. The Define Project Team dialog box appears.
    You can also view a Project Detail form, then click  Team in the toolbar there.
  2. In the Assign drop-down list, select Resources to Projects.
  3. Optionally select one or more options from the drop-down list boxes in the Define Criteria section to filter for resources with appropriate skills and experience.
  4. Select one or more names in the Resources Pool list
    Hold down the Shift key while clicking to select a contiguous group of names.
    Hold down the Ctrl key while clicking to select multiple discrete names.
  5. Optionally click a link under Check Availability to view reports that can detail Resource availability.
  6. Click  Add. The selected Resources are added to the Project.
  7. Click Submit.

You can click  Add All to add all names currently showing in the Resources Pool list.
