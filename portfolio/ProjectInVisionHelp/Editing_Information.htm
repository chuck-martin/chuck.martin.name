Editing Information

In many Portfolio Manager views, information is presented in a spreadsheet format, with rows representing single records and columns representing the data in each record. These views have two modes: Read Only and Editable. You can edit this information directly when you are in Editable mode, which you can select in the Navigation menu.

You cannot add or delete Task records if they are associated with a project schedule file. In these cases, the Project Schedule must be Checked Out, edited, and Checked In. Once Synchronized, the Tasks module is updated with the new information.

You can also use the Detail Form of an item. Detail Forms contain all an item's information, where a list contains a subset. See About Detail Forms.

Your security level determines the Views in which you have access to Editable mode

To edit directly-editable information

  1. In Editable mode, click in a field. If it is directly editable, you won't see an ellipsis button.
  2. Press F2.
  3. Make any desired changes.
  4. Click  Save.

To edit information that is not directly editable

  1. In Editable mode. click in a field that is not directly editable. An  ellipsis button will appear in the field.
  2. Click the ellipsis button. The Find dialog box appears.
  3. Select the desired value.
  4. Click Select.
  5. Click  Save.

You can make multiple changes without saving each change, but you must explicitly save changes before you log out. If you do not save all your changes, you will be asked upon logout.

If you save without completing all required fields, you'll see an error message. If the required fields are visible in the current View applied, the cell appears outlined in Red. If the Required fields are not visible, you must open the Detail Form and complete all required fields before you can save.
