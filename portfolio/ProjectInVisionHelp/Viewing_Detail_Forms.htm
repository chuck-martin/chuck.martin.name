Viewing Detail Forms

A record's Detail Form contains all the information about that record (Project, Task, Resource, Document, Issue, Defect, custom module, etc.), organized into pages. (Lists on the main page show selected details about records.)

For all list views other than for Projects, the list can show either a subset of records, based on a selected Project or group of Projects, or all records. If you do not see the record you want to view, make sure you have selected the Project that the record is associated with, or select All, and check your View and Filter options.

To view a record's Detail Form

  • In a module list view in Read Only mode, click the record number or name or double-click any field of the record that is not a link.
    -or-
    In a module list view in Editable mode, double-click any field of the record.
    -or-
    In a module list view in either mode, click a field that is not a link, then click  Details. The Detail Form for that record appears.
