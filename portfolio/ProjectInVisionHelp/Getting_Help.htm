Getting Help

The help system is accessed from the main Project InVision window and is indexed so you can search for keywords or phrases. The online help is an extension of the printed User Guide, so items in the Guide appear exactly as seen online.

To view the Project InVision User Guide Online

  • Click Help, then Project InVision User Guide Online.

Workgroup Help is specific to your company if you have custom Workgroups or Modules. Information about how to use these custom features is in Workgroup Help.

To view Workgroup Help

  • Click Help, then Workgroup Help.

Project InVision Workgroup Help provides Project Management policies specific to the Workgroup you are assigned to. It contains specific information regarding how Project InVision was configured to meet the specific requirements of your Workgroup. The Workgroup Administrator, using the Project InVision Administration application, configures the Workgroup Help. If no Workgroup Help appears in the Help menu, there is no customized Workgroup Help available to you and you should refer to the Project InVision User Guide.

To view the Project InVision version

  • Click Help, then About Project InVision.
