Adding Project InVision to Your Favorites

The Favorites list in Internet Explorer is a set of saved URLs, or web page addresses. You can save the URL for Project InVision so you don't have to enter it manually each time.

To add Project InVision to your Favorites

  1. Start Internet Explorer and navigate to Project InVision. (See Logging In to Project InVision.)
  2. In the Internet Explorer Favorites menu, select Add to Favorites. The Add Favorite dialog box appears.
    In the Name field of the Add Favorite dialog box, you should see “Project InVision Login.”
  3. Click OK.
