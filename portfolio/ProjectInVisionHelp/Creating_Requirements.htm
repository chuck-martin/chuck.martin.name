Creating Requirements

When you're defining requirements for a Project, each one should be added to the Project InVision Requirements module. Requirements can be associated with either the overall Project or with specific Tasks in the Project. If a Requirement is associated with a specific Task, then the Requirement is displayed in the Time Tracking application so that time can be posted against the Requirement and rolled up into the Project Task. See Time Tracking.

To create a Requirement

  1. In a Projects list, select a Project.
  2. Under Modules, click Requirements.
  3. Click  New. The New Requirements Detail Form appears.
  4. Enter the Requirement information. Fields with bold labels are required.
  5. Click Submit.
