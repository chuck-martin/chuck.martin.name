Deleting Document Records

Document records can be deleted from the Documents module if, for example, one was created by mistake. However, closed Document records should remain in the Documents module so when a Project is archived by the Workgroup Administrator, all associated Document records are archived with the Project.

You can delete Document records only if you have the proper security.

To delete a Document Record

  1. In a Documents module list view, select a Document record.
  2. Click  Delete.
  3. Click Yes (to confirm deletion).

Deleting a Document record does not remove the associated document file from the Project InVision server. See the Workgroup Administrator for information on recovering associated document files for deleted Document records.
