Requirements Module Reports

Requirements module Reports contain information related to Requirements and the Status of Requirements across the Enterprise. The Reports available for the Requirements module are described below (with Report Options), along with a description of the Report and the fields that are captured.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some fields may not exist or may be labeled differently than shown here.

Requirements List - Detail (Projects, Requirements Filter)

This Report shows detailed information about Requirements, grouped by Project. Project fields: Project Number, Project Title, Requirement Number, Requirement Title, Responsible, Submitted by, Date Entered, Evaluation Date, Date Closed, Status, Priority, Complexity, Type, Description, Business Case, and Impact.

Requirements List - Status (Projects, Requirements Filter)

This Report shows all the Requirements for selected Project, grouped by Project. Project fields: Project Number, Project Title, Requirement Number, Requirement Title, Responsible, Submitted by, Status, Type, Priority, Complexity, Date Entered, Evaluation Date, and Date Closed.

Requirements Count (Projects, Requirements Filter, Rollup)

This Report shows a total count of Requirements by Rollup. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project.

Requirements Count - My Requirements (Projects, Requirements Filter)

This Report shows both your Requirements and a total count of Requirements separated into groups by Rollup. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project.

Requirements Count Pie (Projects, Requirements Filter, Rollup)

This Report shows a Pie Chart with Requirements by Rollup, plus a table with detailed Requirements grouped by Rollup. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project. Project fields: Requirement Number, Requirement Title, Status, and Count.

Requirements Pie Chart - Actual Work (Projects, Requirements Filter, Projects Filter, Rollup, Date Range, From Date, To Date)

This Report shows a Pie Chart with Work Hours by Rollup, plus a table with detailed Work Hours grouped by Rollup. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project. Project field: Requirement Number, Requirement Title, Status, Actual Work, and Timesheet Date.

Requirement Matrix (Projects, Requirements Filter)

This Report shows a matrix that compares Requirement Complexity to Requirement Priority, comparing High, Medium, and Low for both.
