Sending Alerts

You can send Alerts directly to other users within Project InVision and via email to anyone with a valid email address.

To send an alert

  1. In the Output menu in the toolbar, choose Send Message. The Send Message dialog box appears.
  2. Select how you want the Alert sent.
    • If you select Alert, you can send to one or more existing Project InVision users.
    • If you select Email, you can send to no more than one external email address.
    • If you select both, you can send to one external email address and one or more existing Project InVision users.
  3. Enter the Alert recipient(s).
    • For existing Project InVision users:
      1. Click a field in the Resource column.
      2. Click the ellipsis button. The Find dialog box appears.
      3. Select a name from the list in the Find dialog box.
      4. Click Select.
        The name will be entered and a new, empty row will appear. Repeat for multiple users.
    • For an external email address:
      1. Click in the bottom empty field in the Email Address column,.
      2. Type a valid email address.
  4. Enter a Subject and Message.
  5. Click Submit.
