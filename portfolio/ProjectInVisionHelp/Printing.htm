Printing

While Project InVision provides a number of formatted reports, you can print a spreadsheet report of the records you're currently viewing.

To print records

  1. Apply a view.
  2. Click Output, then choose Print. A Print Preview window appears.
  3. Click  Print. The Print dialog box appears.
  4. Click Print.

Only the displayed information is printed. For example, if you're viewing 50 records in a list view, then only 50 records are printed, even though more records might be available. You can either click More to add more records to the view, or click All to view all records (may take some time to display), then print. The bottom right of the status bar shows how many records are being viewed and how many total records are available.
