Training and Certification

Hands-on training services and certification courses are conducted by Project InVision Implementation Consultants, either at the customer site or at Project InVision International training facilities.

Training Services

Project InVision User Training
This course provides in-depth training in how to use the basic functionality of Project InVision. All participants of Project InVision User Training receive the Project InVision User Training Guide.

Project InVision Administration Training
This training provides the basics for configuring and maintaining the Project InVision application. All participants of Project InVision Administration Training receive the Project InVision Administration Training Guide.

Project InVision Time Tracking Training
This training course covers using the Time Tracking application to post work hours in weekly Timesheets. All participants of Project InVision Time Tracking Training receive the Project InVision User Training Guide, which contains sections that cover Time Tracking.

Certification Courses

Project InVision Report Writer
This in-depth course is designed for person(s) expected to be able to create Crystal Reports, and incorporate them into the Project InVision application. This course is lessons-based with administrative exercises, guides, and tips, and an exam at the end of the session. Upon passing the examination, report writers will be certified by Project InVision. Participants of the Report Writer Certification training will receive the Project InVision Report Writer Workbook.

Project InVision System Administrator
This training is conducted by Project InVision Implementation Consultants, either at the customer site or in the training facilities at Project InVision International Headquarters. This in-depth course is designed for person(s) expected to provide System Installation and Administration services for the Project InVision application. This course is lessons-based with administrative exercises, guides, and tips, and an exam at the end of the session. Upon passing the examination, System Administrators will be certified by Project InVision. Under a current maintenance agreement, Certified System Administrators will have access to Project InVision Help Desk, at no additional cost for questions that may arise specific to System Administration.

Project InVision Trainer
This class is designed for individuals who will be expected to train other users of Project InVision. This in-depth course is lessons-based with training exercises, guides, and tips, and an exam at the end of the session. Upon passing the examination, trainers will be certified by Project InVision. Participants of the Trainer Certification will receive the Project InVision User Training Guide with rights to customize and reproduce it. Under a current maintenance agreement, certified Trainers will have access to Project InVision Help Desk, at no additional cost, for questions that may arise, or application support that may be required, during their own training sessions.

Project InVision Workgroup Administrator
This in-depth course is designed for person(s) expected to provide Workgroup Administration for the Project InVision installation., including lessons in configuration of the Project InVision application, and maintenance of Projects and files within the application. This course is lessons-based with administrative exercises, guides, and tips, and an exam at the end of the session. Upon passing the examination, Workgroup Administrators will be certified by Project InVision. Participants of the Workgroup Administrator Certification training will receive the Project InVision Administrator Training Guide. Under a current maintenance agreement, certified Workgroup Administrators will have access to Project InVision Help Desk, at no additional cost, for questions that may arise specific to Workgroup Administration.
