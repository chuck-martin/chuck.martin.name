Creating Resource Plans

Before Resource Analysis Reports can be used, a Resource Plan must be created for the selected Project.

To create a Resource Plan

  1. In a Projects or Resources module list view, click  Tools, then Resource Plan.
    -or-
    In a Project Details Form, click  Resource Plan.
    In the Tasks or Resources module, you must be viewing a single Project, not All.
    The Resource Plan dialog box appears with all Resources assigned to the Project listed.
  2. If no Resource Plans previously existed for the Project, skip to step 3.
    • To create a new blank plan, click  New.
    • To create a plan based on existing plan information, select an existing plan name from the Plan drop-down list, then click  Copy.
  3. In the Name field, enter a name for the plan.
  4. In the Scale drop-down list, select a period (Daily, Weekly, Monthly, or Quarterly) to display Resource time.
  5. Enter the plan Start Date and End Date.
    In each date field, click  Calendar, then click on a date. (Click the arrows at the end of the scroll bar to move by month. Click in the scroll bar to move by year.)
  6. For each named or generic Resource, enter the number of that resource needed for each time period in full-time equivalents.
    For example, entering a 1 indicates that resource is needed full time for the time period, entering a 0.5 indicates that resource is needed half time for the time period, and entering a 3 means that three of those Resources are needed for the time period.
  7. Optionally, select Current Plan.
    The Resource Plan marked as Current is the plan used for all Resource Planning Analysis Reports. Only one Resource Plan per Project may be marked as the Current plan.
  8. Optionally, select Baseline Plan.
    Only one Resource Plan per Project may be marked as the Baseline plan.
  9. Click  Save.

To add Resources to the Project for requirements planning, click Define Project Team. See Assigning Resources to Projects.
