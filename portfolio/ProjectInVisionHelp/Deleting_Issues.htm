Deleting Issues

Issues can be deleted from the Issues module if, for example, one was created by mistake. However, closed Issue records should remain in the Issues module so the actions taken to resolve the Issue remain documented. When a Project is archived by the Workgroup Administrator, all associated Issue records are archived with the Project.

You can delete Issues only if you have the proper Workgroup permissions.

To delete an Issue

  1. In an Issues module list view, select an Issue record.
  2. Click  Delete.
  3. Click Yes (to confirm deletion).
