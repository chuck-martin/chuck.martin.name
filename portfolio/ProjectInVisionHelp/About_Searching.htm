About Searching

You can search for records in Project InVision by entering one or more search words and clicking Go. Your Workgroup Administrator configures your search capability to define which modules you can search in.

When you perform a search, the results will show a list of each module where the search term(s) are found with a link to display detailed results in each module.

To search for records

  • Type one or more words in the Search box, then click Go.

Click Last Search to automatically enter the last search into the Search box.
