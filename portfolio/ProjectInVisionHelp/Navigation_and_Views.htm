Navigation and Views

Navigation and choosing what and how information is displayed within Portfolio Manager is done by clicking on links and by selecting choices from the Navigation menu. The Navigation menu also acts as "breadcrumbs," showing you where you are within Portfolio Manager and what views, filters, and groupings you've applied.

Information in Portfolio Manager is displayed in many ways. Common views are records in a spreadsheet-style list, but information can be displayed also in charts, graphs, and reports. List views can also be filtered and grouped. Views, filters, and groups are all selected in the Navigation menu.
