Deleting Custom Module Records

Custom module records can be deleted from the custom module if, for example, one was created by mistake. However, closed custom module records should remain in the custom module so when a Project is archived by the Workgroup Administrator, all associated custom module records are archived with the Project.

You can delete records only if you have the proper security.

To delete a custom module record

  1. In a custom module list, select a record.
  2. Click  Delete.
  3. Click Yes (to confirm deletion).
