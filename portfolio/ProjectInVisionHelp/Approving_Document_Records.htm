Approving Document Records

Approvals may be required before a Document can pass from one Status to the next. Approvals are set up by the Workgroup Administrator based on the internal processes for approving Documents (Workflows) and defines who has the authority to approve these Documents.

Only users with the appropriate security are able to approve Documents. In addition, only the Resource identified in the Approver column is able to approve their own record line.

To approve a Document record

  1. View a Document record.
  2. Under Pages, click Approvals.
    If you are not responsible for approving the Document record at this stage of the Project workflow, you will not see your name in the Approval list.
  3. Select the check box in the Approved field for the line that contains your name.
  4. Optionally, add a note by clicking on the Note field, clicking the ellipsis button, entering the note text in the Notes window, then clicking OK.
  5. Click  Save.
