Quick Sorts

In addition to sorting module lists by a single field (by clicking on the column header), you can quickly sort on one or more fields using the Quick Sort function.

To create a Quick Sort

  1. Right-click on any record in the column you want sorted, then choose Create Quick Sort. The Quick Sort dialog box appears.
  2. Select a Sort Order: Ascending (default) or Descending.
  3. Select a sort option:
    • Select Apply New Quick Sort to filter (default) to remove all existing sorts and use just the selected single sort.
    • Select Append to current sort to add the selected sort as the next sort to any existing sorts.
  4. Click Submit.
