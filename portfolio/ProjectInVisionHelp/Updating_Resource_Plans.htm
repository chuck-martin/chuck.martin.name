Updating Resource Plans

To create “What If?” scenarios, existing Resource Plans must be updated to reflect the different scenarios. This synchronizes the Resource Plan with the Resource Assignments that exist in the project schedule file or in the Project InVision Tasks module, including Costing information.

This function can be set to synchronize automatically or to allow you to update the plan as you desire. See Configuring Project Options.

To update a Resource Plan from Task Assignments

  1. In a Projects or Resources module list view, click  Tools, then Resource Plan.
    In the Resources module, you must be viewing Resource records for only a single Project.
    -or-
    In a Project Details Form, click  Resource Plan.
    The Resource Plan dialog box appears with all Resources assigned to the Project listed.
  2. In the Select Plan drop-down list box, select a Plan.
  3. Click Update from Assignments.
    The Resource Plan will be automatically calculated, synchronizing the Resource Plan with the Task assignments from the associated project schedule file or in the Project InVision Tasks module.
  4. Click Submit.
