About Reports

Reports summarize and organize information in the various modules. You can specify report options for running reports, including the filter to be applied, the Workgroup(s) involved, and the date range to be applied.

Provided that the you have appropriate security, reports can be run for your current Workgroup, all Workgroups where you're a member, or all Workgroups in the Enterprise.

Field names listed here may not match the field names in each Workgroup because the Workgroup is customizable. For Workgroups that have been configured differently from the Default Workgroup, the field names that will appear on the Reports will match the field names configured in the Workgroup. For example, if the Sponsor field has been changed to Executive, then the label Executive will appear on the report for that Workgroup.

The Reports that can be run are:

  • Projects Module Reports
  • Tasks Module Reports
  • Resources Module Reports
  • Documents Module Reports
  • Issues Module Reports
  • Defects Module Reports
  • Journals Module Reports
  • Risks Module Reports
  • Changes Module Reports
  • Requirements Module Reports
  • Time Tracking Reports
