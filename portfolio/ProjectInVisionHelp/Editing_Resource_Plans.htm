Editing Resource Plans

To edit a Resource Plan

  1. In a Projects or Resources module list view, click  Tools, then Resource Plan.
    In the Resources module, you must be viewing Resource records for only a single Project.
    -or-
    In a Project Details Form, click  Resource Plan.
    The Resource Plan dialog box appears with all Resources assigned to the Project listed.
  2. In the Select Plan drop-down list box, select a Plan.
  3. Make any desired changes:
    1. In the Name field, you can change the plan's name.
    2. In the Scale drop-down list, select a period (Daily, Weekly, Monthly, or Quarterly) to display Resource time.
    3. You can change the plan Start Date and End Date.
      In each date field, click  Calendar, then click on a date. (Click the arrows at the end of the scroll bar to move by month. Click in the scroll bar to move by year.)
    4. For each named or generic resource, you can change the number of that resource needed for each time period in full-time equivalents.
      For example, entering a 1 indicates that resource is needed full time for the time period, entering a 0.5 indicates that resource is needed half time for the time period, and entering a 3 means that three of those Resources are needed for the time period.
    5. Optionally select or clear the Current Plan check box.
      The Resource Plan marked as Current is the plan used for all Resource Planning Analysis Reports. Only one Resource Plan per Project may be marked as the Current plan.
    6. Optionally select or clear the Baseline Plan check box.
      Only one Resource Plan per Project may be marked as the Baseline plan.
  4. Click  Save.
