Detail Form Common Pages

Most Detail Forms have pages that are common to all modules. For information on pages specific to a module's Detail Form, see: Projects, Tasks, Resources, Documents, Issues, Defects, Journals, Risks, Changes, Requirements, Collaborations, and Custom Modules.

When entering information on Detail Form pages, the information does not have to be saved on each page before switching to another page. Click Submit when you're finished to save all your changes.

Comments

The Comments page contains free-form text. For the Projects and Tasks modules, this information is transferred to the Project Information Comments and Task Notes sections, respectively, of any associated project schedule file. See Adding Comments and Editing Comments.

Keywords

Workgroup Administrators can define Keywords for the workgroup, values that can be used to filter the records in a module. You can assign one or more Keywords to records.

To define a Keyword

  • In the Keywords tab, click on an empty line in the Keywords list, then select a Keyword.

Links

You can create links to records in any module so users can jump from one record directly to associated data.

To define a Link

  1. In the Links page, click on an empty line in the Item list, then select a type of link.
  2. If you selected URL as the type of link:
    1. Click in the corresponding field in the Link column, then click the down-arrow button. The URL dialog box appears.
    2. Enter the URL label (how the link will appear in Project InVision) and the URL (the full web address).
    3. Click Submit.
    If you selected any other type of link:
    1. Click in the corresponding field in the Link column, the click the down arrow button.
    2. If you want to define a link to an existing record:
      1. Choose Select from Existing Items. The Find dialog box appears.
      2. Select a record from the list, then click Select.
      3. Click  Save.
      If you want to define a link to a new item:
      1. Choose Create New Item. The New Detail Form for that type of record appears.
      2. Enter the information for the new record, then click Submit.
      3. Click  Save.
  3. Click  Save.

Discussions

Discussions are threaded messages related to a record. You can create new Discussion items, respond to a sender of a Discussion item, or respond to a group of users that are involved in the Discussion. See About Discussions.
