Configuring Search Settings

You can configure Project InVision to limit your search results. You can limit the modules and the types of records searches are performed in and define what views the results are presented in.

To configure search settings

  1. Click My Setup.
  2. Under Select Item, click Search Settings.
  3. In the Display column, select the check boxes for the modules searches will run in and clear the check boxes for the modules searches will avoid.
  4. In the View column, select which view each module will use when any search results are presented from that module.
  5. In the Filter column, select a pre-defined filter to determine which records in each module will be searched.
  6. Click  Save.
  7. Click Submit.
