Editing Template Information

The Template information is metadata, the Title, Type, Description, and other information about the Template, and not the Template file itself.

Editing template information can be done only by users with access to Administrative functions.

To edit Template information

  1. Click Administration. The Project InVision Administration window appears.
    The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.
  2. Under Select Screen, click Templates.
  3. In the Navigation menu, select the Workgroup and the Module where the template is located
  4. Select the template, then click  Details.
    -or-
    Double-click the Template name. The Template dialog box appears.
  5. Make any desired changes.
    • Template Title: A short, useful description of up to 50 characters. This is what users will see what selecting templates.
    • Template Type: A category used for sorting.
    • File Extension: The 3-character extension of the file type used for the Template.
    • Description: A more comprehensive description of the Template, including suggested uses, Workflows, etc.
    The File Name field is unavailable because you must edit the Template file separately.
  6. Click Submit.
