About Portfolio Manager

The Portfolio Manager application contains and organizes all the information about your projects and is part of the Project InVision suite. You can use Portfolio Manager to manage a portfolio of projects for a program, a department, or the whole enterprise.

Portfolio Manager contains several modules, which are used to capture information related to projects. Standard modules include Resources, Tasks, Issues, Defects, and more. For more information, see Modules.

Portfolio Manager also includes Portals, which are cross-module views of important information. For example, standard Portals include My InVision, which contains snapshot views of projects you're responsible for, and Project Portal, which contains high-level status of a project or program.

Working in Portfolio Manager

Portfolio Manager contains a main section for displaying information and controls for changing the display of that information and for switching between modules in the Portfolio Manager application. The Application menu in the top-left section of the page (next to the Project InVision logo) controls what application you are in: Portfolio Manager, Time Tracking, or Dashboard. Links in the upper-right are common to all applications and provide access to personalization, messaging, and help.

When you log in to Project InVision, you view all the modules within a specific Workgroup. Module records are often viewed in lists, which share common functionality such as views, filtering, and grouping, as well as charts and graphs. What you can see and do in modules is determined by your security and user permissions, which are set by the Workgroup Administrator.

The Module links on the left duplicate the selections in the Modules menu in the Navigation Menu, allowing one-click access to different modules. The Navigation Menu across the top of the page controls what you see and how you see it. Underneath the Navigation Menu is the toolbar, which contains commands for Project InVision functions. On the left end of the toolbar is the button for the Selector, which toggles a hierarchical list of projects.

The information displayed is determined by the Module that has been selected, the View that has been applied, plus any Filter or Group By selections that have been made. List view information is presented in a spreadsheet format, with rows representing single records and columns representing data for each record.

Module List Modes

Module lists have two modes: Read Only and Editable. In Read Only mode, many fields are links. Clicking on the links will open the Detail Form for that record. In Editable mode, you can click in the field and edit the information directly. See Editing Information.

  • Double-click on a non-link field in any record in a list view to open its associated Detail Form.
  • Click a link field in any Record in a list view in Read Only view to open the Detail Form for that field.
  • Single-click in any column of a record in a list view to edit the specific field (if the View is editable).
  • Click the vertical scroll bar on the right to scroll up and down in a list view.
  • Click the horizontal scroll bar to view more data columns.
  • Click More or All to see more records in a list view.
  • Click and drag on the right border of a column header to change the column width.
  • Click on the column header and drag it to the desired location to rearrange the column order.
  • Click and drag on the bottom border of a row header to change all row heights.

Changes to column width, column order, and row height are reset to their default values when you change views, apply a filter, or refresh the list.
