Selecting Multiple Projects or Groups

Selections in the Selector control what's seen in the window. You can select multiple Projects and groups of Projects to see only the records associated with your selection.

Projects have the Project Number next to the name and do not have any records lower in the hierarchy attached to them. Groups have no number next to the name and have one or more Projects or groups “underneath” them in the hierarchy.

To select multiple Projects and groups

  1. If it is not open, click  Selector to view the Selector pane.
  2. Select one or more check boxes along the left side of the pane to select Projects or groups.
    • Selecting the check box for a group selects the check boxes for all groups and projects within (or “underneath” in the hierarchy) that group.
    • Clearing a selected check box for a record within a selected group clears the check box for the group, but leaves the check boxes for all other records in that group selected.
    Clicking on a Project or group name will clear all check boxes except for that Project or group.
  3. Click Go.

If you do not see a Project or group name in the list, click More or All at the bottom of the pane to see more Projects.
