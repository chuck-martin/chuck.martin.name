Editing Templates

Workgroup or Workflow changes may require that the template files used change as well. To edit template files, the attached files must be checked out, saved to your computer, changed there, then checked back in.

Editing a template file (the file attached to the template) is different from editing the template information.

Editing templates can be done only by users with access to Administrative functions.

To edit a Template file

  1. Click Administration. The Project InVision Administration window appears.
    The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.
  2. Under Select Screen, click Templates.
  3. In the Navigation menu, select the Workgroup and the module where the template is located
  4. Select a template that is checked in (look in the Status field), then click  Check Out.
    The file opens either in a separate Internet Explorer window or (if Internet Explorer cannot open the file) the application you normally use to open the type of file.
  5. From the File menu in that window, select Save As.
  6. Navigate to a folder on your computer and save the file there.
  7. Make any desired changes to the file, then save it again.
  8. Close the separate Internet Explorer or the application window.
  9. Make sure the template is still selected, in the Templates list, then click  Check In. The Check In dialog box appears.
  10. Click Browse. The Choose File dialog box appears.
  11. Find the folder where you saved the file, select the file, then click Open.
  12. Click Check In.
