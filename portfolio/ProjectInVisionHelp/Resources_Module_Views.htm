Resources Module Views

Default Views have been created for the Resources module. Your security access determines what Views you can see.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some View names or fields may not exist or may be labeled differently than shown here.

-Standard-

This View shows general information for Resources. You can view and edit information related to Resource information, such as Address, Department, Region, or Title. Fields: Last Name, First Name, Title, Address, Department, Level, Manager, Region, Title, and Comments.

Actual by Billing Code

This View show information about individual Billing Codes, the Projects in which they have been used, and the individual Resources who have been assigned the Work, grouped by Billing Code and then by Project. Fields: Billing Code, Project Number, Project Title, Resource, and Actual Hours. The Hours are then summarized by Billing Code Total and Grand Total.

Actual by Cost Center

This View shows information about individual Cost Centers, the Projects in which they have been used, and the individual Resources who have been assigned the Work, grouped by Cost Center and then by Project. Fields: Cost Center, Project Number, Project Title, Resource, and Actual Hours. The Hours are then summarized by Cost Center Total and Grand Total.

Actual by Resource

This View shows information about individual Resources and the Projects to which they have been assigned the Work, grouped by Resource and then by Project. Fields: Resource, Workgroup, Project Number, Project Title, and Actual Hours. The Hours are then summarized by Resource Total and Grand Total.

Actual by Workgroup

This View shows information about individual Workgroups, the Projects contained therein, and the individual Resources who have been assigned the Work, grouped by Workgroup and then by Project. Assigned Resources are identified along with the Actual Work posted to the Task through Time Tracking. Fields: Workgroup, Project Number, Project Title, Resource, and Actual Hours. The hours are then summarized by Workgroup Total and Grand Total.

Actual Work Distribution

This View shows information about the Work performed by individuals in a Project. The data is based on the Resources and Projects Filters applied, as well as the Rollup, Grouping, and Date Ranges specified.

Actual Work Pie Chart

This View is a Pie Chart with slices based on Rollup selection and the Date Range specified. Depending on the Rollup used, the Work Hours will display for the specified Date Range, grouped by Rollup. The Rollup can be Resource, Skill Category, Department, or Region. Under the Pie chart is a list with information details. Fields: Project Number, Resource Name, Resource Number, Actual Work, Billing Rate, and Actual Cost.

Analysis Detailed

This View shows information about Resource requirements on a Project-by-Project basis. Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region. The resulting numbers can be represented as a percentage, as # of Resources, or Work hours.

Analysis Overview

This View shows summary information about Resource availability. Depending on the Scale, Rollup, Representation, and Date Range applied to the data, you can see how much of a Resource's time is available, how much is required, and how much is a surplus or a shortage. Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region.

Analysis Requirement

This View shows summary information about Resource requirements, represented in terms of Work Hours to which the Resource has currently been assigned. Depending on the Scale, Rollup, Representation, and Date Range applied to the data, you can see how much of a Resource's time is required. As Resources are assigned to more Tasks, the Required Hours are increased until the numbers appear in red (indicating a shortage) or over-allocation of that Resource (more required hours than the Resource can accomplish in the time frame specified). Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region.

Analysis Surplus / Shortage

This View shows summary information about Resource availability, represented in terms of Work Hours available for Task assignments. Depending on the Scale, Rollup, Representation, and Date Range applied to the data, you can see how much of a Resource's time is available. As Resources are assigned to more Tasks, the available Hours are reduced until the numbers appear in red (indicating a shortage) or over-allocation of that Resource (more required hours than the Resource can accomplish in the time frame specified). Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region.

Assignments

This View shows basic information about Resources assigned to Projects, including their Department, Region, and Manager.

Assignments Detail

This View shows basic information about Resources assigned to Projects, including their Department, Region, and Manager. Fields: Last Name, First Name, Middle Initial, Resource Number, Title, Skill Category, Department, Region, Manager, Active, and Comments.

Curve Chart - Actuals

This View shows a graphical representation of Actual Work over time for the items selected in the View.

Curve Chart - Cost

This View shows a graphical representation of Actual Cost over time for the items selected in the View.

Curve Chart - Work

This View shows a graphical representation of Current Work over time for the items selected in the View.

Gantt Chart

This View shows a graphical representation of the duration of Tasks assigned to the Resource for Tasks belonging to a single Project or all Projects in the Workgroup. Fields: Resource Name, Project Name, Task Name, a bar showing the duration of the Task, and a triangle indicating a Milestone.

Histogram

This View shows a graphical representation of Resource availability. When selecting the View, the scale can be set to Daily, Weekly, Monthly, or Quarterly. Numbers can then be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region and represented as a percentage, as # of Resources, or Work Hours. Availability is represented in yellow, Allocation is represented in blue, and Over-Allocation is represented in red.

Phone List

This View shows Resource contact information. Fields: First Name, Last Name, Phone Number, Email Address, Resource Title, and Address.

Project Calendar

This View shows information about Resource availability on a Project-by-Project basis. When selecting the View, the scale can be set to Daily, Weekly, Monthly, or Quarterly. Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region, then be represented as a percentage, as # of Resources, or Work Hours. An availability graph shows white for Available and blue for Allocated.

SME Chart

This View shows summary about regarding Resource availability. When selecting the View, the scale can be set to Daily, Weekly, Monthly, or Quarterly. Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region. The data are shown graphically as colors: teal for Under-utilized, yellow for Even Allocation, and red for Over-Allocated.

Work Distribution

This View shows information about the Work performed on Projects, the Current or Baseline Hours for the specified Date Range, grouped by Rollup. When selecting the View, the Rollup can be set at Resource, Workgroup, Project, Billing Code, or Cost Center. Numbers can be grouped by Project, Resource, Billing Code, or Cost Center and presented as Current Work or Baseline Work. Fields: Workgroup, Resource, Resource Name, and Current/Baseline Hours.

Work Pie Chart

This View is a Pie Chart with slices based on Rollup selection and the Date Range specified. Depending on the Rollup used, the Work Hours will display for the specified Date Range, grouped by Rollup. The Rollup can be Resource, Skill Category, Department, or Region. Fields: Project Number, Project Title, Resource Name, Current Work, Baseline Work, Billing Rate, Current Cost, and Baseline Cost.

Work Summary

This View shows information about Work performed on Projects based on Rollup selection and the Date Range specified. Depending on the Rollup used, the Current or Baseline Hours display for the specified Date Range. When selecting the View, the Rollup can be set at Resource, Workgroup, Project, Billing Code, or Cost Center. The Content can be presented as Current Work or Baseline Work. Fields: Workgroup, and Current or Baseline Hours.
