Changing Dashboard Layouts

Each Dashboard has a default configuration set by the Workgroup Administrator that defines the Views included and how they are arranged. For each Dashboard, you can change the Views you see and their arrangement.

To change a Dashboard layout

  1. Click  Layout. The Dashboard Layout window appears.
    • To change the number of columns of views, select 1, 2, or 3.
    • To remove a view from the Dashboard layout, click the view in the layout area on the right to select it, then click Remove.
    • To add a view to the Dashboard layout, click the view in the Available Views list to select it, then click Add.
    • To reposition a view, click the view to select it, then click Left, Right, Up, or Down.
    • To resize the height of a view, click the bottom border of the view and drag it to the size you want.
    The sample views rearrange automatically after each selection.
  2. Click Submit.
