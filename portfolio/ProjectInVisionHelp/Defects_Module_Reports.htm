Defects Module Reports

Defects module Reports contain information related to Defects and the Status of Defects across the Enterprise. The Reports available for the Defects module are described below (with Report Options), along with a description of the Report and the fields that are captured.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some fields may not exist or may be labeled differently than shown here.

Defects List - Detail (Projects, Defects Filter)

This Report shows detailed information about Defects, grouped by Project. Project fields: Project Number, Project Title, Defect Number, Defect Title, Submitted by, Responsible, Description, Resolution, Status, Priority, Discovery Date, Evaluation Date, and Completion Date.

Defects List - Status (Projects, Defects Filter)

This Report shows Defect Status, grouped by Project. Project fields: Project Number, Project Title, Defect Number, Defect Title, Submitted by, Responsible, Discovery Date, Evaluation Date, Completion Date, Status, and Priority.

Defects Count (Projects, Defects Filter, Rollup)

This Report shows Defects by Rollup. Numbers can be rolled up by Type, Priority, Severity, Status, Cause, Responsible, Submitted by, or Project.

Defects Count - My Defect (Projects, Defects Filter, Rollup)

This Report shows how many Defects you are responsible for and the total number of Defects, grouped by Rollup. Numbers can be rolled up by Type, Priority, Severity, Status, Cause, Responsible, Submitted by, or Project.

Defects Count Pie (Projects, Defects Filter, Rollup)

This Report shows a Pie Chart of bugs by Rollup selection and a table with details of the same selection. Numbers can be rolled up by Type, Priority, Severity, Status, Cause, Responsible, Submitted by, or Project. Project fields: Defect Number, Defect Title, Status, and Count.

Defects Pie Chart - Actual Work (Projects, Defects Filter, Projects Filter, Rollup, Date Range, From Date, To Date)

This Report shows a Pie Chart with Work Hours for the specified Date Range, grouped by Rollup and a table with details of the Work Hours. Numbers can be rolled up by Type, Priority, Severity, Status, Cause, Responsible, Submitted by, or Project. Project fields: Defect Number, Defect Title, Status, Actual Work, and Timesheet Date.

Defects Matrix (Projects, Defects Filter)

This Report shows a matrix that compares Defect Severity to Defect Priority, comparing High, Medium, and Low for both.
