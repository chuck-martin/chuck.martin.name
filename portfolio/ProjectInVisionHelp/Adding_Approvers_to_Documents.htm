Adding Approvers to Documents

If a Document needs to be approved by a Resource, but that approval is not part of the standard Workflow, you can add that approval requirement to the Document record. Adding an Approval requirement sends a Notification to the selected Resource with a message that they have Documents to approve.

To add an approver to a Document record

  1. View a Document Detail form.
  2. Under Pages, click Approvals.
  3. Click in a blank field in the Approver column, then in the drop-down list, select a Resource name.
  4. Click in the Status field on the same line, then in the drop-down list select the Document Status that requires this approval.
    The Document will not be permitted to reach this Status without this approval.
  5. Click  Save.

Once added as a approver, the Resource is able to access the Document record.
