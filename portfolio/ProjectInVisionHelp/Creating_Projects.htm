Creating Projects

Project records are used to track all the information about projects in Project InVision.

To create a Project

  1. In a Projects module view, Click  New. The New Projects Detail Form appears.
  2. Enter the Project information. Fields with bold labels are required.
  3. Click Submit.

Once a Project has been created, you can associate a project schedule file with it to capture Task information. Conversely, individual Tasks can be added to Project InVision using the Tasks module.
