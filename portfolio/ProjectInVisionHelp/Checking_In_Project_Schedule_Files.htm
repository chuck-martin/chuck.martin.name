Checking In Project Schedule Files

When a project schedule file has been checked out or has never been checked in, the check in process takes a copy of a file from your computer and uploads it to Project InVision. This file then becomes the master copy of the Project Schedule.

Checking in a project schedule file initiates the synchronization process.

To check in a project schedule file

  1. In a Projects list view, select any record.
  2. Click  Check In. The Check In dialog box appears.
    If the file is already checked in, you'll see the  Check Out button instead.
    You can also click Check In when you're viewing the Projects Detail Form.
  3. Click Browse.
  4. Find the file you want to check in, select it, then click Open.
  5. Click Check In.

A confirmation notice appears when the file has been uploaded successfully.
