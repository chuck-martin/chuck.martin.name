Changes Module Views

Default Views have been created for the Resources module. Your security access determines what Views you can see.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some View names or fields may not exist or may be labeled differently than shown here.

-Standard-

This View show all the information captured in the Change Detail Form. Fields: Change Number, Change Title, Responsible, Submitted by, Status, Type, Priority, Complexity, Date Entered, Evaluation Date, Date Closed, Description, Business Case, Impact, Project Number, and Project Name.

Change Actual Work Pie

This View is a Pie Chart with slices based on Rollup selection and the Date Range specified. Depending on the Rollup used, the Work Hours will display for the specified Date Range, grouped by Rollup. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project. Fields: Change Number, Change Title, Status, Actual Work, and Timesheet Date.

Change Count Pie

This View is a pie chart with pie slices based on Rollup selection. Depending on the rollup used, the counts will display grouped by the various values that exist for the Rollup field. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project.

Change Matrix

This View shows a matrix that compares Change Complexity to Change Priority. It separates the counts into High, Medium, and Low Severity versus High, Medium, and Low Priorities.

Change Trend Chart

This View shows how Changes have progressed over time.

Count Chart

This View shows a total count of Changes separated into groups by Rollup. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project.

Detail

This View shows detailed information about Changes that exist for selected Projects, grouped by Project. You can see information related to the Changes, as well as who is responsible for it, when it was evaluated, and the current status of the Change. Fields: Project Number, Project Title, Change Number, Change Title, Responsible, Submitted by, Date Entered, Evaluation Date, Date Closed, Status, Priority, Type, Description, Business Case, and Impact.

Personal Count Chart

This View shows a total count of Changes separated into groups by Rollup and displaying the number of Changes per Rollup group where the current user is the Responsible party. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project.
