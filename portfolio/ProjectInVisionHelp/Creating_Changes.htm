Creating Changes

Change requests that arise after the Project design is finalized can be tracked in the Project InVision Changes module. Changes can be associated with either the overall Project or with specific Tasks in the Project. If a Change is associated with a specific Task, then the Change is displayed in the Time Tracking application so that time can be posted against the Change and rolled up into the Project Task. See Time Tracking.

To create a Change

  1. In the Projects list, select a Project.
  2. Under Modules, click Changes.
  3. Click  New. The New Changes Detail Form appears.
  4. Enter the Change information. Fields with bold labels are required.
  5. Click Submit.
