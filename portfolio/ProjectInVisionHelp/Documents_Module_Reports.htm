Documents Module Reports

Documents module Reports contain information related to Documents and their status across the Enterprise. The Reports available for the Documents module are described below (with Report Options), along with a description of the Report and the fields that are captured.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some fields may not exist or may be labeled differently than shown here.

Documents List - Details (Projects, Documents Filter)

This Report shows detailed information about Documents, grouped by Project. The details include the Type and Status of the documents, whether they are currently Checked Out, and if so, by whom, and the Status of the Document. Project fields: Project Number, Project Title, Document Number, Document Title, Author, Revised Date, Status, Type, Association, Checked Out, Checked Out by, and Date Checked Out.

Documents List - Status (Projects, Documents Filter)

This Report shows Status and Association information about Documents, grouped by Project, as well as details about the Documents the associated file name, and the Status of the Document. Project fields: Project Number, Project Title, Document Number, Document Title, Author, Revised Date, Status, Type, and Association.
