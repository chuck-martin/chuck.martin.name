Keeping Project Schedule File Versions

Project InVision maintains several project schedule file versions, up to a maximum number defined by your Administrator, before deleting the oldest existing file. If you want a Project schedule version to be kept indefinitely, you must mark the version as Kept,” which means that it is never automatically deleted by Project InVision.

To keep a project schedule file version

  1. View a Project Detail form that has an attached Project Schedule file.
  2. Under Pages, click Versions.
  3. For the version you want to keep, select the Keep check box.
  4. Optionally, click in the Label field and enter information about why the version is being kept.
  5. Click Submit.
