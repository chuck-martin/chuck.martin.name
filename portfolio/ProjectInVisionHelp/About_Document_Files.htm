About Document Files

Document files are files that contain information about your project and that can be associated with Document records in Project InVision. Document files can be text files or binary files, can be created by any application, and are stored in Document records, one file per record.

You can check out associated document files and download the file to your workstation to make changes. Only one user can check out a file at a time. Once changes have been made to the checked out file, you can check in the file so that the changes are available to other users.

You can also review a document file, which allows you to download the file to your workstation to look at it, but not make changes to it.
