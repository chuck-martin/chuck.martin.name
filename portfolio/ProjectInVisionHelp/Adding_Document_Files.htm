Adding Document Files

Most Document records are associated with document files. These files can be of any type. You can add document files to Document records in two ways:

  • Adding an existing document by checking it in
  • Creating a new document from an existing template

To add an existing document

  1. View a Document record that does not have an attached document (the Association field in a Documents list view is empty).
  2. Click  Check In. The Check In dialog box appears.
  3. Click Browse.
  4. Navigate to the folder that contains the file you want to check in, select the file, then click Open.
  5. Click Check In.
