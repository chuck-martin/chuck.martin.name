Changes Module Reports

Changes module Reports contain information related to Changes and the Status of Changes across the Enterprise. The Reports available for the Changes module are described below (with Report Options), along with a description of the Report and the fields that are captured.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some fields may not exist or may be labeled differently than shown here.

Change Request List - Detail (Projects, Changes Filter)

This Report shows detailed information about Changes, grouped by Project. Project fields: Project Number, Project Title, Change Number, Change Title, Responsible, Submitted by, Date Entered, Evaluation Date, Date Closed, Status, Priority, Type, Description, Business Case, and Impact.

Change Request List - Status (Projects, Changes Filter)

This Report shows all Changes for selected Projects, grouped by Project,. Project fields: Project Number, Project Title, Change Number, Change Title, Responsible, Submitted by, Status, Type, Priority, Complexity, Date Entered, Date Evaluated, and Date Closed.

Change Requests Count (Projects, Changes Filter, Rollup)

This Report shows a total count of Changes by Rollup. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project.

Change Requests Count - My Change Requests (Projects, Changes Filter, Rollup)

This Report shows both your Changes and a total count of Changes by Rollup. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project.

Change Requests Count Pie (Projects, Changes Filter, Rollup)

This Report shows a Pie Chart of Change Requests by Rollup, plus a table with Change Request details grouped by Rollup. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project.

Change Requests Pie Chart - Actual Work (Projects, Changes Filter, Projects Filter, Rollup, Date Range, From Date, To Date)

This Report shows a Pie Chart of Work Hours by Rollup, plus a table with Work Hours detail grouped by Rollup. Numbers can be rolled up by Type, Priority, Complexity, Status, Responsible, Submitted by, or Project. Project fields: Change Number, Change Title, Status, Actual Work, and Timesheet Date.

Change Requests Matrix (Projects, Changes Filter)

This Report shows a matrix that compares Change Complexity to Change Priority, comparing High, Medium, and Low for both.
