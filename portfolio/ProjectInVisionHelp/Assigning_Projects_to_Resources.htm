Assigning Projects to Resources

You can assign one or more Projects to a Resource. This is useful when a new Resource is added to Project InVision and you want to assign multiple Projects to the Resource. Once Resources are assigned to Projects, they can be assigned to Task or Task-related item assignments.

To assign a Project to a Resource

  1. In a Project module list, select a Project record, then click  Team. The Define Project Team dialog box appears.
    You can also view a Project Detail form, then click  Team in the toolbar there.
  2. In the Assign drop-down list, select Projects to Resources.
  3. In the Resources drop-down list, select a Resource.
  4. Optionally select one or more options from the drop-down list boxes in the Define Criteria section to filter for project.
    Click the link in the Project # field to view the Detail Form for that Project.
  5. Select one or more Projects in the Projects Pool list
    Hold down the Shift key while clicking to select a contiguous group of names.
    Hold down the Ctrl key while clicking to select multiple discrete names.
  6. Optionally click a link under Check Availability to view reports that can detail Resource availability.
  7. Click Add Selected.
  8. Click Submit.

You can click Add All to add all Projects currently showing in the Projects Pool list.
