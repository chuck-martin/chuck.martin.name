Exporting Reports

You can export reports and save them in a variety of formats:

  • Crystal Reports (7 and 8)
  • Microsoft Excel
  • Microsoft Word
  • Rich Text Format
  • Adobe Acrobat

To export a report

  1. View the report you want to export. The report opens in a separate window.
  2. Click  Export Report. The Report Viewer Export dialog box appears.
  3. Navigate to the folder on your computer where you want to save the report file.
  4. In the Save as type drop-down list, select a file type.
  5. Enter a File name.
  6. Click Save.
