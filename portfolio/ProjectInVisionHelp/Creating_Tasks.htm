Creating Tasks

You can create Tasks in Project InVision two ways:

  • By entering them directly as Project InVision Task records (when a Project is not associated with a project schedule file)
  • By entering them in the project schedule file that is associated with a Project (the Tasks are added to the Project InVision Task module upon synchronization)

To create a Task

Tasks can be entered directly in the Tasks module only as part of Projects, and only when the Project is not associated with a project schedule file.

  1. In a Projects module list view, select a Project.
  2. Under Modules, click Tasks.
  3. Click  New. The New Tasks Detail Form appears.
  4. Enter the Task information. Fields with bold labels are required.
  5. Click Submit.

If a project schedule file is associated with the Project InVision Project, enter Tasks by checking out the project schedule file, adding the Tasks in the project schedule file, and checking in the edited file.
