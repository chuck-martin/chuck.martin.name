Project Indicator

The Project Indicator is the field on the left end of the toolbar that shows what Project is selected (if a single Project) or that multiple Projects are selected. No matter what module you're in, no matter what list or report you're viewing, you can see the Project that the data is referring to.

You can select any single Project to view. (To select multiple Projects, use the Selector.)

If the Selector is being used to select Projects and a full section of a hierarchy is selected, the Project Indicator shows the hierarchy level name. If multiple discrete Projects are selected, the Project Indicator shows “Multiple selections.”

To select a Project

  1. Click  Find next to the Project Indicator. The Find dialog box appears.
  2. Find the Project you want to select.
  3. Click Select.

The view will refresh when you select a Project and its name will appear in the Project Indicator.
