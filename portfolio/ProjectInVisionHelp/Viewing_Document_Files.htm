Viewing Document Files

You can view document files that are attached to Document records whether or not the files are checked out. Viewing a document file does not check it out. If a Document file is checked out by another user, it is not editable.

To view a document file

  1. View a Document record that has a Document file associated with it.
    -or-
    In a Documents list view, select a Document record with an associated document file.
  2. Click  Review.
  3. If you're asked whether to Open or Save the document, click Open.

If the document file is a type that can be opened in Internet Explorer (for example, a Microsoft Excel file or a .JPG image file), it will. Otherwise, it will open in its associated application.

In all cases, you must have the program that can view that type of file installed on your computer. (The exception is some types of files, such as HTML files or some graphic files, that Internet Explorer can open without a helper program being installed.) If you do not have the program installed, you'll see a list of programs to choose from. If you know that another program is capable of viewing that file type, and you see that program in the list, select it and click Open.

You can work with the file, make changes to the file, even save the file to your computer. You cannot check the file (and any changes you make) into Project InVision.
