Setting Resource User Categories

User Categories help you organize your Resources and also define what they can see and do in Project InVision.

Setting Resource User Categories can be done only by users with access to Administrative functions.

To set a User Category for a Resource

  1. Click Administration. The Project InVision Administration window appears.
    The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.
  2. Under Select Screen, click Resources.
  3. Click the name of a Resource with a Login.
    Do not click the Resource # link.
  4. Click in the User Category field of a Resource.
  5. In the drop-down list, select a User Category.
  6. Click  Save.
