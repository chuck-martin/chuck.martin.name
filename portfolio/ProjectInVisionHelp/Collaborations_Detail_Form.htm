Collaborations Detail Form

This form contains information about Project Collaborations. The form contains several pages, each containing a specific type of information. Click the links under Pages to view each page. The form also has its own toolbar that allows quick access to common functions.

Collaborations Detail Form Pages 

These are the default pages. Your Workgroup Administrator may have renamed existing pages or added new ones, or may have renamed, added, or deleted fields within any page.

Information

This page shows Collaboration-related information, such as the type of account, who is responsible for the Collaboration, and the Collaboration status.

  • Account defines an account type, which is used in searches.
  • Customer is the name of the person representing the customer in the Collaboration.
  • Contact is the main person on the Project responsible for the Collaboration.
  • Status defines steps within a workflow.
  • Status Date defines the date the Status was last changed.
  • User Defined Date can be used to track other dates relevant to the Collaboration.

Keywords

This page allows you to define keywords for the Collaboration. The keywords you can choose are defined by the Workgroup Administrator. See Detail Forms: Common Pages, Keywords.

Links

This page contains links to other records within Project InVision or to web addresses outside Project InVision. Links within Project InVision help you track associations between Projects and other items, such as Tasks and Resources. See Detail Forms: Common Pages, Links.

Discussions

This page contains threaded Discussions related to the Collaboration. You can create a Discussion item, respond to the sender of a Discussion item, or respond to the group of users that are involved in the Discussion. See Detail Forms: Common Pages, Discussions.
