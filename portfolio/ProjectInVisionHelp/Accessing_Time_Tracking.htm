Accessing Time Tracking

Time Tracking is a separate yet integrated Project InVision application. You can log in directly to Time Tracking or access it through the Application menu at any time from any other Project InVision application.

To access Time Tracking from another Project InVision application

  • Click on the down arrow next to the Application menu (in the top-left corner of the page, next to the Project InVision logo), then select Time Tracking.

You can also access Time Tracking directly from the log in page.
