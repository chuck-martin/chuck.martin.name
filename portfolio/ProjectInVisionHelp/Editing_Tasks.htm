Editing Tasks

Editing Tasks typically involves adding or deleting Resources or changing current, baseline, or actual dates. Tasks can be edited at any time if you have the correct Workgroup permissions or user security. If the Task is linked to a project schedule file, edits are synchronized with the Project file automatically so you don't have to check out the Project file.

If the Project has an associated project schedule file and that file is checked out, Task information cannot be edited.

To edit a Task

  1. View a Tasks Detail Form.
  2. Make any desired changes. Click the links under Pages to see more information.
  3. Click Submit.
