About the Defects Module

The Project InVision Defects Module is where you manage and collaborate on project or product-related Defects as they arise during the course of a project. A Defect is a problem detected in a product (such as a bug in a software product) that may affect the quality of the product being delivered. In the Defects Module, you track Defects from discovery through evaluation to resolution. You can view Defect information in list views or in the Defects Detail Form.

Defects can be associated with either a Project or with a specific Task in a Project. If a Defect is associated with a specific Task, then the Defect is displayed in the Time Tracking application so that time can be posted against the Defect and rolled up into the Project Task. See Time Tracking.
