Editing Skills

Skill information about a Resource identifies what that Resource can do and helps you find those Resources when you have to assign then to Tasks. The items in the Skills list are maintained by the Workgroup Administrator.

Skill information is on the Skills page when you view Resource Details

To add Skills

  1. View a Resources Detail form.
  2. Under Pages, click Skills.
  3. Click in an empty field in the Skills column, then select a Skill from the drop-down list.
  4. Click  Save.

To edit Skills

  1. View a Resources Detail form.
  2. Under Pages, click Skills.
  3. Click an existing skill in the Skills list, then select a different skill from the drop-down list.
  4. Click  Save.

To delete Skills

  1. View a Resources Detail form.
  2. Under Pages, click Skills.
  3. Click an existing skill in the Skills list.
  4. Press Delete.
  5. Click  Save.
