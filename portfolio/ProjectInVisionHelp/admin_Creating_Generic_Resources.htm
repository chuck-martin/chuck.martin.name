Creating Generic Resources

Generic Resources include Resource information such as collections of skills, location, and availability, but without a specific person attached. Generic Resources are used for planning purposes: you can create projects, define the resources you need, then see who can fill those resources. Finally, you replace the generic resources with the people who have the needed skills.

Creating generic Resources can be done only by users with access to Administrative functions.

To create a generic Resource

  1. Click Administration. The Project InVision Administration window appears.
    The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.
  2. Under Select Screen, click Resources.
  3. Click  New Resource. The New Resources window appears.
  4. Select Generic.
  5. Enter the Resource information. Fields with bold labels are required. Click the links under Pages to enter more information.
  6. Click Submit.
