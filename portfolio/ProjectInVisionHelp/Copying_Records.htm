Copying Records

You can make identical copies of any Project InVision record. This can be useful when a record is already configured, such as with a workflow or Resources, and you want to reuse that configuration as a template for a new record.

To copy a record

  1. In any list view, double-click the record name
    -or-
    Click the record name, then click  Details. The Detail Form for that record appears.
  2. Click Copy.
    A new, identical record is created, except that it has no ID #. You should change the Title.
  3. Click  Save.

The Summary and Key Deliverables pages of the Project record cannot be copied because this information comes from Task information. Additionally, the Resources page of the Project record and Assignments page of the Resource record cannot be copied because Resources and Task assignments should be assigned to the Project record on a Project-by-Project basis.
