Finding Information

A  Find or an  ellipsis button next to a field causes the Find dialog box to appear so you can search for a specific item. In some Detail forms, the Find button appears next to a drop-down list instead of a down-arrow if the items in the list are more than a defined amount, set by your Workgroup Administrator.

You can use the Find dialog box at any time, even if the field shows a down arrow, by holding down the Ctrl key when clicking on the down arrow.

In the Find dialog box, you can use either the Search or the Selector page. The Search page allows you to find items based on criteria you select. The Selector page allows you to find items by browsing through a hierarchy of all items.

To find items on the Search page

  1. Create a search phrase from two drop-down lists and the text box under Find Criteria.
    • The first drop-down list contains all the types of things you can search for.
    • The second drop-down list contains conditions or comparisons:
      < Less than the entered value.
      <= Less than or equal to the entered value.
      = Equal to the entered value.
      > Greater than the entered value.
      >= Greater than or equal to the entered value.
      Not Not the entered value.
      Starts With Starts with the entered text.
      Contains Contains the entered text.
      Ends With Ends with the entered text.
      Doesn't Start With Does not start with the entered text
      Doesn't Contain Does not contain the entered text.
      Doesn't End With Does not end with the entered text.
      Sounds Like Close phonetically. (For example, “Smith” would find “Smythe” and perhaps others.)
    • The text box is where you can type in one or more letters or numbers to narrow the search. Leave blank to get a list of all the items that have anything entered in what you selected in the first drop-down list. (For example, if you're searching for a person and you select email in the first drop-down list and leave the text box blank, the result will show everyone who has an email address entered, but will not include anyone who does not have an email address entered.)
  2. Click Go.
  3. Select an item in the Select Value list.
  4. Click Select.

To find items on the Selector page

The Selector page is available only when the Workgroup Administrator has defined hierarchies for the items.

  1. In the Hierarchy drop-down list, select how you want the hierarchy grouped.
  2. Select an item in the Select Value list.
  3. Click Select.
