Editing Document Records

Document records contain information about documents, such as its author or status.

Editing the Document record is not the same as editing the associated document file. A Document record can exist without an associated document (for example, when you're planning a project and you create Document records as placeholders for documents yet to be created), and when a document file is associated with a Document record, the document file must be checked out and edited locally.

You can edit Document records only if you have the proper Workgroup permissions.

To edit a Document record

  1. View a Documents Detail Form.
  2. Make any desired changes. Click the links under Pages to see more information.
  3. Click  Save.
