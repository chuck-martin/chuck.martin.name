Applying Project Templates

Project schedule templates (.mpp files) can be stored in Project InVision and used as a basis for creating project schedules. Use of templates ensures that schedules are structured consistently across projects. A template can be used to create an associated project schedule file, or to populate the Task Module directly.

Project templates must be created in your project management application as standard .mpp files, not as templates (.mpt files).

To apply a template to a Project

  1. In a Projects list, select a project and click  Select Template.
    -or-
    View a Project Detail form    , then click Select Template. The Select Template dialog box appears.
  2. Select a template from the Select from Available Templates list.
    You can click on the link below the description to view details of the template.
  3. Select a Schedule Option:
    • Create Project InVision task list copies the tasks from the template into the Tasks module of the Project InVision Project record, but does not create an associated project schedule file.
    • Create Project Schedule file creates a new project schedule file from the template and associates it with the Project InVision Project record and checks in the project schedule file. Enter a name for the new project schedule file. (Do not enter a file extension.)
  4. Enter a Project Start Date or click  Calendar to select a date.
    All Tasks that do not have a predecessor Task linked to it are updated to this Start Date.
  5. Click Submit.
