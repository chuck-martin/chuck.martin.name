Changing Dashboard View Options

With some of the Views that exist in a Dashboard, there are certain Options that need to be defined for the View. These options include such selection criteria as Filters, Group By, and Content. When one of these Views has been selected as part of the Dashboard, you may wish to change the Options that were originally used for the View.

To change a Dashboard view options

  1. Click on a Dashboard view to select it.
  2. Click  View Options. The View Options dialog box appears.
  3. Make any desired changes to the View Options. (The options vary depending on the view.)
  4. Click View Report.
