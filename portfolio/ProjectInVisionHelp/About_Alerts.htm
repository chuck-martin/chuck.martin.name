About Alerts

Alerts are messages that notify you of events in Project InVision. Many Alerts are system-generated, meaning that pre-defined system events have occurred. Other Alerts are Administrator-defined, meaning that the Workgroup Administrator has set criteria that cause an Alert to be sent. You can also send ad-hoc Alert messages to other users.

You can send Alerts to other Project InVision resources, send email to anyone with a valid email address, or both.

An icon appears next to the Alerts link at the top of the page when you receive a new Alert. If an Alert is received in association with a Project InVision item, the Alert will contain a link to details of that item.

Schedule Synchronization Complete Upon complete synchronization of project schedule file User who performed Check In of project schedule file
Synchronization Failed Upon failure of project schedule file synchronization User who performed Check In of project schedule file
Resource Synchronization Complete Upon complete synchronization of assigned Resources with project schedule file User who performed Check In of project schedule file
Unmatched Resources Upon completion of synchronization, when unmatched Resources exist User who performed Check In of Project schedule
Timesheet Approval Upon submittal of approval of Timesheet entries by Manager Resource who submitted Timesheet
Rejected Timesheet Upon rejected items being submitted by Manager Resource who submitted Timesheet
Document Approval Upon Approval of Document by assigned Manager Resource who is “Author” of the document record
Force Check In Upon Force Check In by the Workgroup Administrator Resource identified as the individual who last Checked Out the associated file
Document Signoff Required Upon change of Document Status to status requiring signoff Resource(s) whose signoff is required

Administrator-Defined Alerts

Workgroup Administrators can create their own set of Alerts, specifying the threshold that activates the Alert, the content of the message that is sent and to who, and the transmission mode (via email, Project InVision Alerts, or both).
