Deleting Projects

Projects can be deleted from the Projects module if, for example, one was created by mistake. However, it is recommended that you keep closed Project records in the Projects module until they are archived by the Workgroup Administrator.

Only users with proper security access can delete Projects.

To delete a Project

  1. In a Projects module list view, select a project record.
  2. Click  Delete.
  3. Click Yes (to confirm deletion).

 If a Project record is deleted, then all records associated with that Project from the other modules are also deleted.
