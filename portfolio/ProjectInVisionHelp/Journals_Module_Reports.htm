Journals Module Reports

The Journal module Report contains information related to Journals. The only Report available for the Journal module is described below (with Report Options), along with a description of the Report and the fields that are captured.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some fields may not exist or may be labeled differently than shown here.

Journal List (Projects, Journal Filter)

This Report shows Journal information for a specific Project or group of Projects, a recap of all the Journals that had been entered that meet the Filter criteria. Project fields: Project Number, Project Title, Entry Number, Entry Title, Date Entered, Entered By, Date Modified, Modified By, and Message.
