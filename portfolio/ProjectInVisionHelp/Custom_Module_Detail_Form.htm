Custom Module Detail Form

The Detail Form for a custom module record contains all the information about that record. The form contains several pages, each containing a specific type of information. Click the links under Pages to view each page. The form also has its own toolbar that allows quick access to common functions.

Custom Module Detail Form Pages

These are the default pages. Your Workgroup Administrator may have renamed existing pages or added new ones, or may have renamed, added, or deleted fields within any page.

Information

This page contains the main information about the Record, where you can view or edit the information.

Comments

This page is for free-form comments that are maintained with the record. See Detail Form Common Pages, Comments.

Keywords

This page allows you to define keywords for the Record. The keywords you can choose are defined by the Workgroup Administrator. See Detail Form Common Pages, Keywords
