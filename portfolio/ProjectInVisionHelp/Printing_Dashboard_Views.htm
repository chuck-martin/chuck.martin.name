Printing Dashboard Views

In a Dashboard , you can select any view and print the data contained in that View.

To print a Dashboard view

  1. Click a view in a Dashboard to select it.
  2. Click Output, then choose Print View. A Print Preview window appears.
  3. Click  Print. The Print dialog box appears.
  4. Click Print.

Only the displayed information is printed. For example, if you're viewing 50 records in a list view, then only 50 records are printed, even though more records might be available. You can either click More to add more records to the view, or click All to view all records, then print. The top right of the status bar shows how many records are being viewed and how many total records are available.
