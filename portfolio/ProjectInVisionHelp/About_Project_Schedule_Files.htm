About Project Schedule Files

Project schedule files can be associated with Project records in Project InVision, which synchronizes information in the project schedule file with some of the project information that Project InVision tracks. You can then access and collaborate on schedule files in Portfolio Manager.

You can check out associated project schedule files and download the file to your workstation to make changes. Only one user can check out a file at a time. Once changes have been made to the checked out file, you can check in the file so that the changes are available to other users.

You can also review a project schedule file, which allows you to download the file to your workstation to look at it, but not make changes to it.

When project schedule files are checked in to Project InVision, they are automatically synchronized with the Project record. The synchronization process populates the Project InVision Tasks module with the tasks and resource assignments from the project schedule file, and updates the project schedule file with information from the Project InVision record, such as the Project #, Title, and Project Team.
