About Views

Views are pre-defined screen layouts that determine what information you will see and how the information will be organized and presented. List views allow you to directly update some fields, which makes data entry easier. See Editing Information.

Your security level determines which views you can see.

The View types in Project InVision include:

  • Grid Views
  • Gantt Charts
  • Custom Charts
  • Crystal Reports

Some views allow you to select view options to further refine the information displayed.
