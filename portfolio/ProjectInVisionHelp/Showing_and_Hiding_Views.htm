Showing and Hiding Views

The Time Tracking application contains several views that you can display or hide, each with different kinds of information:

  • Assignments View - A hierarchical view of your assignments.
  • Daily View - A graphical chart view where you can click on time increments to enter Actual Work Hours throughout the day.
  • Timesheet View - A numerical chart view where you can enter Actual Work Hours in a weekly timesheet format.
  • Assignments Summary Bar - A status bar that shows key information about the selected assignment.

To show or hide a view in the Time Tracking application

  • Click  View Options , then select Show or Hide for a view.

If the view is visible, you'll see the command to Hide the view. If the view is not visible, you'll see the command to Show the view.
