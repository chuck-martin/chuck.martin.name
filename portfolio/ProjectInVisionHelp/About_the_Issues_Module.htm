About the Issues Module

The Project InVision Issues module is where you manage and collaborate on project-related Issues as they arise during the course of a project. An Issue is any event that may impede the successful completion of a Project or Task. In the Issues Module, you track issues from discovery through evaluation to resolution. You can view Issue information in list views or in the Issues Detail Form.

Issues that arise during the implementation of a Project can be associated with either that Project or with a specific Task in that Project. If an Issue is associated with a specific Task, then the Issue is displayed in the Time Tracking application so that time can be posted against the Issue and rolled up into the Project Task. See Time Tracking.
