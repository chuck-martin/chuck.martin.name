Retiring Pull-down List Items

Pull-down items eventually may become disused because conditions have changed and the value is no longer required. Because this value may be linked to many records in the application, it should not be deleted. In these cases, the values can be retired. When retired, the value no longer appears when a user selects the pull-down list.

Retiring Pull-down list items can be done only by users with access to Administrative functions.

To retire a Pull-down list item

  1. Click Administration. The Project InVision Administration window appears.
    The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.
  2. Under Select Screen, click Pull-down Lists.
  3. In the Navigation Menu, click the down-arrow to the right of Module, select a module, then select a list.
  4. Select the Retired check box for a list item.
  5. Click  Save.
