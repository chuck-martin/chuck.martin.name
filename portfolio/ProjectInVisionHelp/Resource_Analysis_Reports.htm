Resource Analysis Reports

Resource Analysis Reports allow managers to compare the requirements for resources on projects to the availability of these resources to work on projects, identifying periods of time where there may be a surplus or shortage of resources. If Resource Plans are created for all projects in the portfolio, these reports allow organizations to make educated decisions on how resources should be best assigned, and how projects should be prioritized within the portfolio.

Resource Analysis Reports compare data from two different areas of Project InVision:

  • Resource Availability – Defines the availability of individual resources to work on projects. This data is maintained as part of the Resource Availability profile on the Availability page of the Resources Detail Form. See Editing Availability.
  • Resource Plans – Defines the resources required to work on a Project. This information is maintained in Resource Plans. See About Resource Plans.

The primary Resource Analysis Reports available in Project InVision include:

  • Resource Analysis Overview
  • Resource Analysis Detailed
  • Resource Analysis Requirement
  • Resource Analysis Surplus / Shortage
  • Resource Histogram
  • Resource Utilization
