Editing Pull-down List Items

Editing Pull-down list items can be done only by users with access to Administrative functions.

To edit a Pull-down list item

  1. Click Administration. The Project InVision Administration window appears.
    The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.
  2. Under Select Screen, click Pull-down Lists.
  3. In the Navigation Menu, click the down-arrow to the right of Module, select a module, then select a list.
  4. Click in each field that contains content you want to change, then make the change.
    If the field is a text field, click in the field and enter the appropriate text.
    If the field is a list field, click in the field to see the drop-down list arrow, click on the arrow, then select an item from the list.
  5. Click  Save.
