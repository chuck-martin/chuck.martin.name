Deleting Journal Entries

Journal entries can be deleted from the Journals module if, for example, one was created by mistake. However, Journal records should remain in the Journals module so when a Project is archived by the Workgroup Administrator, all associated Journal records are archived with the Project.

You can delete Journal entries only if you have the proper security.

To delete a Journal entry

  1. In a Journals list view, click a Journal name.
  2. Click  Delete.
  3. Click Yes (to confirm deletion).

When you're in the Central Discussion view, you should be careful not to delete top-level Discussion items before deleting bottom-level discussion items. Otherwise, the bottom-level items become “orphans.”
