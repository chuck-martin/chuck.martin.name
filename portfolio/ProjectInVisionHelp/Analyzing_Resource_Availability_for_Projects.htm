Analyzing Resource Availability for Projects

When you're adding or removing Resources from a Project team, you can quickly view reports that show resource availability to help you select the appropriate resources for that Project.

To analyze Resource availability for a Project

  1. In a Project module list, select a Project record, then click  Team. The Define Project Team dialog box appears.
    You can also view a Project Detail form, then click  Team in the toolbar there.
  2. Click Histogram or Resource Project Calendar. The Report Options dialog box appears with the respective report selected.
    You can also click the More Reports link for other reports that may give you useful information about resource availability.
  3. Select report options on the right side of the dialog box.
  4. Click View Report.
