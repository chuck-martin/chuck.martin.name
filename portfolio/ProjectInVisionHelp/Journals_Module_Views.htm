Journals Module Views

Default Views have been created for the Journals module. Your security access determines what Views you can see.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some View names or fields may not exist or may be labeled differently than shown here.

-Standard-

This View shows current status information for selected Project Tasks: the individual Tasks, the Current Finish Date of the Task, and the % of the Task that is complete. You can view and edit information related to Actual and Baseline Dates, as well as comments and Resource information. Fields: Project Number, Project Title, Task Name, Task Number, Current Finish, % Complete, Resource Initials, Actual Start, Actual Finish, Base Start, Base Finish, Current Start, Comments, Task Code, and Resource Name.

Central Discussions

This View shows the discussions related to the Project like in the Discussions pages of the Detail Forms. Standard Journal records do not appear in the Central Discussions View of the Journals module. Only those items that are entered as record discussions appear in the View.
