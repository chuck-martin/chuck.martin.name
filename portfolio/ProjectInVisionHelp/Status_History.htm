Status History

Each time the Status of a record is changed, Project InVision logs the date, time, and user that made the change.

In addition, Workgroup Administrators have the ability to revert a Project Status (setting the Status back to a previously selected value) using the Project InVision Administration application. This helps in situations where a Project has been advanced incorrectly and should be in a Status that users can no longer select from the Status pull-down list.

To view the Status History of a record

  1. Select a record.
  2. From the Tools menu, choose Status History.

You can also view the Status History In the Projects Detail Form: When Status has been updated, the label turns into a blue hyperlink. Click on the hyperlink to view the Status History for that Project.
