Creating Document Records

Document records contain information about documents associated with Projects.

The Document record is not the same as an associated document file. A Document record can exist without an associated document file (for example, when you're planning a project and you create Document records as placeholders for documents yet to be created). Document files are associated with, or attached, to Document records.

You can either create new document files based on pre-defined templates or add currently existing document files to the record. See About Templates.

To create a Document record

  • In a Projects module list view, select a Project.
  • Under Modules, click Documents.
  • Click  New. The New Documents Detail Form appears.
  • Enter the Document information. Fields with bold labels are required.
  • Click Submit.
