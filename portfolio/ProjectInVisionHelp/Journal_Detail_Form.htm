Journal Detail Form

The Journal Detail Form contains information about a Journal entry.  The form contains several pages, each containing a specific type of information. Click the links under Pages to view each page. The form also has its own toolbar that allows quick access to common functions.

Journal Detail Form Pages 

These are the default pages. Your Workgroup Administrator may have renamed existing pages or added new ones, or may have renamed, added, or deleted fields within any page.

Information

This page contains a Title and a Message for the Journal entry. The Message area is a text field to enter more written details.

Keywords

This page allows you to define keywords for the Journal entry. The keywords you can choose are defined by the Workgroup Administrator. See Detail Forms: Common Pages, Keywords.

Links

This page contains links to other records within Project InVision or to web addresses outside Project InVision. Links within Project InVision help you track associations between Projects and other items, such as Tasks and Resources. See Detail Forms: Common Pages, Links.
