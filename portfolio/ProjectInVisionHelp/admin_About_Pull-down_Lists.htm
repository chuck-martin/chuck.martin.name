About Pull-down Lists

Pull-down lists, the Project InVision term for drop-down lists, provide users with a pre-defined set of options when completing Detail Forms. Pull-down lists ensure that data integrity is maintained throughout the various Workgroups.

You can edit only Workgroup options in the browser-based Administration application. To edit Enterprise-level Pull-down lists, you must use the Project InVision Administration application.
