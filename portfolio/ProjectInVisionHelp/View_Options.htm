View Options

In some Views, such as chart or other report views, rather than list views, you can apply options that refine the information displayed. Those options are applied through the View Options dialog box. If a View can have options applied, a View Options link will appear in the status bar directly below the view.

To set view options

  1. If a view can be changed with view options, click the View Options link in the status bar below the view. The [ViewName] Options dialog box appears.
  2. Make any desired changes. (See the option descriptions below.)
  3. Click Submit.

View Option Descriptions

Options for a view can include one or more of these.

Filter The Filter list allows you to apply a pre-determined Filter against the module information.
Rollup The Rollup list allows you to roll up data by specific Field criteria. The rolllups available depends on the view.
Scale Select Daily, Weekly, Monthly, or Quarterly.
Content The Content list allows you to determine the type of data to be viewed, which depends on the list. For example, for the Resources Actual Work Pie Chart View, the Content would contain Actual Work and Actual Cost, but for the Resource Work Pie Chart, the Content would change to Current Work, Baseline Work, Current Cost, and Baseline Cost.
Representation The Representation list allows you to determine how data is represented in a View, which depends on the module. For example, for the Resources Project Calendar View, the Representation would be a Percentage, the Number of Resources, or Work Hours.
Group The Group list allows you to determine how to group data in a View, which depends on the module. For example, for the Resources Actual Work Pie View, the Group would be Project, Type, Priority, Status, Condition, Region, or Department.
Date Range Select Current Year, Last Year, Next Year, Current Quarter, Last Quarter, Next Quarter, Current Month, Last Month, Next Month, Current Week, Last Week, and Next Week.
From If no specific Date Range selection is chosen, select a start date.
To If no specific Date Range selection is chosen, select an end date.
