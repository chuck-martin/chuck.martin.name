Resources Module Reports

Resources module Reports contain information related to Resources and their status across the Enterprise. The Reports available for the Resources module are described below (with Report Options), along with a description of the Report and the fields that are captured.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some fields may not exist or may be labeled differently than shown here.

Resource List (Projects, Resources Filter)

This Report is a contact list of Resources for the Workgroup or for a group of Projects. Project fields: Name, Title, Department, Phone Type, Phone Number, Email Type, Email Address, and Address.

Resource Assignment - Overview (Projects, Resources Filter)

This Report shows individual Resources, their Projects, and their Tasks, grouped by Resource and then by Project. Project fields: Resource Number, First Name, Last Name, Middle Initial, Project Number, Project Title, Task Number, Task Name, Current Work, Units, Start Date, and Finish Date.

Resource Assignment - Detail (Projects, Resources Filter)

This Report shows individual Resources, their Projects, and their Tasks, grouped by Resource and then by Project. Project fields: Resource Number, First Name, Last Name, Middle Initial, Project Number, Project Title, Task Number, Task Name, Current Work, Baseline Work, Actual Work, Start Date, and Finish Date.

Resource Analysis - Overview (Projects, Resources Filter, Projects Filter, Scale, Rollup, Representation, Date Range, From Date, To Date)

This Report shows Resource Availability. You can see how much of a Resource's time is available, how much is required, and how much is a surplus or a shortage so you can see if Resources have been over-allocated to Tasks or if they are available to be assigned new Tasks. The Scale can be set at Daily, Weekly, Monthly, or Quarterly. Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region and be represented as a percentage, as # of Resources, or as Work Hours.

Resource Analysis - Requirement (Projects, Resources Filter, Projects Filter, Scale, Rollup, Representation, Date Range, From Date, To Date)

This Report shows Resource requirements in terms of currently assigned Work Hours of the Resource. You can see the individual and overall Work requirements for the specified Date Ranges. Red indicates a shortage or over-allocation of that Resource (more Required Hours than the Resource can accomplish in the time frame). The Scale can be Daily, Weekly, Monthly, or Quarterly. Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region and he resulting numbers can be represented as a percentage, as # of Resources, or as Work Hours.

Resource Analysis - Surplus / Shortage (Projects, Resources Filter, Projects Filter, Scale, Rollup, Representation, Date Range, From Date, To Date)

This Report shows Resource availability in terms of Work Hours available for Task assignments. You can see how much of a Resource's time is available. Red indicates a shortage or over-allocation of that Resource. The Scale can be set at Daily, Weekly, Monthly, or Quarterly. Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region and then be represented as a percentage, as # of Resources, or as work hours.

Resource Analysis - Detailed (Projects, Resources Filter, Projects Filter, Scale, Rollup, Representation, Date Range, From Date, To Date)

This Report shows Resource Requirements on a Project-by-Project basis. The Scale can be set at Daily, Weekly, Monthly, or Quarterly. Numbers can then be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region and represented as a percentage, as # of Resources, or as Work Hours. The Report is especially useful to review the availability of a particular Skill Category or individual Resource. If Filters are applied based on a Skill Category, but displayed by Resource, you can see the Resources belonging to the Skill Category available for assignment to Projects within the selected time.

Resource Curve Chart (Projects, Resources Filter, Projects Filter, Scale, Content, Date Range, From Date, To Date)

This Report shows a line chart of Actual Work or Actual Cost over time.

Resource Histogram (Projects, Resources Filter, Projects Filter, Scale, Rollup, Representation, Date Range, From Date, To Date)

This Report shows a bar chart of Resource availability. Availability is yellow, Allocation is blue, and Over-Allocation is red. The Scale can be set at Daily, Weekly, Monthly, or Quarterly. Numbers can then be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region. The resulting numbers can then be represented as a percentage, as # of Resources, or Work Hours.

Resource Project Calendar (Projects, Resources Filter, Projects Filter, Scale, Rollup, Representation, Date Range, From Date, To Date)

This Report shows Resource availability on a Project by Project basis. The Scale can be set at Daily, Weekly, Monthly, or Quarterly. Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region and then represented as a percentage, as # of Resources, or as Work hours. The data are displayed both in a table and in an Availability Graph, which shows white for Available and blue for Allocated.

Resource Utilization Chart (Projects, Resources Filter, Projects Filter, Scale, Rollup, Date Range, From Date, To Date)

This Report shows Resource availability on a timescale basis. The Scale can be set at Daily, Weekly, Monthly, or Quarterly. Numbers can be rolled up by Entire Resource Pool, Resource, Skill Category, Department, or Region. The data is shown graphically as colors: teal for Underutilized, yellow for Even Allocation, and red for Overallocated.

Billing Actual Work by Billing Code (Projects, Resources Filter, Projects Filter, Date Range, From Date, To Date)

This Report shows Billing Codes, the Projects where they are used, and the Resources assigned to the Work, grouped by Billing Code and then by Project, plus Billing Code Total and Grand Total. The assigned Resources are identified, along with the Actual Work posted to the Task, through Time Tracking. Project fields: Billing Code, Project Number, Project Title, Resource, and Actual Hours.

Billing Actual Work by Cost Center (Projects, Resources Filter, Projects Filter, Date Range, From Date, To Date)

This Report shows Cost Centers, the Projects where they have been used, and the Resources assigned to the Work, grouped by Cost Center and then by Project, plus Cost Center Total and Grand Total. The assigned Resources are identified, along with the Actual Work posted to the Task, through Time Tracking. Project fields: Cost Center, Project Number, Project Title, Resource, and Actual Hours.

Billing Actual Work by Resource (Projects, Resources Filter, Projects Filter, Date Range, From Date, To Date)

This Report Resources and the Projects they have been assigned to, grouped by Resource and then by Project, plus Resource Total and Grand Total. Project fields: Resource, Workgroup, Project Number, Project Title, and Actual Hours.

Billing Actual Work by Workgroup (Projects, Resources Filter, Projects Filter, Date Range, From Date, To Date)

This Report shows the Resources who have been assigned work in each Project , grouped by Workgroup and then by Project, the Actual Work posted to the Task through Time Tracking, and the Workgroup Total and Grand Total hours. Project fields: Workgroup, Project Number, Project Title, Resource, and Actual Hours.

Resources Pie Chart (Projects, Resources Filter, Projects Filter, Rollup, Group, Content, Date Range, From Date, To Date)

This Report shows the Work performed on Projects as both a Pie Chart and a table. The Work Hours will display for the specified Date Range, grouped by Rollup. The Rollup can be Resource, Skill Category, Department, or Region. The resulting numbers can be grouped by Project, Type, Priority, Status, Condition, Region, or Department and represented as Current Work, Baseline Work, Current Cost, or Baseline Cost. Project fields: Project Number, Project Title, Resource Name, Current Work, Baseline Work, Billing Rate, Current Cost, and Baseline Cost.

Resources Pie Chart - Actual (Projects, Resources Filter, Projects Filter, Rollup, Group, Content, Date Range, From Date, To Date)

This Report shows a Pie Chart with slices based on Rollup selection and the Date Range specified. Depending on the Rollup used, the Work Hours will display for the specified Date Range, grouped by the various values that exist for the Rollup field. The Rollup can be Resource, Skill Category, Department, or Region. Under the Pie chart is a list with information details. Project fields: Project Number, Project Title, Resource Name, Actual Work, Billing Rate, and Actual Cost.

Resources Work Distribution (Projects, Resources Filter, Projects Filter, Rollup, Group, Content, Date Range, From Date, To Date)

This Report shows the Work performed on Projects, grouped by Rollup. The Rollup can be set at Resource, Workgroup, Project, Billing Code, or Cost Center. Numbers can be grouped by Project, Resource, Billing Code, or Cost Center. The Content can be presented as Current Work or Baseline Work. Project fields: Workgroup, Resource, Resource Name, and Current/Baseline Hours.

Resources Work Distribution - Actual (Projects, Resources Filter, Projects Filter, Rollup, Group, Date Range, From Date, To Date)

This Report shows the Actual Work performed on Projects based on Rollup selection and Date Range, grouped by Rollup value. The Rollup can be Resource, Workgroup, Project, Billing Code, or Cost Center. Project fields: Workgroup, Resource, Resource Name, and Actual Hours.

Resources Work Distribution Summary - Actual (Projects, Resources Filter, Projects Filter, Rollup, Date Range, From Date, To Date)

This Report shows the Actual Work performed on Projects based on Rollup selection and Date Range. The Rollup can be Resource, Workgroup, Project, Billing Code, or Cost Center. Project fields: Workgroup and Actual Hours.

Resources Work Distribution Summary (Projects, Resources Filter, Projects Filter, Rollup, Content, Date Range, From Date, To Date)

This Report shows the Work performed on Projects, grouped by Rollup. The Rollup can be set at Resource, Workgroup, Project, Billing Code, or Cost Center. The Content can be presented as Current Work, or Baseline Work. Project fields: Workgroup, and Current or Baseline Hours.
