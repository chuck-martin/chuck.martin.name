Project Options

The default Project synchronization and Time Tracking options are configured for the Workgroup by the Workgroup Administrator. If you have the appropriate security, you can change these options on a project-by-project basis.

To change project options

  1. View a Project Detail form.
  2. Under Pages, click Options.
  3. Make any desired changes. (See details below.)
  4. Click Save.

Project Option Details

Disable Auto-Post Time
Project InVision will not automatically post time from the Time Tracking application to an associated project schedule file. Time has to be manually posted against the specific Project using Project InVision Administration.
Disable Auto-Synchronize of Project Schedule
Project InVision will not automatically synchronize changes in the Project InVision application with the project schedule file. Synchronization of the Project schedule must be done manually using Project InVision Administration.
Disable Auto-Synchronize of Current Resource Requirements Plan
Project InVision will not automatically synchronize the Resource Requirements Plan with the project schedule file or the Project InVision Tasks module. Synchronization of the Resource Requirements Plan for the specific Project must be manually completed using Project InVision Administration.
Disable Auto-Synchronize of Calendar
Project InVision will not automatically synchronize changes in the Project InVision calendar with the project schedule file. Synchronization of the Calendar for the specific Project must be manually completed using Project InVision Administration.
Disable Auto-Synchronize of Cost Rate
Project InVision will not automatically synchronize changes in the Project InVision Cost Rates with the project schedule file. Synchronization of the Cost Rates for the specific Project has to be manually completed using Project InVision Administration.
Disable Auto-Synchronize of Billing Rates
Project InVision will not automatically synchronize changes in the Project InVision Billing Rates with the project schedule file. Synchronization of the Billing Rates for the specific Project has to be manually completed using Project InVision Administration.
Resource Current Cost Source
Select the source of Resource Current Cost information. Choices include the associated project schedule file (default), the current resource plan, from user-entered costs, or from Task module assignments.
Resource Baseline Cost Source
Select the source of Resource Baseline Cost information. Choices include the associated project schedule file (default), the baseline resource plan, or from user-entered costs.
Resource Actual Cost Source
Select the source of Actual Cost information. Choices include the associated project schedule file (default), from Time Tracking actual hours booked against the Project, or a user-entered costs.
Resource Cost Rate Source
Select the source of Resource Cost Rate information. Choices include the Resource, the Skill Category, the Resource and then the Skill Category (if the Resource Cost Rate is null), or the Skill Category and then the Resource (if the Skill Category Cost Rate is null).
Resource Billing Rate Source
Select the source of Resource Billing Rate information. Choices include the Resource, the Skill Category, the Resource and then the Skill Category (if the Resource Billing Rate is null), or the Skill Category and then the Resource (if the Skill Category Billing Rate is null).
Calculate Task Assignments Based On
Select whether Task assignments are based on Fixed Units or Fixed Work.
