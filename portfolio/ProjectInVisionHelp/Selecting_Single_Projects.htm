Selecting Single Projects

Selections in the Selector control what's seen in the window. You can select a single Project to see only that Project (in the Projects module) or just the records associated with that Project (in any other module).

Single Projects have the Project Number next to the name and do not have any records lower in the hierarchy attached to them.

To select a single Project

  1. If it is not open, click  Selector to view the Selector pane.
  2. Click the name of a Project.

If you do not see a Project name in the list, click More or All at the bottom of the pane to see more Projects.
