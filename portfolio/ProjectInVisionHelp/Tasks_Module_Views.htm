Tasks Module Views

Default Views have been created for the Tasks Module. Your security access determines what Views you can see.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some View names or fields may not exist or may be labeled differently than shown here.

-Standard-

This View shows current status information for selected Project Tasks. You can view and edit information related to Actual and Baseline dates, as well as Comments and Resource information. Fields: Project Number, Project Title, Task Name, Task Number, Current Finish, % Complete, Resource Initials, Actual Start, Actual Finish, Base Start, Base Finish, Current Start, Comments, Task Code, and Resource Name.

Actual Status

This View shows how selected Tasks are progressing based on Actual Work Hours posted to the Task or Task-related item in the Time Tracking application. This allows you to see the most accurate information regarding the true Status of Project Tasks. Fields: Project Number, Project Title, Task Name, Actual Start, Actual Finish, Actual Duration, and Actual Work.

Current vs Baseline Dates

This View shows a comparison of current schedule dates versus the original Baseline Dates. Fields: Project Number, Project Title, Task Name, Current Start, Current Finish, Baseline Start, Baseline Finish, Finish Variance, % Complete, and Resource Initials.

Gantt Chart

This View shows a graphical representation of Task duration. Fields: Task Number, Task Name, a bar showing the duration of the Task, and a triangle showing the Task is a Milestone.

Notes

This View shows information on Tasks specific to their current condition, plus any comments that have been added, grouped by Project. Fields: Project Number, Project Title, Task Number, Task Name, Current Start, Current Finish, % Complete, and Comments.

Status

This View shows the Current Start, Current Finish, and % Complete of Tasks. grouped by individual Project. Fields: Project Number, Project Name, Task Number, Task Name, Current Duration, Current Start, Current Finish, % Complete, and Resource Name.
