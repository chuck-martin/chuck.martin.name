Marking Tasks as Key Deliverables

Key Deliverables are Tasks that are tracked to understand the progression of a Project in relation to Key Milestones. The Project Detail Form lists all Key Deliverables for the Project on its Key Deliverables page so you don't have to review the entire Tasks list or project schedule file to understand the status of these key Tasks.

When a project schedule file is associated with a Project:

  • Project InVision synchronizes the Marked field on a task in the Project file with the Key Deliverable option in the corresponding Project InVision Task.
  • You can mark Tasks as Key Deliverables either in the project schedule file or in the Project InVision Task.

To mark a Task in a project schedule file as a Key Deliverable

  1. In the checked out project schedule file, set the “Marked” field to “Yes.”

To mark a Task in Project InVision as a Key Deliverable

  1. View a Task Detail form.
  2. In the Information section, select the Key Deliverable check box.
  3. Click Submit.
