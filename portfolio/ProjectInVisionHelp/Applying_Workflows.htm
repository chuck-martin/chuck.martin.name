Applying Workflows

The process for applying a workflow to a record is the same for all records where workflows can be applied. Workflows can be applied to records in:

  • Projects
  • Documents
  • Issues
  • Defects
  • Risks
  • Changes
  • Requirements

To apply a workflow

  1. View the Detail Form for the record.
  2. If the Information page is not displayed, click Information.
  3. In the Workflow drop-down list, select a Workflow.
    The default initial status for the selected Workflow is applied to the item.
    Other fields may become required or non-editable based on the Workflow selected.
  4. Click Submit.
