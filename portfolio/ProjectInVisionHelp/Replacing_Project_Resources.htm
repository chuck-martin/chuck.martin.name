Replacing Project Resources

Project InVision allows easy replacement and reassigning of Resources on Projects. Situations where you might have to replace Resources include:

  • The first Resource has changed positions or is no longer available to the Project.
  • The original Resource assigned to the Project is a Generic Resource that has been used for Resource Planning purposes, but now you have to assign the actual individual who will perform the work

Resources can be replaced this way at the Project and Task level, but not on any Issues, Defects, Risks, Changes, or Requirements assignments. These items may need to be manually converted from the original Resource to the new one.

To reassign Project resources

  1. In a Project module list, select a Project record, then click  Team. The Define Project Team dialog box appears.
    You can also view a Project Detail form, then click  Team in the toolbar there.
  2. Click Replace Resources. The Replace Resource dialog box appears.
  3. Select a current team member from the Replace drop-down list box.
  4. Click  Find. The Find dialog box appears.
  5. Select a name from the list, then click Select.
  6. Click Replace.

The newly assigned Resources are automatically posted to the Resource Sheet of the associated project schedule file. All of the associated Task assignments for the Resources are also replaced, provided that the Actual Work Hours are zero for the Resource being replaced. If Actual Work Hours have been posted to any Task, the first Resource is not replaced with the second for that specific Task.
