Checking Out Project Schedule Files

Once a project schedule file has been associated with a Project InVision Project record, you can check out the Project file to make changes to it. Only one Resource can check out a Project schedule file at a time, which ensures that any changes are propagated to the master copy before another individual checks out the file.

To check out a project schedule file

  1. In a Projects list view, select any record.
  2. Click  Check Out.
    If the file is already checked out, you'll see the  Check In button instead.
    You can also click Check Out when you're viewing the Projects Detail Form.
    The file opens in an Internet Explorer window.
  3. From the File menu, choose Save As, then save the file to your hard disk.

Because of the way files are transferred over the Web, spaces in the filename are replaced with the characters “%20” during check out. If you save the file as-is, those characters will remain in the filename and you'll have two similarly named files. To resolve this, review the File name field and replace any instances of %20 with a space by selecting the “%20” characters and pressing the spacebar. You can eliminate this as an issue by creating files with no spaces in the filenames.

Once you save the file to your hard drive, you can open and edit it. When you're finished editing, you should check the file back in to Project InVision.
