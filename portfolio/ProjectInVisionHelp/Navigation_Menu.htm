Navigation Menu

The Navigation Menu is where you define what you see in Project InVision. Click the down arrowhead next to any of the selections to switch your workgroup or module or to change the view, filter, or grouping.

Workgroup

See About Workgroups.

Module

Defines what area of Project InVision you'll be in and what type of information you'll see, and contains the same items as under the Module heading on the left. While the text of the items are configurable, the first item (often Project) is the highest level, and defines what appears in the other selections. The items defined in the first selection appear also in the Selector tree, which you can use to filter the view as well.

View

A view is a particular subset of information in the selected list, organized in a particular way. Views can be lists, charts, graphs, and more.

Filter

A filter defines a subset of items in the selected list. A common filter is items that are assigned to you.

If you have the appropriate security, click on the word “Filter” to define custom filters.

Group By

A grouping is a way to group records within the selected list and view. A common grouping is by department.

Mode

Defines which view mode lists are presented in. Read Only shows many fields as links, which you can click on to see more information. Fields without links are static. Editable mode removes all links from fields, but provides editing of field content either directly or with the Find dialog box.
