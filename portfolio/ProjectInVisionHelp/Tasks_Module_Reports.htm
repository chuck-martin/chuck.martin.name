Tasks Module Reports

Tasks module Reports contain information related to Tasks and their status across the Enterprise. The Reports available for the Tasks module are described below (with Report Options), along with a description of the Report and the fields that are captured.

Reports may be customized by the Workgroup Administrator or a Report Writer, so some fields may not exist or may be labeled differently than shown here.

Tasks Status (Projects, Tasks Filter)

This Report shows the Current Start, Current Finish, and % Complete of a Tasks, grouped by Project. Project fields: Project Number, Project Name, Task Number, Task Name, Current Duration, Current Start, Current Finish, % Complete, and Resource Name.

Tasks Status - Actual (Projects, Tasks Filter)

This Report shows the true Status of Tasks, grouped by Project, based on Actual Work Hours posted to Tasks through the Time Tracking application. It allows Managers to see exactly how Tasks are progressing, when they actually started, the actual target Finish Date, how many Work Hours have been posted to each Task, and the Current Cost of the Task based on the Cost rates charged per Actual Work by Resources assigned to the Task. Project fields: Project Number, Project Title, Task Number, Task Name, Actual Start, Actual Finish, Actual Duration, Actual Work, and Actual Cost.

Tasks Status - Current vs. Baseline (Projects, Tasks Filter)

This Report compares the Current Start and Finish Dates against the Baseline Start and Finish Dates to show the Variance between them, grouped by Project. Project fields: Project Number, Project Title, Task Number, Task Name, Current Start, Baseline Start, Start Variance, Current Finish, Baseline Finish, and Finish Variance.

Tasks Status - Baseline (Projects, Tasks Filter)

This Report shows original Baseline values information about Tasks, grouped by Project. Project fields: Project Number, Project Title, Task Number, Task Name, Base Start, Base Finish, Base Duration, Base Work, and Base Cost.

Tasks Status - Costs (Projects, Tasks Filter)

This Report shows cost comparison information about Tasks, grouped by Project. Project fields: Project Number, Project Title, Task Number, Task Name, Current Cost, Base Cost, Actual Cost, Remaining Cost, and Cost Variance.

Tasks Status - Current (Projects, Tasks Filter)

This Report shows the Current Status of Tasks based on the updated Project Schedule, grouped by Project. Project fields: Project Number, Project Title, Task Number, Task Name, Current Start, Current Finish, Current Duration, Current Work, and Current Cost.

Tasks Status - Work (Projects, Tasks Filter)

This Report shows Work comparison information on Tasks specific to their original Baseline values, Current values, and Actual values, grouped by Project. Project fields: Project Number, Project Title, Task Number, Task Name, Current Work, Base Work, Actual Work, Remaining Work, and Work Variance.

Tasks - Notes (Projects, Tasks Filter)

This Report shows the completion percentage of and notes attached to tasks, grouped by Project. Project fields: Project Number, Project Title, Task Number, Task Name, Current Start, Current Finish, % Complete, and Comments.

Key Deliverable Current Status (Projects, Tasks Filter)

This Report shows Status information on Tasks that have been identified as a Key Deliverable, grouped by Project. A Task is considered a Key Deliverable in your project management application if the “Marked” field has been changed to Yes. Project fields: Project Number, Project Title, Task Number, Task Name, Current Start, Current Finish, and Current Duration.

Key Deliverable Current vs. Base (Projects, Tasks Filter)

This Report compares the Current Start and Finish Dates, grouped by Project, against the Baseline Start and Finish Dates of Key Deliverable Tasks to show the Variance between them. A Task is considered a Key Deliverable in your project management application if the “Marked” field has been changed to Yes. Project fields: Project Number, Project Title, Task Number, Task Name, Current Start, Current Finish, Baseline Start, Baseline Finish, and Finish Variance.

Variance - Work by Task Code (Projects, Tasks Filter)

This Report shows the Current and Baseline Work for Tasks and the Variance between these two in terms of a percentage and an average. Project fields: Task Code, Task Code Description, Current Work, Base Work, Variance Percentage, and Variance Average.

Variance - Duration by Task Code (Projects, Tasks Filter)

This Report shows an analysis of the Current and Baseline Durations for various Tasks and the Variance between these two in terms of a percentage and an average. Project fields: Task Code, Task Code Description, Current Duration, Base Duration, Variance Percentage, and Variance Average.
