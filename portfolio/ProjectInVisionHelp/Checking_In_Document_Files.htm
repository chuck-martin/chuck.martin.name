Checking In Document Files

After you create a new Document record, you can associate it with a file of almost any type. Associating a file with a Document record means checking the file in to Project InVision.

Checked in files are saved in Project InVision and locked to changes except by users with the appropriate security.

To check in a document file

  1. In a Documents list view, click the Document Title of a Document record that does not already have a checked-in file.
    The Documents list can show either a subset of Documents, based on a selected Project or group of Projects, or all Documents. If you do not see the Document record you want to edit, make sure you have selected the Project that the Document record is part of, or select -All-.
  2. Click  Check In. The Check In dialog box appears.
    If the file is already checked in, you'll see the  Check Out button instead.
    You can also click Check In when you're viewing the Documents Detail Form.
  3. Click Browse.
  4. Find the file you want to check in, select it, then click Open.
  5. Click Check In.
