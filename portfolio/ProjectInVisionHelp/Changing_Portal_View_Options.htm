Changing Portal View Options

Most views have a number of options to control what and how data is presented. You can change these options for any view in a Portal that has view options.

To change a Portal view's options.

  1. When viewing a Portal, click any view to select it (click anywhere in the view except on the view name or the icons in the title bar of the view).
  2. Click  View Options. The Options dialog box appears.
  3. Make any desired changes to the view options.
  4. Click Submit.
