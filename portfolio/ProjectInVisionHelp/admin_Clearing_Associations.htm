Clearing Associations

Project schedule files can be associated with Project InVision Projects. At times, you may have to clear this association with the Project record, possibly due to corruption in the file or because a new association needs to be made.

Clearing associations can be done only by users with access to Administrative functions.

To clear an association between a project schedule file and a Project InVision Project

  1. Click Administration. The Project InVision Administration window appears.
    The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.
  2. Under Select Screen, click Clear Association.
  3. Select a Project by clicking anywhere in the Project row except the Project # field.
    Do not click the Project # link.
  4. Click  Clear Association.
  5. Click Yes.
