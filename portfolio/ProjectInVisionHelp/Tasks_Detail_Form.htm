Tasks Detail Form

The Tasks Detail Form contains information about a Task, including its Billing Code, Cost Center, and Task Code. The form includes several pages, each containing a specific type of information. Click the links under Pages to view each page. The form also has its own toolbar that allows quick access to common functions.

Tasks Detail Form Pages 

These are the default pages. Project InVision administration and security allow pages to be added, removed, or renamed for both workgroups and for user categories, so the pages you see might not match what's listed here.

Information

This page contains the basic Task information, such as the Task Number, Task Name, Billing Code, Task Code, and Cost Center, plus Start and Finish Dates, Duration, and Work.

  • Project # is generated by Project InVision and is not editable. It is one way to identify the Project.
  • Project Title is a brief (up to 40 characters) description of the Project.
  • Billing Code identifies what category Project costs belong to.
  • Cost Center identifies what department is responsible for paying Project costs.
  • Task Code identifies the Task category and is used to categorize and search for Project records.
  • Start Date identifies the Current, Baseline, and Actual start dates.
  • Finish Date identifies the Current, Baseline, and Actual finish dates.
  • Duration identifies the Current, Baseline, Actual and Remaining Task duration, along with any Variance.
  • Work identifies the Current, Baseline, Actual and Remaining Task work, along with any Variance.
  • Critical identifies if the Task is critical.
  • Key Deliverable identifies if the Task is a key deliverable.
  • Summary identifies if the Task is a summary Task.
  • Milestone identifies if the Task is a milestone Task.

Comments

This page is for free-form comments that are maintained with the Task record. See Detail Form Common Pages, Comments.

Resources

This page shows all Resources that have been assigned to the Task. If you have an associated project schedule file, then Resources that are assigned to the Task in Project InVision are also posted to the Resource Sheet of the associated project schedule file.

Links

This page contains links to other records within Project InVision or to web addresses outside Project InVision. Links within Project InVision help you track associations within Project InVision. See Detail Form Common Pages, Links.

Discussions

This page contains threaded Discussions related to the Task. You can create a Discussion item, respond to the sender of a Discussion item, or respond to the group of users that are involved in the Discussion. See Detail Form Common Pages, Discussions.
