Viewing Document Versions

When you check out a document, make changes to it, and check it back in, Project InVision maintains the previous copy, a version of that document file. This is useful if you want to compare one version to another or discard the changes you made and go back to a previous version.

The number of versions that Project InVision saves is set by your Workgroup Administrator. You can view any version of a document file that is stored with a Document record, including the most recent version, known as the Master version.

To view a version of a document file

  1. View a Document Detail form.
  2. Under Pages, click Versions.
  3. Click a version in the list.
    The version with the highest number is the most recent (Master) version.
  4. Click  Review.
    -or-
    In the Association column, click the link for the version you want to review.

Viewing a version of a document file does not check it out.
