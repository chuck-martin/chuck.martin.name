Deleting Tasks

Tasks can be deleted from the Tasks module if, for example, one was created by mistake. However, closed Task records should remain in the Tasks module so the actions taken to complete the Task remain documented. When a Project is archived by the Workgroup Administrator, all associated Task records are archived with the Project.

You can delete Tasks only if you have the proper security. If the Project has an associated project schedule file, Tasks must be deleted by checking out the associated file and deleting tasks in that file.

To delete a Task

  1. In a Tasks module list view, select a task record.
  2. Click  Delete.
  3. Click Yes (to confirm deletion).

If Actual Work Hours have been entered against a Task, the Task may not be deleted. See your Workgroup Administrator for assistance with deleting Tasks that have been assigned Actual Work Hours.
