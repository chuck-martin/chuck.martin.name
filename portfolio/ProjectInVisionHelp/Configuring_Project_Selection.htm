Configuring Project Selection

Projects can be selected is different ways in Project InVision: including with the Selector, clicking in a Projects module list, using Find, or using Search. Knowing what Projects are selected is important because the selected Projects control what records you see in other modules. You can select only one Project at a time in any Project module list view, but in the Selector, you can select multiple Projects in several different ways.

Because Project selection drives what you see in other modules, and because you can select Projects in both a Project module list and in the Selector, Project InVision provides two Project selection priority modes.

To configure Project selection

  1. Click My Setup.
  2. Under Select Item, click Selector Settings.
  3. Select a Project selection priority mode. (See below for definitions.)
  4. Click  Save.
  5. Click Submit.

Project Selection Priority Modes Defined

Project record selection in View takes priority
  • The Project Indicator shows the selected Project.
  • The selected Project and any selected records in any other module list will be highlighted in pale yellow.
  • If you view another module, the Selector pane will close, but will retain its existing Project selections.
Selector takes priority
  • The Project Indicator shows the text “Level:,” followed by the hierarchy level name, if you've selected an entire hierarchy level, the text “Multiple selections” if you've selected multiple discrete Projects, or the Project name if you've selected a single Project.
  • Highlighting in all modules lists will be pale blue.
  • If you select a Project in a Projects module list, the Projects selected in the Selector will still drive what you see when you view other modules. (The Project selection will not be overridden by the selection in the Projects module list.)
  • You can hold down the CTRL key when clicking or on a Project in the Projects module list to select it and temporarily have that Project selection take priority and drive what you see when you view records in other modules. The record background will be highlighted in pale yellow until you click on another Project name.
  • You can right-click on a Project in the Projects module list and select the Override Hierarchy Selection command to select it and temporarily have that Project selection take priority and drive what you see when you view records in other modules. The record background will be highlighted in pale yellow until you click on another Project name.
