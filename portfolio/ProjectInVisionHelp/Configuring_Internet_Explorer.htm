Configuring Internet Explorer

Project InVision requires configuration of Internet Explorer to run ActiveX controls and Scripts properly. You should define Project InVision as a Trusted Site, which will not affect security settings for other sites or zones.

Project InVision requires Internet Explorer version 5.0 or newer. 

To configure Internet Explorer to use Project InVision

  1. From the Internet Explorer Tools menu, select Internet Options.
  2. Click the General tab.
  3. In the Temporary Internet files section, click Settings.
  4. Under Check for newer versions of stored pages, select Every visit to the page.
  5. Click OK.
  6. Click the Security tab.
  7. In the Select a Web content zone to specify its security settings group, click the Trusted sites icon.
  8. Click Sites. The Trusted sites dialog box appears.
  9. If your Project InVision URL begins with “https” (note the “s”), select the Require server verification (https:) for all sites in this zone check box. If your Project InVision URL begins with “http” (no “s”), clear the check box.
  10. In the Add this Website to the zone text box, type the URL you've been given to access Project InVision, then click Add.
    The URL should be the root name of the Website (i.e., http://www.WebSite.com or https://www.SecureWebsite.com).
  11. Click OK.
  12. In the Security level for this zone group, click Custom Level. The Security Settings dialog box appears.
  13. Select Enable for the following options:
    • Download signed ActiveX controls
    • Initialize and script ActiveX controls not marked as safe
    • Run ActiveX controls and plug-ins
    • Script ActiveX controls marked safe for scripting
  14. Click OK.
  15. Click OK.
