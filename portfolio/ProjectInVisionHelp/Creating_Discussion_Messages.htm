Creating Discussion Messages

You can create a new Discussion message on the Discussions page of a Projects, Tasks, Issues, Defects, Risks, Changes, or Requirements Detail Form.

To create a Discussions message

  1. View the Detail Form of the record where you want to create a new Discussion message.
  2. Under Pages, click Discussions.
  3. Click New Topic. The New Topic window appears.
  4. Enter a Subject and Message.
  5. Click Submit.

Once posted, a Discussions message cannot be edited or deleted on the Discussions page of the Detail Form. To edit or delete these messages once entered, access the message in the Journals module. See Editing Journal Entries.
