Timesheet Approval Filter By Options

When you are approving Timesheets, you can filter the view by the options in the Filter By list. The options are:

Project Manager
Shows Timesheet records for all Resources who are assigned to Tasks on specific Projects for the selected Project Manager, regardless of the Resource's manager.
Resource Manager
Shows Timesheet records for all Resources who have the selected Resource as their manager, regardless of the Projects to which the Tasks belong.
Project
Shows Timesheet records for all Resources assigned to any Task on a single Project.
Resource
Shows Timesheets records for a single Resource, regardless of the Project to which the Tasks belong.
