Viewing Project Schedule Files

You can view project schedule files that are attached to Project records whether or not the files are checked out. Viewing a project schedule file does not check it out. If a project schedule file is checked out by another user, it is not editable.

To view a project schedule file

  1. View a Project Detail form that has a project schedule file associated with it.
    -or-
    In a Projects module list view, select a Project record with an associated project schedule file.
  2. Click  Review.
  3. If you're asked whether to Open or Save the document, click Open.
