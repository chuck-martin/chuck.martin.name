Removing Resources from Projects

You can remove one or more Resources from a single Project in the Projects module or remove a single Resource from one or more Projects in the Resources module.

To remove Resources from a Project in the Projects module

  1. In a Project module list, select a Project record, then click  Team. The Define Project Team dialog box appears.
    You can also view a Project Detail form, then click  Team in the toolbar there.
  2. In the Assign drop-down list, select Resources to Projects.
  3. Select one or more names in the Team list
    Hold down the Shift key while clicking to select a contiguous group of names.
    Hold down the Ctrl key while clicking to select multiple discrete names.
  4. Click  Remove.
  5. Click Submit.

You can click  Remove All to remove all names currently showing in the Team list.

To remove a Resource from Projects in the Resources module

  1. In a Resource module list, select a Resource record, then click  Team. The Define Project Team dialog box appears.
    You can also view a Resource Detail form, then click  Team in the toolbar there.
  2. In the Assign drop-down list, select Projects to Resources.
  3. Select one or more projects in the Team list
    Hold down the Shift key while clicking to select a contiguous group of projects.
    Hold down the Ctrl key while clicking to select multiple discrete projects.
  4. Click  Remove.
  5. Click Submit.

You can click  Remove All to remove all projects currently showing in the Team list.
