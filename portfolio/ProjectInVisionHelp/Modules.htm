Modules

Modules are the functional areas within the Portfolio Manager application. Each module captures a specific type of project management information. Records in all modules are associated with specific Projects. (Resources can be associated with multiple Projects.) The default modules are:

  • Projects
  • Tasks
  • Resources
  • Documents
  • Issues
  • Defects
  • Journal
  • Risks
  • Changes
  • Requirements
  • Collaboration

The Modules menu on the left side of Portfolio Manager contains links to the main functional areas of Project InVision. Any custom modules created specifically for your Workgroup or your company will also appear on the Modules menu.

Module List Indicators

Some module lists contain a field with graphical Indicators that provide additional information about the record. For more information, see List Indicators
