Editing Links

To edit a link to a Project InVision record

  1. View a Detail Form.
  2. Under Pages, click Links.
  3. Click an item in the Item column that is not a URL item.
  4. Click in the corresponding field in the Link column, the click the down arrow button.
  5. If you want to edit a link to an existing record:
    1. Choose Select from Existing Items. The Find dialog box appears.
    2. Select a record from the list, then click Select.
    3. Click  Save.
    If you want to define a link to a new item:
    1. Choose Create New Item. The New Detail Form for that type of record appears.
    2. Enter the information for the new record, then click Submit.
    3. Click  Save.
  6. Click  Save.

To edit a link to a web address outside of Project InVision

  1. View a Detail Form.
  2. Under Pages, click Links.
  3. Click a URL item in the Item column.
  4. Click in the corresponding field in the Link column, then click the down-arrow button. The URL dialog box appears.
  5. Make any desired changes to the URL label (the plain text description that will appear in lists) and the complete URL (web address), then click Select.
  6. Click Save.
