About the Tasks Module

The Project InVision Tasks module is where you manage and collaborate on project schedules. The Tasks module provides simple Web-based task management for smaller projects and offers the capability to integrate and synchronize with other project scheduling software tools for larger more complex projects. You can store and manage Microsoft Project format (.mpp or .mpx) Project schedule files using the Project InVision check-out and versioning features and can view Task information in list views or in the Tasks Detail Form.

Tasks are activities that have beginnings and ends. Task records in Project InVision allow you to track tasks to completion. In Project InVision, all Tasks are associated with Projects. Resources assigned to Tasks can post Work Hours against those Tasks in the Time Tracking application.

For advanced scheduling capabilities, such as Task dependencies and date constraints, you can perform all scheduling activities and Resource assignments in an associated project schedule file. That file can be checked in to and synchronized with Project InVision to plan, track, and communicate activities for the Project in relation to the other Projects in the Workgroup.

The Project InVision Tasks module can be populated and managed by either associating a project schedule file or by entering task information directly into Task Detail forms

If you associate a project schedule file, the following applies:

  • Tasks can be edited in the associated project schedule file and then checked in to Project InVision, or Tasks may be edited directly in the Project InVision Tasks module. Synchronization ensures that the project schedule file and Project InVision Tasks module are the same.
  • To add or delete Tasks, the associated project schedule file must be checked out. Tasks cannot be added or deleted directly through the Project InVision Tasks module when there is an associated Project schedule file. See About Project Schedule Files.
