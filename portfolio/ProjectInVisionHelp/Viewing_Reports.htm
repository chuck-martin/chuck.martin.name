Viewing Reports

Viewing reports involves first defining the report options, then viewing the report in a separate window.

To view a report

  1. Click  Output, then Reports. The Report Options dialog box appears.
  2. In the Report List, select a report type. Click the plus (+) sign next to each group to see the reports in that group.
  3. Under Options, define the report options. Different reports will have different options to choose from.
  4. Click View Report.

You can use the Size drop-down list box to resize the appearance of the report.
