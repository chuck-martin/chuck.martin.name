Shortcuts and Tips

Project InVision is both powerful and flexible. It often provides many ways and quick ways to do tasks. Listed here are some useful shortcuts and tips.

  • Find dialog box - If you type one or more characters in a field, then click Find, the Find Using criteria will be pre-filled with the characters you typed and the results will be pre-filtered.
  • Column resize in list views - In list views, if you hover the mouse over the border between header labels, the mouse pointer turns into a two-headed arrow. Click and you can drag the column border to resize the column. This width is maintained for the current session only.
  • List view modes - List views have 2 modes (if configured by your Workgroup Administrator): Editable and Read Only. In Editable mode, you can click in any field and press F2 to either edit the text directly or display the Find dialog box (if you can select only from a fixed list of items).
  • Filter removal - When in a list view that is using filters, you can right-click anywhere in the list and select No Filer to remove all filters from that view.
  • Column sorting - In list views, you can click on a column header to sort by that field.
  • Overriding Selector selection - When you're using the Selector to drive what records you see in other modules, you can hold down the Ctrl key when you click on a Project record to temporarily override the Projects selected in the Selector.
  • Time entry templates - In Time Tracking, click the Copy Previous Week button to use the previous week's timesheet as a template for the current week.
