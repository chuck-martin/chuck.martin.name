Creating Templates

In Project InVision, templates are created from existing program files. New Templates can be added and made available to the Documents or Projects modules. For the Projects module, the Template must be a Microsoft Project-compatible .mpp file. For the Documents module, the Templates can be of almost any file type (doc, xls, ppt, wk1, etc.).

Creating templates can be done only by users with access to Administrative functions

To create a Template

  1. Click Administration. The Project InVision Administration window appears.
    The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.
  2. Under Select Screen, click Templates.
  3. In the Navigation menu, select the Workgroup and the module where the template will be located
  4. Click  New. The New Template dialog box appears.
  5. Enter the Template information.
    • Template Title: A short, useful description of up to 50 characters. This is what users will see what selecting templates.
    • Template Type: A category used for sorting.
    • File Extension: The 3-character extension of the file type used for the Template.
    • Description: A more comprehensive description of the Template, including suggested uses, Workflows, etc.
  6. Attach a file to the Template.
    1. Next to the File Name field, click Browse.
    2. Select a file, then click Open.
  7. Click Submit.
