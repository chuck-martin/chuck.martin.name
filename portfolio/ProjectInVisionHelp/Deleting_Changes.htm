Deleting Changes

Changes can be deleted from the Changes module if, for example, one was created by mistake. However, closed Change records should remain in the Changes module so the actions taken to resolve the Change remain documented. When a Project is archived by the Workgroup Administrator, all associated Change records are archived with the Project.

You can delete Changes only if you have the proper security.

To delete a Change

  1. In a Changes module list view, select a Change record.
  2. Click  Delete.
  3. Click Yes (to confirm deletion).
