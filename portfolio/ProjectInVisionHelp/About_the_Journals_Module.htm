About the Journals Module

The Project InVision Journal Module is where you maintain project-related Journal entries and where you review and collaborate on Discussion entries from all Module records. You can view Journal entries in list views or in the Journal Detail Form, but Discussion items are best viewed through the Central Discussions outline view.

Journal notes are used as a knowledge base for status notes on how Projects have progressed and other events that happen during a Project that do not relate to any of the other modules. The Journal module capturing Discussion items from all the module allows you to follow all threaded discussions related to a particular Project.
