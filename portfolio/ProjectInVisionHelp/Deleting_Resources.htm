Deleting Resources

You can delete Resources from Project InVision (for example, if the Resources leave the Enterprise), but it is better to mark such Resources as Inactive (by clearing the Active checkbox in the Information section of the Resources Detail form). The Workgroup Administrator should delete Resources.

If a Resource leaves a Workgroup, they should not be deleted from the Resources module, but rather removed from the Workgroup. Contact the Workgroup Administrator to have the Resource removed from the Workgroup.

You can delete Resources only with the proper security.

Resources should be deleted from Project InVision only when absolutely necessary. Because Resources link to Projects, Resources, Tasks, and Task-related items, Resources should be made Inactive, rather than be deleted from the application.

To delete a Resource

  1. In a Resources module list view, select a Resource record.
  2. Click  Delete.
  3. Click Yes (to confirm deletion).

Deleting Resources from the Resources module removes them from the Enterprise. Memberships to all Workgroups, Projects assignments, Task assignments, and Task-related assignments are lost.
