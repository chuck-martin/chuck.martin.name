Configuring Project InVision

You can customize how Project InVision works in several ways. The My Setup windows allows you to

  • Configure how all toolbars appear
  • Change your password
  • Configure your search settings
  • Configure how the Selector operates.

Click the My Setup link in the upper right of any Project InVision application window to access the My Setup window.

To configure toolbar appearance

  1. Click My Setup.
  2. Under Select Item, click Toolbar Settings.
  3. Select one of the toolbar appearance options:
    • Icon Only - To view only icons in toolbars.
    • Text Only - To view the text of the toolbar commands only.
    • Both Text and Icons - To view icons alongside the command text.
  4. Click  Save.
  5. Click Submit.
