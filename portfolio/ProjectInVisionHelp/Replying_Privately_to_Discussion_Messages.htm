Replying Privately to Discussion Messages

You can reply privately to Discussion messages so the reply will go directly to a Resource and not be posted publicly.

To reply privately to a Discussion message

  1. View the Detail Form of the record that contains the Discussion message you want to reply to.
  2. Under Pages, click Discussions.
  3. In the message list, click the message you want to reply to.
  4. Click Reply to Sender. The Send Message window appears.
  5. In the Message box, enter your reply text.
  6. Click Submit.
