Creating Resource Logins

Once a Resource is created, a login (with password) must be created (along with the Resource being assigned a User Category) before that person can use Project InVision.

Creating Resource logins can be done only by users with access to Administrative functions.

To create a Resource login

  1. Click Administration. The Project InVision Administration window appears.
    The link is in the upper-right part of the window. If you do not have System Administrator or Workgroup Administrator access, you will not see this link.
  2. Under Select Screen, click Resources.
  3. Click the name of a Resource with a blank Login field.
    Do not click the Resource # link.
  4. Click Edit User Login. The Edit User Logins dialog box appears.
  5. Enter a New Login and a Password (twice).
  6. Click Submit.
