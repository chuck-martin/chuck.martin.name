Modifying a Time-Off Request

You can make changes to time-off requests that you have not submitted for approval. If you have already submitted a time-off request for approval, you must reopen the time-off request before you can make any changes. (The person you sent the time-off request to for approval will be notified the time-off request is re-opened.)

To modify a time-off request

  1. On the Time-Off Requests page, click the link for a time-off request.
  2. Make any desired changes in the time-off type, start and end dates, or notes.
  3. Click Save.

Note: Saving the time-off request does not submit it for approval.
