Customer or Project Detail allows you to input information about a new customer or project, or modify the information for an existing customer or project. The Customer or Project Detail page is only available from the Customer List.

  • If you are creating a new customer, the form is titled Customer Detail.
  • If you are creating a new project, the parent customer and any parent project names appear in the title, followed by Project to indicate that you are creating that project.
  • If you are modifying an existing customer or project, the customer name and project names will appear in the title.

When you create a new project the information for that project is copied from the parent customer or project. It can be modified for this particular project. That modified information will carry over to new sub-projects of this project, but will not affect the parent project or customer.

What do you want to do?

Create a new customer or project
Modify an existing customer

To create a new customer or project

  1. Enter the appropriate information. Make sure the Customer Name is appropriate for your company's customer naming system.
  2. Click Save.

To modify an existing customer

  1. Modify the appropriate information.
  2. Click Save.
