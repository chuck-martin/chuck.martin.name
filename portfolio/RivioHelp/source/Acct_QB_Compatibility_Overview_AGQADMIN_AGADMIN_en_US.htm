QuickBooks Compatibility Overview

For security reasons, there is no direct connection to QuickBooks on your computer. This system does not directly make changes to your data on your computer. Instead, the Accounting area provides QuickBooks compatibility through QuickBooks IIF (Information Interchange Format) files.

About IIF Files

QuickBooks creates IIF files when you export data from your QuickBooks system. Those files are then used by this system to import that same data. Going the other way, this system creates IIF files to export data that you can then use to import that same data into your QuickBooks system.

When you use your QuickBooks system to prepare a file for export, you tell it what data to include. This system will accept IIF files containing your:

  • Customer List
  • Employee List
  • Chart of Accounts
  • Credit Terms

QuickBooks IIF files are in a standardized format that identifies what type of data is included. If you export data from your QuickBooks system that is not importable into this system, those records will be simply ignored when you attempt the import.

When you use this system to prepare data for export, you can tell it what data to include. This system will generate IIF files containing your:

  • Customer List
  • Employee List
  • Expense Reports

These IIF files are in the standardized format that QuickBooks recognizes. To load this data into your QuickBooks system, you must instruct QuickBooks to import your saved, export IIF file.

Incremental Changes

Each Export File is an incremental change file. It includes information about only your data changes since the last time you created an Export File. If you create one Export File at the end of the day and create another the next day at Noon, the second file will only contain that morning's changes. Thus your Export Files must all be installed into your QuickBooks system, one after another, in the order they were created.

But what happens if something happens to last night's Export File before you get it installed into your QuickBooks? Creating a new Export File will only get you the mornings changes, not yesterday's.

This is what Re-Export is for. This system keeps a copy of every Export File you generate. (These have the same encryption security as the rest of your data, are encrypted during transmission, and are only decrypted as they are saved on your hard drive.) When you open Re-Export you will see a list of every Export you have done. From here you can select any previous Export File you need and get a copy of it.

As a worst case, suppose your computer completely dies and you don't have a current backup. You can restore your system with whatever backup you do have, and then recover all of the exportable data from this system by simply re-exporting all of the files you need and re-importing them into your QuickBooks.
