Adding Procedures

Procedures involve two parts: the overall procedure, and the steps within the procedure.

You must have HR Administrator or System Administrator access rights to add a procedure.

To create a procedure

  1. Click , then under Company Policies, click Setup.
  2. Click Procedures.
  3. Click Add Procedure.
  4. Enter the Procedure Name.
  5. Enter any notes.
  6. Select Mark as new if this is a new procedure.
  7. Optionally, you can select the checkbox to notify employees of this new procedure by e-mail.
  8. For the procedure content, choose Upload File From Disk or choose Enter Text.
    • If you choose Enter File From Disk, click the Enter File From Disk link and enter the full path to the file, or click Browse to locate the file. Then, click Upload.
      Note: Files must be Microsoft Word (.DOC) or Rich Text Format (.RTF) files. Each file can be no more than 100 KB, and the total size of all Procedure files uploaded cannot exceed 1 MB.
    • If you choose Enter Text, you can enter text or simple HTML to create steps for the procedure:
      1. Click Add Step.
      2. Enter the name of step.
      3. Enter details for the step.
      4. Select Mark as new if this is a new step in the procedure.
      5. If there are additional steps, click Add Another Step and repeat steps 2 through 4.
      6. Click Save & Return.
  9. Click Save & Return.
