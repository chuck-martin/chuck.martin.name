Modifying a Form

All the Company Forms forms are in Microsoft Word's .RTF (Rich Text Format). You can modify any of the forms with Word or any other word processor that can import an .RTF file.

To modify a form

  1. View the form you want to modify and save it to your local hard disk.
  2. Find the file you saved and open it (typically by double-clicking).
  3. Make any desired changes and save the file.
