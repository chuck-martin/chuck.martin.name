Earning Types Detail

On the Earning Setup page, you define the different ways that employees earn their pay. If you are already using a payroll provider, use this page to duplicate the type of information you already use with your current provider (This will assist when creating the payroll report you fax to your current provider).

The table below will help you enter the correct information on this page.

Column Description
Name This is a short text description or abbreviation of the different types of earnings available. You can enter up to 7 characters. You might want to use abbreviations of REG for regular hours, OT for overtime hours, PTO for paid time off, and so on.
Description You can enter a longer description of up to 20 characters here.
Is this based upon hours worked?
  • Select Yes if the earning code is for an hourly wage calculation. Typically, you would also select Nonexempt when you select Hours.
  • Select No if the earning code is for payments usually based on fixed dollar amounts (that do not need to be calculated), such as bonuses. The fixed amount may be based on a unit of time (week, month, etc.), or an agreed-upon amount of work. Typically, you would also select Exempt when you select Dollars.
Who is this earning applicable to?

In general, there are two types of employees: exempt and nonexempt. Typically, exempt employees are salaried employees who receive a set amount of pay every time period and do not receive overtime. Nonexempt employees are typically paid a set amount for each hour they work and are eligible for overtime. (The amount of time an employee works before receiving overtime is typically defined by law on a state-to-state basis.)

When creating a payroll, there is a time sheet allocation page. Exempt and nonexempt employees are separated by these categories on this page.
