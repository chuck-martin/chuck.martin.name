Submitting a Time-Off Request for Approval

To submit a time-off request for approval

  1. On the Time-Off Requests page, click the link for a previously created time-off request or create a new time-off request.
  2. Click Submit for Approval.
  3. From the Send To drop-down list box, select a name
    -or-
    In the E-Mail box, type a valid e-mail address.
    Note: Only employees who have been assigned Manager access rights appear in the drop-down list.
  4. If desired, type an additional comment or due date.
  5. Click Send.
