Review Details

Use the following information to enter the appropriate information when adding or modifying a review.

Item Description

Review Date

The date that the review will take effect.

Review Period

The start and end date of the time that the review covers.

Reviewer

Select one name from the list of all users.

Rating

Select one from Exceptional, Exceeds Expectations, Meets Expectations, Below Expectations, and Needs Improvement.

Review Results?

Select any of the items if the review results in any of the actions listed.

Next Review Date

Enter the date for the next scheduled compensation review.

Comments

Enter any optional comments.
