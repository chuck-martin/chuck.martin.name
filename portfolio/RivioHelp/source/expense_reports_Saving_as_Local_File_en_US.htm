Saving an Expense Report as a Local File

You can save an expense report as a file on your local or networked hard disk as an Adobe Acrobat file. Viewing the saved file requires the Adobe Acrobat Reader, available free of charge at Adobe.

A saved expense report includes the list of expenses and the history of the users who have acted on the expense report.

To save an expense report

  1. Open an expense report.
  2. Click Print/Preview.
  3. In the window that opens, click the Acrobat button.
  4. Click the Save button.
  5. In the Save As dialog box, enter a file name, navigate to the folder you want to save the file in, and click Save.
