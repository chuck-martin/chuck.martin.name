Adding an Employee

Information about an employee includes:

  • Name, e-mail address, and User ID and password
  • Personal information
  • Work information

You must have System Administrator or HR Administrator access rights to add employees to the Employee Directory.

To add an employee to the Employee Directory

  1. Click  , then under  Employee Records, click Employee Info.
  2. Click Add Employee.
  3. Enter the employee information.
    Note: The User ID must be unique.
  4. Click Save & Return.
