Adding a Time-Off Request

To add a time-off request

  1. On the Quick View or the Tasks page, from the drop-down list in the Task List section, select Time-Off Request, then click Add.
    -or-
    On the Time-Off Requests page, click Add Time-Off Request.
  2. From the Time-Off Type drop-down list box, select a time-off type.
  3. Enter a start and end date.
  4. Click Save.

Note: Saving the time-off request does not submit it for approval.
