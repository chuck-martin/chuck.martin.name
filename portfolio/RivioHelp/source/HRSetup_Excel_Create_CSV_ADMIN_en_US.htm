Many spreadsheet programs can import CSV files. Specifically, the sample CSV file is designed for Microsoft Excel or a similar spreadsheet program.

The sample contains a column-header row that briefly identifies the fields and simple instructions.

To create a CSV file using Excel

  1. Once the sample file is saved on your local hard disk, start Excel or your preferred spreadsheet program and open the file.
  2. The CSV file does not have formatting information such as column widths, so make adjustments to easily read the data.
  3. Enter (or cut-and-paste from another application) your employee data, following the samples and the column names. While you are preparing the file, you can save it in the spreadsheet program's native format.
  4. When you have entered all your employee data, delete the instructions.
  5. Save the file as a CSV file. You can now use the Data Import system to select and import this file.
