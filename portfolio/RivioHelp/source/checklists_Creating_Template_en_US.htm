Creating a Checklist Template

You can create a template for checklists that you have to perform frequently. You can create new templates based on existing templates, and further modify the individual tasks if you wish. Templates contain your own steps and groups, along with an optional description and a title for the template.

To create a checklist template

  1. On the Quick View or the Task List page, from the drop-down list in the Task List section, select Checklist, then click Add.
    -or-
    On the Checklists page, click Add Checklist.
  2. Click New Blank Checklist link.
  3. Type a title (required) and a description (optional).
  4. Add optional steps and groups:
    • To add a step, which will contain a check box, type the text you want to use for the step in the Text box and click Insert Step.
    • To add a group, which does not contain a check box (you can use it for organizing and grouping steps), type the text you want to use for the group description in the Group box and click Insert Group.
    As you add steps and groups, they will appear in the list with the just-added item selected. Items that you continue to add will be placed in the list immediately below the currently selected item.
    In the list, groups appear along the left margin, and steps appear indented.
  5. If you want to reorder the items in the list, highlight an item you want to move up or down and click the up or down Move Item buttons.
  6. Click Save as Template.
