Data Import and CSV Files in Depth

What is a CSV File?

A CSV file is a text file with a very specific format, generally with a filename extension of ".csv". If you use Microsoft Excel or Microsoft Access to create the CSV file, the proper format will be created for you, and you will not need to be concerned about the following details.

Each line in a CSV file is a data record made up of data fields, one record per line, with fields separated by commas.

If you load a CSV file into a spreadsheet, each line is a row, and each field is a column. If you load it into a database program, you have a great deal of flexibility in your display.

If a field (column) has a comma within it as text, the entire field between the separating commas must be enclosed within a pair of double-quote characters ("). These will not show in the spreadsheet display. If a double quote character is to be included within a field, the entire field must be in quotes and the internal double-quote character is represented by two double-quotes ("").

What is in an Employee List CSV file?

In an Employee List CSV file, each employee record must be on a line by itself, with no blank lines. In Excel this means no blank rows before the last row of data.

Within a line, all fields must be in the proper order and each field must be accounted for, even if it is empty. For an empty field, simply have no text between the separating commas. Excel and Access do this automatically.

All required fields must have appropriate data. To ensure this, be very careful how you add your data to your Excel or Access file. Use the column headings in the sample file as a guide.

Having the wrong number of fields or having inappropriate data in a field may cause the entire record to be rejected during the data import process. Be careful that your data matches the sample file and that there is nothing else in the file.

What Happens when I Import my CSV File?

The lines of the file are processed, one at a time, as if you had entered the data into the Add an Employee form. If the system finds an error, it will attempt to give you some information to help you find and correct the error. When it has completed processing the file, the system will tell you that it is done and how many records were successfully imported.

You should check your new employee records for accuracy to ensure that your data is in the proper fields.
