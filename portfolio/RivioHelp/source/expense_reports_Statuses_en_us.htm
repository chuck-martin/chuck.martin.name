Expense Report Statuses

The following table explains the meaning of the different expense report statuses:

Status Meaning
New Expense report has been created but not sent or forwarded to anyone. Changes can be made.
Open Two possibilities:
  • Expense report has been created and forwarded to someone (not submitted for approval). Only the person it has been forwarded to can make changes.
  • Expense report has been reopened (or retracted), and thus returned. Changes can be made.
Pending Approval Expense reports has been submitted for approval and is awaiting action. Changes cannot be made.
Approved for Payment Expense report has been approved and submitted to be paid.
Accepted for Payment Expense report has been processed and will be paid at next opportunity.
Completed Optional status you can use when you have received payment to indicate that no more action needs to be taken on the expense report.
