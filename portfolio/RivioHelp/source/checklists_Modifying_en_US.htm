Modifying a Checklist

You can add, delete, rename,or reorder the groups and steps that are part of an existing checklist.

To add a group or step to a checklist

  1. Open the checklist.
  2. Click Edit Checklist.
  3. In the steps list, select the item that you want to add a group or step after.  
    In the list, groups appear along the left margin, and steps appear indented.
  4. Add steps and groups:
    • To add a step, which will contain a check box, enter the text you want to use for the step in the Text box and click Insert Step.
    • To add a group, which does not contain a check box (you can use it for organizing and grouping steps), enter the text you want to use for the group description in the Group box and click Insert Group.
    As you add steps and groups, they will appear in the list with the just-added item selected. Items that you continue to add will be placed in the list immediately below the currently selected item.
  5. Click Save.

To delete a group or step from a checklist

  1. Open the checklist.
  2. Click Edit Checklist.
  3. In the steps list, select the item that you want to delete.
  4. Click Delete Item
  5. Click Save.

To rename a group or step in a checklist

  1. Open the checklist.
  2. Click Edit Checklist.
  3. In the steps list, select the item that you want to rename.
  4. In the box below the Group box, make your changes to the text.
  5. Click Rename Item.
  6. Click Save.

To reorder groups or steps in a checklist

  1. Open the checklist.
  2. Click Edit Checklist.
  3. In the steps list, select the item that you want to reorder.
  4. Click the up or down Move Item button.
  5. To reorder other items, repeat steps 3 and 4.
  6. Click Save.
