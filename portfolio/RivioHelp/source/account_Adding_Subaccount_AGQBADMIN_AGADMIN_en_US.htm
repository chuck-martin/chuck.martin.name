Adding a Sub-account

You can add sub-accounts manually only when the QuickBooks® interface is off.

Up to four levels of sub-accounts can be created for an account.

To add a sub-account

  1. Click , then under Accounting, click Account List.
  2. Select an existing account or sub-account.
    The Add Subaccount button appears.
  3. Click Add Sub-Account.
  4. Enter an Account Name and an optional description.
    Note: Because this is a sub-account, the account type is set as the same as the "parent" account.
  5. Click Save.
