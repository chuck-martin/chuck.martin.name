Setting Up Earning Types

Earning types are the types of income your employees can earn, such as regular time, overtime, time-and-a-half, double time, bonus, commission, and so on. These items can be set up for entry of hours (which are sent to your payroll provider and used to calculate gross dollar amounts) or a gross dollar amount.

To set up earning types

  1. Click , then under Payroll, click Setup.
  2. Click Earnings. More information
  3. In the Name field, enter the text you want to use for the different earning rates (integrated payroll services will use only the first four characters.) and in the Description field enter a plain text description.
  4. For Other Earnings, select the appropriate options for hours based and exempt vs. non-exempt.
  5. Click Add if you need to add more earning types.
  6. Click Save & Return.

Note: For InterPay users, once you define your original earning types and the InterPay processing setup is complete (typically 7-12 days from the date this information is sent to InterPay), these fields become non-editable. If you need to make changes after this, contact InterPay directly.
