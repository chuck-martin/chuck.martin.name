Exporting Data to QuickBooks

Exporting creates an IIF file that can be read by and imported into QuickBooks.

To export a data file from this system

  1. Click , then under Accounting, click Export from Site.
  2. If you want to export fewer types of information than you have selected in Accoutning Setup, clear the check boxes of the information types you don't want to export.
  3. Click Prepare Export File.
  4. Click Download.
  5. If you see a File Download dialog box, select Save this file to disk, then click OK.
  6. In the Save As dialog box, browse to the location on your computer where you want to save the file, then click Save.
  7. Follow your QuickBooks instructions to import the IIF file into QuickBooks.
