Adding Expense Report Item Details

The expense report form contains fields for entering basic expense report item details. You can add further details on the detail page.

To add expense report item details

  1. Open an expense report with a status of New or with a status of Open that has not been routed to anyone, or start a new expense report by clicking Add Expense Report.
  2. Click Detail next to a new or existing expense item.
  3. Enter the details for the expense report item:

    Item

    		 Description

    Expense Date

    Enter the date that the expense was incurred. Required field.

    Expense Type

    Select an account from the list. Required field.

    Currency

    Select the currency in which the expense item was incurred. Required field.

    Original Amount

    Enter the amount of the expense item in the currency in which the expense was incurred. Required field.

    Exchange Rate

    Enter the exchange rate of the expense item at the time the expense was incurred. Required field.
