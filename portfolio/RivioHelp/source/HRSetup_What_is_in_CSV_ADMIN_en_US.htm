Most spreadsheet programs accept CSV files. The sample CSV file can be imported into Excel, Lotus 1-2-3, or a similar spreadsheet program. The sample contains a column-header row that briefly identifies what should be in each column, a few sample employee entries to illustrate the column contents, and some simple instructions. While these are all useful in preparing your data, none of the sample file entries should be in your final data file.

Here are the columns in the sample file, with their intended uses:

# Heading Purpose
A First Name First name of employee (required)
B Middle Name Middle name
C Last Name Last name (required)
D Salutation Salutation of employee (Ms., Mrs., Mr., or Dr.)
E Address Line 1 Home address line 1
F Address Line 2 Home address line 2
G City City
H State State abbreviation
I ZIP ZIP Code
J Social Security Social security number (xxx-xx-xxxx format required)
K Home Phone Home phone number
L Work Phone Work phone number
M Work E-mail Employee e-mail address
