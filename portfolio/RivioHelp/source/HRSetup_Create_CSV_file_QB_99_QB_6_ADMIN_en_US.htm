Creating a CSV file using QuickBooks 6

Although the Data Import feature on this site supports QuickBooks 99 and QuickBooks 2000, it is possible to import an Employee List file from QuickBooks 6 by creating the file in CSV-format. After the CSV file is created, you must use a spreadsheet program such as Excel or Lotus 123 to edit the data before importing it to the system. In particular, you must ensure that all the fields exist and are in the proper order.

To create a QuickBooks Employee List file in CSV-format

  1. Open your Employee List. With that as your active window, open the file menu and choose Print List. Ignore any prompts to format your output before printing it.
  2. Click the File button to save the list as a file instead of sending it to a printer.
  3. Choose Excel/Lotus 123 spreadsheet from the File Format drop-down list. This will create a CSV-format file (Lotus 123 calls this type a .PRN file).
  4. Click the Print button.
  5. In the Create Disk File window, verify the directory in which you want to create the file. If you do not wish to save the file in the default directory (e.g., C:\Program Files\Intuit\QuickBooks) use the Save in field to browse to a new directory.
  6. Enter the file name (e.g. Employees.CSV) in the File name field. Be sure to specify the .CSV extension, since QuickBooks will otherwise automatically supply a .PRN extension.
  7. Click Save to save the file and return to QuickBooks.
  8. Use Excel or Lotus 1-2-3 to paste this file together with the sample file and ensure that all data is in the proper columns. Then remove the sample data and save the file as a CSV file.
