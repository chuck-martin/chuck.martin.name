Setting the Starting Expense Report and/or Invoice Numbers

The Expense/Invoice Setup page lets you specify starting numbers that are compatible with your Expense Report and Invoice numbering systems. There are three fields to each number:

  • Prefix: Use this if your number format has letter or number codes that come before the sequential number. Include any separating punctuation, such as a dash or slash.
  • Start Number: This is a sequential number that is increased each time an approved Expense Report or Invoice is generated. This field must be a number.
  • Suffix: Use this if your number format has letter or number codes that come after the sequential number. Include any separating punctuation, such as a dash or slash.

Setting these numbers is completely optional. If you do not specify anything, no prefix or suffix will be used and the first Expense Report and first Invoice will each be numbered 000001.

To set the starting number

  1. Enter the prefix and/or suffix that your numbering system requires, if any.
  2. Enter a starting number that will be compatible with your other numbering.
  3. Click Save.
