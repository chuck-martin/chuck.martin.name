About Invoices

An invoice is a type of task that contains standard information for invoicing vendors for products or services received.

Before you start using Invoices, you should set up your customers and projects in the Accounting area.

What do you want to do?

Add an invoice
Modify an invoice
Submit an invoice for approval
Approve an invoice
Decline an invoice
Sending an invoice to a client
Accept an invoice for payment
Print an invoice 
Set up invoice numbering
Archive an invoice
Reopen an invoice
