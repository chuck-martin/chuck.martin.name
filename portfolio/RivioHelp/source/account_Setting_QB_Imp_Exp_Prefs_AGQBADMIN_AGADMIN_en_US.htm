Setting QuickBooks Import/Export Preferences

You can define the maximum types of information you can import from and export to QuickBooks®. The maximum is also the default when importing and exporting.

These preferences are available only when QuickBooks integration is enabled.

To set QuickBooks import/export preferences

  1. Click , then under Accounting, click Setup.
  2. Click Set QuickBooks Integration Preferences.
  3. In the Import Data and Export Data sections, select the data types you want to integrate and clear the data types you don't want to integrate.
  4. Click Save & Return.
