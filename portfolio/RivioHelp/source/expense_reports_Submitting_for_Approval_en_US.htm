Submitting an Expense Report for Approval

When you complete an expense report, you can submit it for approval to authorized users (those with manager access rights) or to anyone with a valid e-mail address.

To submit an expense report for approval

  1. On the Quick View or the Expense Reports page, click the link for the expense report.
  2. Click Submit for Approval.
  3. From the Send To drop-down list, select the user name of a user
    -or-
    In the E-mail box, type a valid e-mail address.
    Note: Only employees who have been assigned Manager access rights appear in the drop-down list.
  4. If desired, enter an additional comment or Take Action By date.
  5. Click Send.
