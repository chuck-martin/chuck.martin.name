When you run reports from the Employee Records List, you select the employees first, and then select the type of report. If you start on the Reports page, you pick the report type first, and then choose the employees.

What do you want to do?

Generate a parameter-based Employment Report, Compensation Report, or Activity Log Report
Generate a simple report (not a parameter-based report)

To generate a parameter-based Employment Report, Compensation Report, or Activity Log Report

  1. On the Employee list, select the employees you wish to be covered by the report.
  2. In the Create Report drop-down list, select one of the above report types.
  3. Click Create. The parameter form appears.
  4. Set the parameters for the report you are creating.
  5. Click Generate. The report appears in a new browser window.

To generate a simple report (not a parameter-based report)

  1. On the Employee list, select the employees you wish to be covered by the report.
  2. In the Create Report drop-down list, select the report type (not one of the above parameter-based reports nor an Export Data report).
  3. Click Create. The report appears in a new browser window.
