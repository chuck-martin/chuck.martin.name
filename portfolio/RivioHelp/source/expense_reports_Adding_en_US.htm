Adding an Expense Report

To add an expense report

  1. On the Quick View or the Tasks page, from the drop-down list in the Task List section, select Expense Report, then click Add.
    -or-
    Click My Work, then in To Do, click Expense Reports, then click Add Expense Report.
  2. Enter a title in the Title box to identify this expense report.
  3. Enter the information for the expense report.
    Enter any amount advanced to the employee (optional).
    Enter a summary comment that provides more detail about the entire expense report.
    For each expense item:
    1. Enter the date of the expense.
    2. Select the account type.
    3. Enter the expense amount.
    4. Select the customer/project that the expense is related to (if applicable).
    5. Enter a note to describe the expense in more detail (optional).
    6. Click the Details link to:
      • Convert foreign currency to U.S. dollars
      • Identify an expense item as company-paid
      • Note that a receipt is not available
  4. If you need to add additional expense items, click Add Expense Item.
  5. Click Save & Return.

Click Submit for Approval when the expense report is completed to submit the expense report to the appropriate person and receive your reimbursement.
