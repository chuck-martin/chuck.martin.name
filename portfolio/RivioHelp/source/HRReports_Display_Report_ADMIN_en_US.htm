The parameter-based reports (Employment Report, Compensation Report, or Activity Log Report) and the simple reports appear in a new browser window. This window displays the report data, and has icons that allow you to print the report and to save the report in one of two formats:

  • PDF format is viewable with Adobe Acrobat. It allows you to print a formatted version of the report.
  • CSV format is readable by spreadsheet and database programs. This format allows you to combine data from multiple Export Data reports and use either spreadsheet or database functions to manipulate that data.

What do you want to do?

Download a report as a PDF file
Download a report as a CSV file
Print a report to your printer

To download a report as a PDF file

  1. Click the Adobe icon.
  2. Select a folder and filename to save the file.
  3. Click Save.

To download a report as a CSV file

  1. Click the Spreadsheet icon.
  2. Select a folder and filename to save the file.
  3. Click Save.

To print a report to your printer

  1. Click the Printer icon.
