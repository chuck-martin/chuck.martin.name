Adding a Time Sheet

The default time sheet contains two rows, so you can record the time spent on different projects or clients. You can add more rows by clicking the Add Row button.

The time sheet also contains four default categories to report time off.

To add a time sheet

  1. On the Quick View or the Tasks page, from the drop-down list in the Task List section, select Time Sheet, then click Add.
    -or-
    On the Time Sheets page, click Add New.
  2. Enter a Start Date and an End Date, then click OK.
    The default start date is the current date and the default end date is seven days from the current date.
  3. Click Save & Return.

Note: A time sheet shows seven days per page. If your time sheet contains more than seven days, click the Next and Prev links on the date row (at the top of the time sheet grid) to see more dates.

After you have set the start and end dates for a time sheet, you can enter your hours.
