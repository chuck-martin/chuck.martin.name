Selecting Employees for a Report

When you run reports from the Reports page, you select the type of report first, and then select the employees. When you have selected the employees, you then either set report parameters (for parameterized Employment Report, Compensation Report, or Activity Log Report) or generate the report directly.

What do you want to do?

Search by name or department
Hide the filter/search fields
Restore the filter/search fields

To search by name or department

You may enter a single search item or a combination of first name, last name, and/or department.

Your name search will match anything that includes your entry as part of the corresponding name. For example, if you enter Ann in the First Name Contains field, it will match anyone named Ann, Anne, Maryanne, Annette, and so on.

To select a department, choose an entry from the dropdown list.

When you have entered your search criteria, click Search.

To hide search area

Click Close Search Box to hide the filter/search fields and have more room to display items in the list.

To Restore the filter/search fields

When the filter/search fields are not visible, click Open Search Box to display them.
