Several pre-defined reports are available from Reports page, including:

  • Anniversary List: Employee hire-date anniversaries
  • Birthday List: Employee birthdays
  • Compensation History Report: Compensation History for a single employee
  • Compensation Report: Compensation for selected employees
  • Department List: All departments
  • Emergency List: Emergency Contacts for selected employees
  • Employee Census: Employment Status for selected employees
  • Employee Directory: Basic information for selected employees
  • Employee Profile: Expanded information for selected employees
  • Employment Report: Employees by start date
  • Home Phone List: Home phones for selected employees
  • Activity Log: Audit report of changes made by selected employees
  • New Hire List: From selected employees, which were hired in the last 12 months
  • Supervisor List: From selected employees, who are supervisors and who are their direct reports.
  • Termination List: Details on terminated employees
  • Work Phone List: Work phones for selected employees

Generate a report
Save the data in a report as a CSV or PDF file

To generate a report

  1. Click , then under Employee Records, click Reports.
  2. Click a type of report.
  3. Select the employees to include in the report. You may use the search-filter to limit the list of employees shown.
  4. Click Generate Report.
  5. If additional parameters are needed, a new page appears. Enter the appropriate criteria.
  6. The report will be displayed in a new browser window. If the system is unable to recognize the type of browser you are using, a page will appear asking to select your browser type. Select the first choice, Java compatible, and click Save.

To save the data in a report as a CSV or PDF file

The displayed report has two icons, one to save the report as an Adobe Acrobat PDF file and another to save the data as a CSV file that can be loaded into most spreadsheet and database programs.

  1. Click the appropriate icon.
  2. When the Save File dialog box appears, browse to an appropriate location.
  3. Modify the default file name if you wish.
  4. Click Save.
