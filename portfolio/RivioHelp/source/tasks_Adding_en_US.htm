Adding a Task

The Tasks area provides 6 custom task styles, as well as the standard To Do. You can add a task from your Quick View page, the Task List page, or from any of the task pages.

To add a task from the Quick View or the Task List page

  1. From the drop-down list under the Task List heading, select the type of task you want to add.
    Task type Description
    Checklist Can contain one or more check boxes (either from a template or user-defined).
    Expense Report Contains standard expense report items to be filled in, then can be submitted to a manager for approval.
    Invoice Contains standard invoice items to be filled in, then can be submitted to a manager for approval.
    Time-Off Request Contains selections for the type of time off and the dates, then can be submitted to a manager for approval.
    Purchase Request Contains standard fields for describing items to be purchased (item description, quantity, cost, etc.), then can be submitted to a manager for approval.
    Time Sheet Contains standard fields for submitting work time to a manager for approval.
    To Do Contains a standard title and a freeform description.
  2. Click Add.
  3. Fill out the fields in the task, then click Save & Return.

For more details about the specifics of each task type, click the link for that task in the above table.
