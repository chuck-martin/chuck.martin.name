Adding an Event

The Calendar tracks events of all types and shows you both your own events and company-wide events. You can add your own events to your calendar.

To add an event

  1. On any Calendar view page, click Add Event.
    -or-
    On a Day tab, click a time link.
    A blank Calendar Event page appears with the current date. If you clicked on a time link, that time is entered as the event start time.
  2. Enter the event information.
    • Enter a required title.
    • From the Type drop-down list box, select an event type.
    • Enter a date, time, and duration (if different from the existing information).
    • Enter a location.
  3. Complete any desired event options:
    • Select the All-day event check box if an all-day event.
    • Enter an additional note.
    • If a repeating event, click + next to Repeating and select a repeat option.
    • If you want to send a reminder via e-mail, click + next to Reminders, then select the Send a reminder option, select a time period from the drop-down list, and enter your e-mail address.
    • If you will be unavailable during the event time and want to let the rest of the company know, click + next to Company Notification via Employee Locator and select notification options.
    • If you want to invite other people to the event, click + next to Invitations, then enter the e-mail addresses of the people you want to invite.
  4. Click Save.
