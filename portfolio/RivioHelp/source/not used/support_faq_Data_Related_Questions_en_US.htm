Can I revoke account access of an employee for whom I have created a user account? If I revoke access, can I grant it again in the future?

Yes, you can revoke access and grant it again if necessary. Revoking an employee's access means that the employee no longer has access to the application, but you can still view their data. Revoking access has no effect on the employee's record within the human resources database. You can see employees who do not have access in the Access Rights page. Click  , then under System Admin, click Access Rights, then click Assign Access to view employees who have not yet been granted access.

To grant access again to an employee (or to grant it initially), select the employee to whom you want to grant access, select the access right, then click Create Account.

Can an existing database of employee's personnel information be imported to the system?

Yes. You can import an existing database of employee information using a Comma Separated Value (CSV) format. You can also import employee data directly from QuickBooks.

When I enter a date (such as a birth date), the system converts the format. What date formats can I use?

The system displays dates in the format "mm/dd/yyyy". The system converts the date you enter to its display format. If the system cannot recognize the date, it does not save the data. You will see a data validation error message and be prompted to enter a recognizable date.

Can I import and export database files?

Currently, users can import and export data in a Comma Separated Value (CSV) or QuickBooks format (.IIF). We want our system to be as useful to you as possible and plans are underway to import and export data in a variety of formats.

When I create an employee record, do I have to enter their information all at once, or can the employee's information be entered at a later date?

You are not required to complete all of the information when adding an employee record. The only fields you need to complete to create an employee record are first and last name (on the Personal tab of the employee record). Either you or the employee can complete the personal, work and other information at a later date.

Can I delete an employee from the database? If I delete an employee, can I add them back later?

Yes, you can delete an employee record. Click , then under Employee Records, click Employee Info. Click the employee record you want to delete. On the Personal tab, click Delete.

Deleting an employee record permanently erases all information associated with that record. If you have created a user account for that record, deleting the record also deletes the user account; that employee will no longer be able to sign in.

Once you delete an employee record, you will not be able to add it back later. You will have to create a new employee record for that employee.
