Creating a New Payroll

Creating a payroll is the process of collecting current employee HR, payroll, and time sheet information that will be consolidated into one report for submission to your payroll provider. Included in this process are the steps by which updates to employee data are reviewed and approved prior to being sent to your payroll processor.

To create a payroll

  1. Click , then under Payroll, click Processing.
  2. Select a Pay Frequency and a Pay Period.
    Both of these were defined in payroll setup.
  3. Click Create Payroll.
  4. In the Pay Period Settings section, enter a Check Date (must be at least two business days from now if you are using InterPay). In the Special Settings section, enter any information specific to this payroll only. In the Hold Settings section, select any items that should be put on hold for this payroll only.
  5. Click Next.
  6. Select the employees for this payroll.
  7. Click Next.
  8. For any employees who have had changes in their payroll records since the last payroll run, select whether these changes should be used for this payroll run.
    Select New Hire when the employee is a new hire so InterPay can use that information to process new hire paperwork.
    Select Last Payment when the person is no longer employed or this will be the last payroll the employee will be part of. InterPay may process final paperwork (such as a W-2)and send it to you.
  9. Click Next.
    To view or edit the employee's time data, click the employee's name.
    In the list of Exempt and Non-Exempt employees, flags appear next to an employee name when there are relevant changes flowing through from the time sheet application that you might want to verify.
  10. Click Next.
  11. Click Override if you want to override any compensation or deduction amount settings for any employee (for this payroll only).
  12. Select the employees to include on the printable payroll report (or for InterPay users, select the employees whose data you want to send to InterPay).
  13. Click Submit to InterPay.
    Click Print Preview to view the payroll. (From the view page, you can also save it as a local file and print it out).
