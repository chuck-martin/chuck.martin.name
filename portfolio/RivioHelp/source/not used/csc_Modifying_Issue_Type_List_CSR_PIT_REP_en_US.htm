Modifying the Issue Type List

The Issue Type list is a drop-down list of issue types that the customer service representative (CSR) can choose from when entering issues. Each issue type is associated with a specific third-party where the issue will be escalated.

Only users with Rivio Administrator access privileges can modify the Issue Type list.

To modify the issue list

  1. Click the Customer Service Console tab, then in CSC, click Company List.
  2. Find any company by entering information about a company in the search fields and then clicking Find.
  3. Click any company name.
  4. Click Escalate Issue.
  5. Click Report an Issue.
  6. Next to the Issue Type drop-down list, click Edit.
    • To add a new issue type, enter the issue type and the e-mail address on a blank line of the third-party who will handle escalated issues of this type, then click Save.
      If there are no blank lines, click Add Row.
    • To modify an issue, make any desired changes to the text of the issue or to the person's e-mail address.
    • To delete an issue type, click Delete next to that issue.
  7. Click Save.
