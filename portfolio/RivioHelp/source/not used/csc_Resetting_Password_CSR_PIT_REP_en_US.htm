Resetting a Password

Sometimes a user might lose the password that allows access to the system. A customer service representative (CSR) can have  sent to the e-mail account of the user.

Only users with Rivio Administrator, CSR Supervisor, or CSR with Reset Passwords access privileges can reset a password.

To reset a password

  1. Click the Customer Service Console tab, then in CSC, click Company List.
  2. Find the user whose password you want to reset by entering the user ID in the User ID search fields and then clicking Find.
  3. Click the company name next to the User ID.
  4. Click User Information and verify the user's information:
    • Ask for the e-mail address.
    • Ask the Secret Question, enter the answer given, and click Check Answer. (make sure the answer is correct).
  5. Click Reset Password.

Note: The new password will also be displayed at the top of the page.
