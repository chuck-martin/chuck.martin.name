What are the different roles and access rights?

The System Administrator can assign different roles or access rights to different users. The information a user can see and the activities he can perform depend on his access rights. There are six different levels of access rights:

  • System Administrator
  • Human Resources (HR) Administrator
  • Accounting Administrator
  • Payroll Administrator
  • Manager
  • Employee

To see the descriptions for these levels, open System Admin, click on Access Rights, then click Assign Access. Open the user account for which you want to view or modify access rights.

Can an employee change or edit company-related data such as facts, addresses, and the company calendar?

No. Only the System Administrator can change company information. The employee (without System Administrator or HR Administrator status) can only view company information.

Top

Can the System Administrator view and edit another employee's Checklists?

Yes. The System Administrator or HR Administrator can view and edit any employee's Checklists. However, an employee (without System Administrator or HR Administrator status) cannot view or edit another employee's list.

Top

Can the System Administrator view the contents of all the employee's Task Manager messages that were created and assigned by other employees?

Yes. The System Administrator or Human Resources (HR) Administrator can view the list of any employee's Task Manager messages that were created and assigned by other employees. However, an employee (without System Administrator or HR Administrator status) cannot view another employee's list.

Top

Can an employee view another employee's confidential data?

No. Only the System Administrator, HR Administrator, or Payroll Administrator can view and edit confidential data. An employee (without System Administrator, HR Administrator, or Payroll Administrator status) can view his or her own data, but not someone else's.
