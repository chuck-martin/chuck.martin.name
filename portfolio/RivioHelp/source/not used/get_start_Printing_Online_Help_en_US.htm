Printing Online Help

If you are using Internet Explorer, you can use your browser’s built-in print function. If you are using Netscape Navigator, you can cut-and-paste the text and place it into a text editor.

To Print Online Help Using Internet Explorer

  1. Click the right mouse button.
  2. Select print from the popup menu.

To Print Online Help Using Netscape Navigator

  1. Use your mouse to highlight the text you want to print.
  2. Press Ctrl C on your keyboard to copy the text.
  3. Open a text editor such as Microsoft Word, or Notepad.
  4. Press Ctrl V on your keyboard to paste the text.
  5. Use the print function in your text editor to print the text.
