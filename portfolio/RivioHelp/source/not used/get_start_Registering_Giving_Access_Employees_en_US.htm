Registering and Giving Access to Employees

To maintain security, giving system access to employees is separate from entering employee information (making a new employee record). A new employee record must have a minimum amount of information:

  • First Name
  • Last Name
  • Work E-mail Address

You may want to enter other employee information at the same time as you enter this, but the above is the minimum required.

If you already have a database of employee information, in a spreadsheet, a database, or in QuickBooks, you can create employee records by importing that data into the system. You can import employee data at any time after you have completed your registration.

The import process is completely compatible with QuickBooks 99 and QuickBooks 2000. For other data, you simply prepare a spreadsheet with your data in columns in the appropriate order, save it, and enter that filename on the Import page.

After you enter or import your employee information, you can provide access to the system for those employees. Several levels of access are available, but most are for specific administrative tasks. You'll want to give most employees Employee access rights.

To create a new employee record from System Setup

  1. Click Set Up Human Resources. You see the Human Resources Setup page.
  2. Click Enter Data Manually You see a blank Add Employee form.

To create a new employee record from anywhere in the system

  1. Click the My Company tab.
  2. Open the Employee Records section of the menu.
  3. Click Employee Info. You see the Employee Records List. Your name will be on the list.
  4. Click Add Employee. You see a blank Add Employee form.

To create a minimal Employee Record

  1. On the Add Employee form, Personal tab, enter the employee's first and last names.
  2. Click the Work tab.
  3. Select the employee's Work Address from the drop-down list. This list contains the address you used when you registered, and any other company addresses you have entered using Company Info > Locations.
  4. Enter the employee's Work E-mail Address.
  5. Click Save.

To grant an employee access to the system

  1. Click the My Company tab.
  2. Open the System Admin section of the menu.
  3. Click the Access Rights menu item. You see the Access Rights page, open on the No Access list.
  4. Click Assign Access. You see a list of employees that currently have no access.
  5. Click an employee name.
  6. Select a level of access rights.
  7. Click Create Account(s). You now see the Access list, where the employee's name has been added.

The employee now has a User ID that has been automatically generated by the system. Before you notify the employee of their new access rights, you may want to change that User ID.

To change an employee's User ID

  1. In the Access list, click on the employee's name. You see the Access Rights page for that employee.
  2. Enter a new User ID for the employee.
  3. Click Save. You return to the Access list.

To notify employees that they have access to the system

  1. Select the employees you want to notify. Make sure that the User IDs are appropriate.
  2. Click Notify Selected Employee(s).

E-mail notices are automatically sent that inform the employees of the Web site address, the Company ID, their User ID, and a random password. They will be asked to change their password when they first sign in.
