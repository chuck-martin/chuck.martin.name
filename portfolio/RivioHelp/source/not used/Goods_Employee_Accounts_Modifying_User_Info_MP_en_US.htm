Modifying User Information
Comment this out or delete above here before Check-In -->

The Modify User Information page enables you to change the user ID and password originally submitted to set up an account at a vendor site. You can open this page through the Vendor Accounts page that lists the ID and password information.

To modify user information

  1. Enter the changed address in the  E-Mail field.
  2. Enter the changed password in the  Password field.
  3. Click  OK.
