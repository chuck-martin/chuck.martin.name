Escalating an Issue

Sometimes, when a customer service representative (CSR) receives an inquiry from a customer, the CSR determines that the issue must be escalated.

Only users with Rivio Administrator, CSR Supervisor, or CSR with Transfer Issue access privileges can escalate an issue.

To escalate an issue

  1. Click the Customer Service Console tab, then in CSC, click Company List.
  2. Find the company that has the issue you want to escalate by entering information about the company in the search fields and then clicking Find.
  3. Click the user name.
  4. Click Escalate Issue.
  5. Click Report an Issue.
  6. Select an issue category and either select an issue type (which will define who the issue is escalated to) or enter the e-mail of the person who should receive the escalated issue.
    If a third-party vendor is not in the Issue Type drop-down list, you must enter an e-mail address.
  7. Enter a complete description of the issue and any other relevant information, then select an issue Status (Draft, Pending, or Completed).
  8. Click Forward.
