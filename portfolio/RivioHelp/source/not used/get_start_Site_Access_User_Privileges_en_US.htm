Site Access and User Privileges

For security purposes, only users with System Administrator access rights can grant access privileges to other employees.

The following table describes the privileges for each type of access.

Access Type Privilege

System Administrator (SA)

Providing access rights, sign-in privileges. Has all privileges below.

Human Resources Administrator (HR)

Employee records, departments, reports, and company information. Access to all Accounting privileges.

Accounting Administrator (AA)
  • QuickBooks users: Can import data records for easy setup.
  • Non-QuickBooks users: Create chart of accounts, and customer projects.

Payroll Administrator (PA)

Payroll setup, tracking and managing payroll records, and creating pay register receipts.

Manager (MG)

Approving expense reports, timesheets, purchase requests, invoices, and time-off requests.

Employee (EE)

View and edit specific fields in own record, and view company information. Create and submit time sheets, purchase requests, invoices, time-off requests, expense reports, and purchase requests.

No Access (NA)

An employee record. No sign-in privileges.

Personal and confidential information is not accessible by non-authorized employees. Employees without administrative access cannot view, modify, or delete another employee’s information.
