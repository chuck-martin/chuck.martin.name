Viewing Item Details
Comment this out or delete above here before Check-In -->

You can open detailed descriptions of products and services from:

  • Any Search Results page
  • The Shopping List area (during processing)
  • The Shopping Cart main page
  • The Order Detail page

If opened from the Search Results page, you can add the product to your shopping cart.

If opened from the Shopping Cart main page, the Shopping List area, or the Order Detail page, you can only read the description, and return to the previous page.
