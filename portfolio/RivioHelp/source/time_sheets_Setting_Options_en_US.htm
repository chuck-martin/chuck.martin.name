Setting Time Sheet Options

You can set a date to take action by for a time sheet and set a notification to be e-mailed to you when the time sheet is either overdue or not completed by a set date.

To set time sheet options

  1. On the Time Sheets page, click the link for a time sheet.
  2. Click Options.
  3. Enter the desired options:
    • Enter a date to take action by.
    • Select to be notified if overdue.
    • Select to be notified if not completed by a date and enter that date.
     Note: Your e-mail address is already pre-entered.
  4. Click Save Options.
