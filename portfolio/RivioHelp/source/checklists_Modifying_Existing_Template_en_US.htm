Modifying an Existing Checklist Template

You can make changes to the custom checklist templates that you have created, and you can also use the General Templates as a basis for new templates (although the General Templates themselves can't be modified).

To modify an existing checklist template

  1. On the Checklists page, click the Customize Templates tab.
  2. Click the name of the template (in the Personal Templates or Company Templates column) that you want to modify.
  3. Make any desired modifications to the template steps or groups: adding, deleting, renaming, or reordering.
  4. If you want to change the template's category, or create a new category to put the template in, make those changes in the Template Category section. Either choose one of the categories from the existing lists or type a new category name in one of the text boxes.
  5. Click Save.

To modify a general template for use as a new template

  1. On the Checklists page, click the Add Checklist tab.
  2. Click the name of the General Template you want to modify.
  3. Click Edit Checklist.
  4. Click Save as Template.
  5. Make any desired modifications to the template steps or groups: adding, deleting, renaming, or reordering.
  6. If you want to change the template's category, or create a new category to put the template in, make those changes in the Template Category section. Choose one of the categories from either existing list, or type a new category name in one of the text boxes.  
    The default is in the Personal Templates group with no sub-category.
  7. Click Save.
