Setting Activity Log Report Parameters

The Activity Log Report Parameters form allows you to specify selection criteria.

  • You may specify date ranges that are this month to date, this year to date, or between specific dates.
  • You may specify the report to show all changes or only changes to employees' Personal, Work, Employment, Emergency Contacts, Dependents, Pay Rate or Review.
  • You may group the report by Employee Record, Type of change, Date changed, or Field changed.

To set parameters and generate a Compensation report

  1. Click the selection button to choose between month-to-date, year-to-date, or date range.
  2. If you choose date range, fill in the appropriate dates in the format shown.
  3. Select the types of changes from the drop-down list, or select All.
  4. Select the grouping parameter to use.
  5. Click Generate Report.
