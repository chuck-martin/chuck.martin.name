Tasks overview

The Tasks area provides several types of tasks that you can use to track and manage the things you need to do. While you can have basic To Do tasks, you can also use 6 other types of tasks:

  • Checklist
  • Expense Report
  • Invoice
  • Leave Request
  • Purchase Request
  • Time Sheet

These are in addition to To Dos, which, by contrast, contain just a description of a task, sometimes with a priority and due date attached

For checklists, which can contain one or more items to be checked off, you can use more than 2 dozen templates for common tasks. You can customize the existing templates or create your own from scratch.
