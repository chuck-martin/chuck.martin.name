Using Access to Create a CSV File

Many database programs, such as Microsoft Access, can import and export CSV files, although usually not as easily as with Microsoft Excel or a similar spreadsheet program.

You must be careful how you specify the file type in Access. You must specify that Access use a delimited file, with a comma delimiter, and with a text qualifier of the double-quote character. You must do this both when you import the sample file into Access and when you export your data to a file to be uploaded and imported into this system,

The sample contains a column-header row that briefly identifies the fields, a few sample employee entries, and simple instructions.

To create an importable CSV file in Access

  1. Once you have the sample file saved on your local hard disk, start Access and create a blank database.
  2. Select an appropriate location to save the database and name it.
  3. Select File > Get External Data > Import.
  4. In the Import window, specify Files of Type "Text Files (*.txt, *.csv, ...)" and click Import.
  5. Choose Delimited format. Specify a comma delimiter, a text qualifier of the double-quote character, and that the first row contains field names.
  6. Specify to save the data in a new table.
    Do not add a primary key or it will be included when you save your data, which will prevent proper importing.
  7. Enter (or cut-and-paste from another application or database) your employee data, following the samples and the column names.
  8. When you have entered all your employee data, delete the instructions, the sample entries, and the column captions (if they are in the data). Make sure that the only rows remaining are one for each employee.
  9. Select File > Save As > Export > To an External File or Database. Specify Save as Type "Text Files (*.txt, *.csv, ...)".
  10. Choose Delimited format. Specify a comma delimiter, a text qualifier of the double-quote character, and do not include field names in first row.
  11. Save the file as a CSV file. You can now use the Data Import form to select and import this file.
