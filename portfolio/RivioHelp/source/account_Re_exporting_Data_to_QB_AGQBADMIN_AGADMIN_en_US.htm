Re-exporting Data to QuickBooks

The system saves information about previously exported files, providing a "snapshot" of your data at that time. You can re-export that "snapshot" to a QuickBooks-compatible IIF file at any time.

To re-export data to a QuickBooks IIF file

  1. Click , then under Accounting, click Export Old Data.
  2. Select the date and time that the file was originally exported.
  3. Click Re-Export.
  4. If you see a File Download dialog box, select Save this file to disk, then click OK.
  5. In the Save As dialog box, browse to the location on your computer where you want to save the file, then click Save.
