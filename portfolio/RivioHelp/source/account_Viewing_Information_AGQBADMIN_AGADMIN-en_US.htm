Viewing Account Information

Information about an account includes the account name, type, description, and status (active or inactive).

Item

Description

Account Name

Descriptive name that you enter. This text is what appears in the expense report Account Type drop-down list box.

Account Type

Account Types help categorize accounts.

Description

(Optional) Seen only when viewing account information and can help more fully describe the use of the account.

Status

Select Active to make the account appear in the active account list and the expense report Account Type drop-down list box.

To view information about an account

  1. Click , then under Accounting, click Account List.
  2. Select an account or sub-account.
  3. Click View Selected.
