The Employment Report parameters page allows you to specify selection criteria. You may specify start and/or termination date ranges or choose All Dates to see all records. You may group the report by Department, Full-Time/Part-Time, or Employee Type.

To set parameters and generate an Employment report

  1. Enter the appropriate dates in the appropriate fields, or click All Dates.
  2. Select the grouping parameter to use.
  3. Click Generate Report.
