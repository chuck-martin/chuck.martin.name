Saving a Form

All the Company Forms forms are in Microsoft Word's .RTF (Rich Text Format). You can save any of the or Company Forms to your own hard disk.

To save a form

  1. View the form you want to save.
  2. From the File menu, choose Save As.
  3. In the Save As dialog box, navigate to the folder on your local drive that you want to save the file in.
  4. Type a file name or use the provided name.
  5. Click Save.
