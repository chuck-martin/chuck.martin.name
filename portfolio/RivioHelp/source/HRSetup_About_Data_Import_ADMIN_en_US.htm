About Data Import

If you already have a database of employee personnel information, you can easily import that information so you won't have to enter employee data manually.

Use the Import Wizard to guide you through the data import process.

Your employee information must be in CSV (Comma Separated Value or Comma-delimited) file format, which can be created using a spreadsheet such as Microsoft Excel, a database such as Microsoft Access, an accounting program such as QuickBooks, or a text editor such as Notepad.

The CSV file must have the required columns (called fields in Access). To make it easy to create the proper format, you can download a sample CSV file to use as a template.

Alternately, you can transfer employee data from your QuickBooks Pro files from the Accounting area.

To import employee data, you must have HR Administrator or System Administrator privileges.
