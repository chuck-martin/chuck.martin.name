Forwarding a Task

You can forward a task to anyone else in your company or to anyone who has a valid e-mail address.

Note: If you forward a task via e-mail, the recipient will get the task information as text in the body of the e-mail.

To forward a task

  1. From your Quick View page, the Tasks page, or any task type page, click the task name.
  2. Click Forward.
  3. In the Select Name drop-down list box, select the name of a user.
    -or-
    In the E-Mail Address box, type a valid e-mail address.
  4. If desired, type an additional comment or Take Action by date.
  5. Click Send.
