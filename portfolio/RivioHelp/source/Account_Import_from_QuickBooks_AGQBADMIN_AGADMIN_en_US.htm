Importing Data from QuickBooks

Importing information from QuickBooks involves getting the information from a QuickBooks IIF file into this system. The system is smart enough to import only the types of information from the IIF file that you select.

Note: You define the types of information that you are able to import on the Accounting Setup page. For any specific import, you can import fewer types of information, but not more types.

Before you begin the import process, you should export your QuickBooks information to an IIF file.

To import data from QuickBooks

  1. Click  , then under Accounting, click Import Into Site.
  2. If you want to import fewer types of information than you have selected in Accoutning Setup, clear the check boxes of the information types you don't want to import.
  3. Click Browse.
  4. Navigate to the QuickBooks IIF file that you exported. select it, then click Open.
  5. Click Import.

You will receive a confirmation showing the number of new records created, records updated, and other statistics.
