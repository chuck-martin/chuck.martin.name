Exporting Files from QuickBooks

Before you can import data into this system, you must first export it from QuickBooks. Exporting involves saving information from QuickBooks to an IIF file on your local hard disk.

To export a file from QuickBooks Pro 99

  1. In QuickBooks 99, select File > Export.
  2. Follow the instructions to create an IIF file on your local hard disk.

To export a file from QuickBooks Pro 2000

  1. In QuickBooks 2000, select File > Utilities > Export.
  2. Follow the instructions to create an IIF file on your local hard disk.
