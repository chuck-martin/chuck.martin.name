Submitting a Purchase Request for Approval

When you complete an purchase request, you can submit it for approval to authorized users (those with manager access rights) or to anyone with a valid e-mail address.

To submit a purchase request for approval

  1. Open the purchase request.
  2. Click Submit for Approval.
  3. From the Send To drop-down list box, select a user
    -or-
    In the E-mail box, type a valid e-mail address.
  4. If desired, type an additional comment or Take Action By date.
  5. Click Send.
