Searching for an Employee

You can search for any employee if you know any part of the employee's first or last name.

To search for an employee

  1. Click , then under Employee Records, click Employee Info.
  2. Click Open Seach Box.
  3. Enter all or part of a first or last name, and optionally limit the search to a single department.
    Tip: To get a list of all employees in a single department, select a department, but leave the first name and last name fields empty.
  4. Click Search.

Your name search will match anything that includes your entry as all or the start of the corresponding name. For example, if you enter Ann in the First Name Starts With field, it will match anyone named Ann, Anne, Annette, and so on.
