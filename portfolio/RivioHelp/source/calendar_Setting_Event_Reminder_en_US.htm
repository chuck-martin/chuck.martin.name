Setting an Event Reminder

Events can have reminders, which are messages sent to an e-mail address you specify. When you add a new event, a reminder is not set by default.

To set an event reminder

  1. Open the event by clicking on the event name.
  2. Click the + next to Reminders.
  3. Select the Send a reminder option and select a time period from the drop-down list.
  4. Enter the e-mail address you want the reminder sent to.
  5. Click Save.
