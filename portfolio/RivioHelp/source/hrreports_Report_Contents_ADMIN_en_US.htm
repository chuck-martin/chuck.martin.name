Report Contents

All reports list employees sorted by the first field in the report (or first field within a group). Names are always listed as Last Name, First Name.

Report Contents

Employee

Employee Directory

Name, Job Title, Supervisor, Department, Phone Number, Cell Phone, Work E-mail, Fax Number, Location

Employee Profile

All employee information, one employee per page

Employee Census

Name, Birthdate, Gender, ZIP, Spouse's Age, Number of Children, COBRA,

Work Phone List

Name, Job Title, Department, Work Number

Home Phone List

Name, Department, Job Title, Home Phone, Cell Phone, Home E-mail

Emergency List

Name, Emergency Contact, Relationship, Primary Phone, Secondary Phone; lists only employees who have emergency contacts entered

Birthday List

Birthday, Employee Name, Department, Job Title
Employment

Employment

Name, Employee ID, Employment Type, Full time/Part Time, Start Date, Department

New Hire List

Name, Job Title, Start Date, Deprtment, Supervisor, Salary; emplyees hired within last 12 months

Termination List

Name, Termination Date, Reason, Job Title, Department, Supervisor, Salary, Start Date
Organization

Department Directory

Name, Employee ID, Job Title, Work Numner; grouped by department

Supervisor Directory

Supervisor Name, Number of Employees Supervised; Employees Supervised List: Name, Job Title, Work Number, Location; list only employees who supervise employees and only employees who have supervisors identified

Anniversary List

Start Date, Name, Job Title, Department, Work Phone
Compensation

Pay Rate

Name, Employee ID, Start Date, Annual Compensation, Effective Date, Previous Compensation, Department, FLSA Classification; grouped by Department, contains only employees who have compensation entered

Pay Rate History

Name, Status, Title, Supervisor, Department, Effective Date, Reason, Amount, Period, Annualized, Hourly, % Increase; contains only employees who have compensation entered
Audit

Employee Record Activity Log

Date & Time Entered, Updated By, Employee Recored (Name), Type of Change, Field Changed, New Data, Old Data
